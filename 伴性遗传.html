基因与性状 · 伴性遗传与人类遗传病

基因如何控制性状、性别怎么决定、伴 X 隐性/显性/伴 Y 遗传的规律、系谱图判显隐与定位置、伴性概率计算、遗传病的类型与优生——把"基因坐哪条染色体、跟性别怎么绑"理顺,系谱题一眼看穿。

🗣️ 一句话总览: 基因是说明书,它通过指挥造蛋白质/酶来决定性状——有时直接管结构(镰刀型贫血),有时管代谢(白化病)。而伴性遗传说的是:某些基因不巧坐在了性染色体(X 或 Y)上,于是它的传递就和"你是男是女"绑在了一起。
男生只有一条 X(X 上的隐性基因藏不住,一有就发病),所以红绿色盲、血友病这类伴 X 隐性病男患者多得多;而 X 显性病(抗维生素 D 佝偻病)反过来女患者多。掌握"数一数 X、看男看女"这套账,系谱和概率题就都通了。

① 基因对性状的控制

基因不是直接"变出"性状,而是通过控制蛋白质的合成来控制性状。两条路径:管结构蛋白是直接控制,管酶影响代谢是间接控制。

途径怎么控制例子
直接控制(管蛋白质结构)基因 → 控制蛋白质的结构 → 直接影响性状镰刀型细胞贫血症:血红蛋白基因中一个碱基改变,谷氨酸→缬氨酸,红细胞变镰刀形
间接控制(管酶→代谢)基因 → 控制酶的合成 → 控制代谢 → 影响性状白化病:缺酪氨酸酶,黑色素合不成 → 白化
关键观念:基因与性状不是简单的一一对应。一个基因可影响多个性状,一个性状常受多个基因 + 环境共同影响。基因型 + 环境 = 表现型。

⚠ 镰刀型贫血的根本原因是基因突变(碱基替换),直接原因是血红蛋白结构异常;白化病根本原因是缺酶,是"基因→酶→代谢"的间接控制。别把这两条路径的例子记反。

② 性染色体与性别决定

人类体细胞 23 对染色体:22 对常染色体 + 1 对性染色体。女性 XX,男性 XY——这是 XY 型性别决定。性别由精子决定(含 X 或 Y)。

♀ 44 + XX　　♂ 44 + XY

同源区段:X 与 Y 上互相对应、可发生联会的一小段,上面的基因 X、Y 都有(可正常成对)。
非同源区段:X 特有的一大段(基因只在 X 上,记 X^A / X^a),Y 特有的一段(基因只在 Y 上,记 Y^b)。伴性遗传主要指非同源区段上的基因。

⚠ 常见的红绿色盲、血友病基因都在 X 的非同源区段,Y 上没有对应基因——所以男性(X^aY)只要 X 上带 a 就发病,没有"另一条 X 来掩盖"。

③ 伴性遗传:三种类型总览

位于性染色体上的基因,随性染色体遗传,总是和性别相关联——这就是伴性遗传。按位置与显隐分三类:

类型基因位置典型病例核心规律
伴 X 隐性X 非同源区段(隐性)红绿色盲、血友病男患多于女;隔代交叉遗传;女患者其父其子必患
伴 X 显性X 非同源区段(显性)抗维生素 D 佝偻病女患多于男;男患者其母其女必患
伴 Y 遗传Y 非同源区段外耳道多毛症限雄遗传;父传子、子传孙,全部男性患,女性不患

伴 X 隐性(以红绿色盲为例)

基因型:女 X^B X^B(正常)、X^B X^b(携带者、不患)、X^b X^b(患);男 X^B Y(正常)、X^b Y(患)。

三大特点(死记):
① 男患者 > 女患者(女要两条 X 都带 b 才患,概率小)。
② 隔代交叉遗传:外祖父(患)→ 女儿(携带者,不患)→ 外孙(患),基因走"外公→母→外孙"的斜线。
③ 女患者(X^bX^b)其父必患、其子必患(父给她一个 X^b、她给每个儿子一个 X^b)。

⚠ "交叉遗传":男性的 X 只能传给女儿(儿子拿的是他的 Y);女性的 X 一半给儿子一半给女儿。所以父亲的伴 X 基因永远到女儿身上,不会传给儿子。

伴 X 显性(抗维生素 D 佝偻病)

基因型:女 X^D X^D、X^D X^d(均患)、X^d X^d(正常);男 X^D Y(患)、X^d Y(正常)。

规律:① 女患者 > 男患者(女只要一条 X 带 D 就患)。② 男患者(X^D Y)其母必患、其女必患(母给他 X^D、他把唯一的 X^D 给所有女儿)。③ 世代连续遗传(显性,常代代有患者)。

伴 Y 遗传

限雄遗传:基因只在 Y 上,Y 只传给儿子。故父 → 子 → 孙全为男性患者,女性永不患病、也不携带。系谱里若"全是男的患,且父传子代代不断",优先考虑伴 Y。

伴 X 隐性交叉遗传动画(色盲:外公→母→外孙)

看色盲基因 X^b 如何走"外祖父 → 母亲(携带者) → 外孙"的交叉斜线,隔一代重现。

④ 系谱图分析:判显隐 + 定位置

系谱题两步走:先判显隐,再定位置(常/X)。口诀背熟就能秒断,断不了就"设基因型求概率"。

第一步:判显隐

无中生有为隐性:双亲都不患,生出患病孩子 → 病为隐性(且孩子拿到两个隐性基因)。
有中生无为显性:双亲都患病,生出正常孩子 → 病为显性(正常孩子是隐性纯合)。

第二步:定位置(常染色体 / X 染色体)

隐性病,看"父女 / 母子":若父患而女不患(或母患而子不患)→ 排除伴 X 隐(伴 X 隐时父患→女必携带并遗传,母患 X^bX^b→子必患)→ 判为常染色体隐性。
显性病,看"母子 / 父女":若母患而子不患(或父患而女不患)→ 排除伴 X 显 → 判为常染色体显性。

⚠ 口诀只能"证否":出现矛盾就排除伴 X。若无矛盾,常与 X 都可能,则不能确定,需比较概率或题目补充信息(如"每个个体都调查")。此时通常按题意"更可能"或"计算两种情形概率"。

⑤ 伴性概率计算

伴性遗传概率题的核心:先写清双亲基因型,再按"儿子看母亲的 X、女儿看父母各一条 X"分性别算。别把"孩子患病率"和"儿子中患病率"混为一谈。

色盲经典组合(X^b 为色盲):
母 X^B X^b(携带) × 父 X^B Y(正常) → 儿子:1/2 X^BY(正常)、1/2 X^bY(色盲);女儿:1/2 X^BX^B、1/2 X^BX^b,都不患。
所以后代患病率 = 1/4;但"儿子中患病率 = 1/2"。区分"占全部孩子"还是"占某一性别"。

⑥ 人类遗传病的类型

遗传病 = 遗传物质改变引起的疾病。三大类:

类型说明例子
单基因遗传病受一对等位基因控制(可常/X、显/隐)白化病、镰刀型贫血、红绿色盲、血友病、抗维 D 佝偻病、多指
多基因遗传病受多对基因控制,还受环境影响,家族聚集、群体中发病率高唇裂、原发性高血压、青少年型糖尿病、无脑儿、哮喘
染色体异常遗传病染色体数目或结构改变21 三体(先天愚型/唐氏)=多一条 21 号;性染色体病如 XYY;猫叫综合征(5 号缺失)

⚠ 21 三体属于染色体数目变异,不是基因突变,更不是单基因病!它是减数分裂时 21 号同源染色体未分开导致。别把它归到"单基因隐性病"里。

人类基因组计划(HGP)

测定人类基因组全部 DNA 的碱基序列。人有 24 条不同的染色体需要测序:22 条常染色体 + X + Y(因为 X 与 Y 不同,女性 XX 只算一种 X)。

⚠ 是 24 条(22 常 + X + Y),不是 23、不是 46。原因:同源染色体序列近乎相同只测一条,但 X、Y 不同源,必须各测一条。

⑧ 易错集

① 交叉遗传规律:男性伴 X 基因只传女儿,女性传儿女各半。色盲父亲一定把 X^b 给女儿(女儿至少携带者),绝不给儿子。

② 显隐口诀:"无中生有为隐性,有中生无为显性"——一定要双亲都是那个状态才能用。单看一个患者定不了显隐。

③ 常 vs X 的判断:隐性看"父患女不患 / 母患子不患"→ 若出现即为常染色体;显性看"母患子不患 / 父患女不患"→ 若出现即为常染色体。无矛盾时不能确定,别硬下结论。

④ 多基因病受环境影响:唇裂、高血压等群体发病率高、家族聚集但不符合单基因比例,是多基因病;别用 3:1、1:1 去套。

⑤ 21 三体是染色体数目变异(多一条 21 号),既不是基因突变也不是单基因遗传病;成因是减数分裂同源染色体未正常分开。

⑥ 携带者不患病:X^BX^b、Aa 这类杂合子表现正常,但能把致病基因传下去。"不患 ≠ 不带基因"。

⑦ "儿子中患病率" ≠ "孩子患病率":题目问哪个口径,答哪个。分性别时把儿子/女儿各算一遍再按需合并。

⑨ 尖子生拔高视角

▸ "看男看女"两秒定位:看到"男患远多于女患"→ 优先伴 X 隐;"女患远多于男患"→ 优先伴 X 显;"只有男患、父传子不断"→ 伴 Y。这是判位置的快速直觉,再用系谱验证。

▸ 系谱"证否"思维:口诀本质是找矛盾。伴 X 隐:父患→女必携带;母 X^bX^b→子必患。伴 X 显:母患→子女……用不到时,只要出现一个反例就排除伴 X,剩常染色体。没有反例则并存,转求概率。

▸ 两病自由组合计算:常染色体病(如白化 Aa)与伴 X 病(如色盲 X^BX^b)位于不同染色体,相互独立,用乘法原理:先分别算各病在儿/女的概率,再按性别相乘,最后合并。"至少患一病 = 1 −(都不患)"。

▸ 同源区段的特殊题:若基因在 X、Y 同源区段(记 X^A/X^a、Y^A/Y^a),男性也有一对等位基因(如 X^aY^A),此时男女均可为杂合、患病比例可能相近——遇到"男女患病数差不多"的伴性题,考虑同源区段。
