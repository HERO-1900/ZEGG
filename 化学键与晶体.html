化学键与晶体结构

离子键还是共价键?极性键还是非极性键?这块物质是离子晶体、分子晶体、共价晶体还是金属晶体?——一整条从"微粒怎么连"到"物质什么性质"的推理链。把每种键、每种晶体讲成人话,再动手拨一拨,判断键型、写电子式、比熔沸点这些高频送分题就稳了。

🗣️ 把物质想成一堆用不同"胶水"粘起来的积木: 原子之间靠化学键粘在一起。活泼金属遇上活泼非金属,一方把电子整个抢走,变成阴阳离子互相吸引 → 这就是离子键;两个原子谁也抢不动,只好共用一对电子 → 这就是共价键。
而一大块固体是靠什么"堆"起来的,决定了它硬不硬、熔点高不高、导不导电。四种堆法(离子晶体 / 分子晶体 / 共价晶体 / 金属晶体)背后是四种"胶水"。考试就考:给你一种物质或一组性质,你能反推它是哪种键、哪种晶体,再倒推出它的其它性质。抓住"键决定类别、类别决定性质"这一条主线,这一章就通了。
本包导航: ① 离子键与共价键 → ② 化学键与物质类别 → ③ 电子式与结构式 → ④ 四大晶体类型 → ⑤ 熔沸点比较规律 → ⑥ 全题型地图 → ⑦ 易错集 → ⑧ 尖子生拔高 → ⑨ 自测。每节都配可拨动的可视化。

① 离子键与共价键

化学键是相邻原子(或离子)之间强烈的相互作用。中学主要看两种:离子键、共价键。

离子键

定义: 阴、阳离子之间通过静电作用形成的化学键。一个原子把电子完全给出、另一个完全得到,变成带电离子,再相互吸引。

成键对象: 一般是活泼金属(ⅠA、ⅡA)与活泼非金属(ⅥA、ⅦA)之间。如 Na 与 Cl → NaCl。

关键判据: 含有离子键的化合物 = 离子化合物。只要物质里存在离子键,它就是离子化合物(可能同时还含共价键)。

共价键

定义: 原子之间通过共用电子对形成的化学键。谁也抢不走电子,就一起用。

共价键分类成键原子电子对是否偏移例子
非极性键同种元素原子不偏移(共用对居中)H—H、Cl—Cl、N≡N、O=O
极性键不同种元素原子偏向吸引力强的一方H—Cl、O—H、C=O
关键判据: 只含共价键的化合物 = 共价化合物(如 HCl、H₂O、CO₂)。注意:共价化合物里一定没有离子键。

⚠ 同种元素 → 非极性键;不同种元素 → 极性键。判断"极性/非极性键"看的是成键两原子是否同种,不要和"分子极性"混为一谈。

交互 1 · 离子键 vs 共价键 是怎么形成的

点一种成键方式,看电子的去向:离子键是电子整个转移(Na 给出、Cl 得到,变成 Na⁺ 和 Cl⁻ 互相吸);共价键是两原子共用一对电子(谁也不放手)。

交互 2 · 键的极性判断(同种 / 异种原子)

选一根键,看共用电子对是居中(非极性键,同种原子)还是偏向一边(极性键,异种原子)。偏移箭头指向吸引电子能力强的原子。

② 化学键与物质类别

把"含什么键"和"是什么物质"对应起来,是本章最核心的一张表:

物质含离子键?含共价键?类别
Na、Fe、金刚石(单质)无金刚石有;金属靠金属键单质
NaCl、KBr有无离子化合物
NaOH有(Na⁺与OH⁻间)有(O—H)离子化合物
Na₂O₂有(Na⁺与O₂²⁻间)有(O—O 非极性键)离子化合物
NH₄Cl有有(N—H)离子化合物
HCl、H₂O、CO₂无有共价化合物
一句话总纲: 有离子键 → 离子化合物(哪怕同时有共价键);只有共价键 → 共价化合物。判类别只看"有没有离子键"这一件事。

⚠ 离子化合物里可能含共价键(如 NaOH、Na₂O₂、NH₄Cl),但共价化合物里绝不含离子键。别把"含共价键"当成"是共价化合物"。

③ 电子式与结构式

电子式: 用小圆点 · 或叉 × 表示原子最外层电子的式子,把成键的电子对、离子的电荷都画出来。

四类电子式写法要点

① 原子: 元素符号周围按最外层电子数点点。如 钠 Na· ,氯周围 7 个点。

② 简单离子: 阳离子直接写离子符号(丢了电子,不画电子),如 Na⁺;阴离子要画满电子并用 [ ] 括起、右上标电荷,如 [ ׃C̈l̤׃ ]⁻。

③ 离子化合物: 阴阳离子分开写,阴离子加方括号和电荷,阳离子不加括号。相同离子不合并简写(如 Na₂O₂ 两个 Na⁺ 要分列在过氧根两侧)。

④ 共价分子: 用共用电子对(一对点)连接原子,不写方括号、不写电荷。孤对电子也要画。

⑤ 结构式: 把每一对共用电子对换成一根短线 —,单键一根、双键两根、叁键三根;孤对电子不画。如 H—O—H、O=C=O、N≡N。

⚠ 高频扣分点:①离子化合物的阴离子忘加 [ ] 和电荷;②Na₂O₂、Na₂O 的两个 Na⁺ 被错误合并成 2Na⁺;③CO₂ 电子式写成离子式(它是共价分子,不带电荷、无方括号);④漏画孤对电子。

④ 四大晶体类型

同样是固体,内部"堆"法不同,性质天差地别。四种晶体 = 四种作用力:

⑤ 熔沸点比较规律

🗣️ 先看"晶体大类",再看"同类内部"。 熔化/沸腾要破坏的是微粒间作用力,谁的"胶水"最结实,谁的熔沸点就最高。

规律一:不同类晶体之间(一般规律)

共价(原子)晶体 > 离子晶体 > 分子晶体　(金属晶体跨度大,视具体金属)

共价键极强(要断很多共价键)→ 原子晶体熔点最高;离子键较强 → 离子晶体次之;分子间作用力最弱 → 分子晶体最低。

规律二:同类晶体内部比

① 离子晶体: 离子电荷越多、离子半径越小 → 离子键越强 → 熔点越高。如 MgO > NaCl(Mg²⁺、O²⁻ 电荷高、半径小)。

② 共价晶体: 原子半径越小、键越短越强 → 熔点越高。如 金刚石(C—C) > SiC > 晶体硅(Si—Si)。

③ 分子晶体: 组成结构相似时,相对分子质量越大 → 分子间作用力越大 → 熔沸点越高。如 I₂ > Br₂ > Cl₂ > F₂;HI > HBr > HCl。但有氢键的反常升高:HF > HCl,H₂O > H₂S(氢键使熔沸点异常偏高)。

⚠ 只有同类晶体、且结构相似,才能用"分子质量大→熔点高"。有氢键的分子(HF、H₂O、NH₃)会打破这个顺序,出现"反常"高点。

⑦ 易错集(高频翻车点)

1. NaOH 只含离子键?错。NaOH 既含离子键(Na⁺与OH⁻之间)又含共价键(O—H)。

2. Na₂O₂ 里没有共价键?错。Na₂O₂ 含离子键(Na⁺与O₂²⁻)又含非极性共价键(O—O)。同理 H₂O₂ 中也有 O—O 非极性键。

3. 离子化合物一定含离子键(对);但反过来"含共价键就是共价化合物"是错的——NaOH 含共价键却是离子化合物。判类别只认离子键。

4. 稀有气体是分子晶体,由单原子分子构成,内部没有化学键;固态靠微弱范德华力,熔沸点极低。

5. 共价化合物不一定是分子晶体!SiO₂、SiC 是共价化合物,但属于共价(原子)晶体,熔点极高。共价键既可存在于分子晶体,也可存在于共价晶体。

6. "分子间作用力"不是化学键。熔化分子晶体(如干冰升华)只破坏分子间作用力,不破坏分子内的共价键。

7. 金属能导电靠自由电子(物理过程);离子晶体导电必须熔融或溶解使离子能自由移动(伴随化学状态改变)。固态离子晶体不导电。

8. 判极性键 / 非极性键看成键两原子是否同种:O—O 非极性、O—H 极性。这与"分子有无极性"是两码事(CO₂ 含极性键却是非极性分子)。

⑧ 尖子生拔高视角

▸ 一条主线贯穿全章: 微粒 → 作用力 → 键类型 → 晶体类型 → 宏观性质。任何一道题,先定位它在这条链的哪一环,再顺/逆推。

▸ "熔化时破坏什么"是终极判据: 熔化只破坏分子间作用力 → 分子晶体;破坏离子键 → 离子晶体;破坏共价键 → 共价晶体。这一句能解绝大多数晶体类型判断题。

▸ 氢键的双重身份: 氢键(F、O、N 与 H 之间)比范德华力强、比化学键弱。它让 H₂O、HF、NH₃ 熔沸点反常升高,还解释冰的密度小于水、DNA 双螺旋。见到 F/O/N—H 就警惕氢键。

▸ 相似相溶与晶体类型: 极性溶质易溶于极性溶剂(NaCl 溶于水),非极性溶质易溶于非极性溶剂(I₂ 溶于 CCl₄)。这背后是分子极性与作用力匹配。

▸ 典型物质的"身份证"背熟: 干冰=分子晶体;SiO₂/金刚石/Si/SiC=共价晶体;NaOH/Na₂O₂=离子化合物且双键型;稀有气体=单原子分子晶体无键。这些是命题高频"陷阱库"。
