金属的冶炼 · 镁铜等金属

自然界里绝大多数金属都以化合态（正价）存在，冶炼就是想办法把它们还原成 0 价的单质。金属越活泼、越难还原，用的方法就越"暴力"。把这条"活动性→方法"的线索理顺，再吃透镁、铜两块常考金属，选择填空就能稳拿分。

🗣️ 冶炼的本质只有一句话：把化合态的金属离子还原成金属单质（化合价从正价降到 0）。
金属越活泼，它的离子越"抱得紧"、越难被抢回电子，就得用越强的手段。于是有了一条鄙视链：最活泼的（K、Ca、Na、Mg、Al）只能用电"硬拽"（电解法）；中等活泼的（Zn、Fe、Cu）请个"还原剂中间商"（C、CO、H₂、Al）来抢电子（热还原法）；很不活泼的（Hg、Ag）加把火烤一烤就分解出来（热分解法）；至于金、铂这种"躺平不反应"的，直接从矿石里挑出来（物理富集）。一句话：越活泼→越靠电解，越不活泼→越省事。
本包导航： ① 冶炼原理（还原化合态金属） → ② 四种冶炼方法 → ③ 活动性顺序与方法对应 → ④ 镁 Mg 的特性 → ⑤ 铜 Cu 及其化合物 → ⑥ 金属通性与合金 → ⑦ 全题型地图 → ⑧ 易错集 → ⑨ 尖子生拔高 → ⑩ 自测。每节都配可拨动的可视化。

① 金属冶炼的原理

Mⁿ⁺ + n·e⁻ → M（化合态 → 单质，得电子被还原）

核心：金属在矿石里几乎都是正价（化合态），如 Fe₂O₃ 中 Fe 为 +3、CuSO₄ 中 Cu 为 +2。冶炼就是给这些金属离子补上电子，把它还原为 0 价单质。所以冶炼过程金属一定被还原，是还原过程。

依据金属活动性选方法：金属越活泼（越靠活动性顺序前面），失电子能力越强、其阳离子得电子（被还原）越难，越需要强还原手段。故"选方法"本质是"看这个金属有多难还原"。

金属活动性阳离子被还原的难度需要的手段
很活泼（K…Al）极难还原电解（最强还原：阴极电子）
中等（Zn、Fe、Cu）较易还原热还原（C、CO、H₂、Al）
不活泼（Hg、Ag）很易还原热分解（加热即可）
极不活泼（Au、Pt）本就是游离态物理富集（不需还原）

⚠ 冶炼＝还原，任何"氧化金属单质"的说法都错。判断题里"金属冶炼是把金属离子氧化为单质"是典型陷阱。

② 四种冶炼方法

① 电解法（对付很活泼金属 K、Ca、Na、Mg、Al）

电解熔融（不是溶液！）的氯化物或氧化物，阴极得电子析出金属。

2NaCl（熔融）--电解--> 2Na + Cl₂↑
MgCl₂（熔融）--电解--> Mg + Cl₂↑
2Al₂O₃（熔融）--电解--> 4Al + 3O₂↑　（加冰晶石助熔）

⚠ 冶炼 Al 电解 Al₂O₃ 而非 AlCl₃——AlCl₃ 是共价化合物，熔融不导电。冶炼 Na、Mg 用氯化物（熔点低更省能）。电解要"熔融"不能用水溶液（活泼金属阳离子在水中放电轮不到它）。

② 热还原法（对付中等活泼金属 Zn、Fe、Cu）

用还原剂 C、CO、H₂、Al 在高温下夺走金属氧化物里的氧。

Fe₂O₃ + 3CO --高温--> 2Fe + 3CO₂　（高炉炼铁）
2CuO + C --高温--> 2Cu + CO₂↑
WO₃ + 3H₂ --高温--> W + 3H₂O　（H₂ 还原，产物纯）
2Al + Fe₂O₃ --高温--> 2Fe + Al₂O₃　（铝热反应，放热焊接钢轨）

⚠ 铝热反应用来冶炼比 Al 不活泼的高熔点金属（Fe、Cr、Mn、V 等），放出大量热使金属熔化。Al 自己是电解冶炼，不能用铝热反应"炼铝"。

③ 热分解法（对付不活泼金属 Hg、Ag）

这些金属的氧化物不稳定，直接加热就分解出金属，无需还原剂。

2HgO --Δ--> 2Hg + O₂↑
2Ag₂O --Δ--> 4Ag + O₂↑

④ 物理富集（对付极不活泼金属 Au、Pt）

金、铂在自然界以游离态（单质）存在，不用还原，靠"淘金""重力/浮选"等物理方法富集分离即可。

④ 镁 Mg 的特性

Mg 是活泼金属，冶炼用电解熔融 MgCl₂。它化学性质活泼，还原性强，能和很多"看似不活泼"的物质反应。

① 与非金属燃烧（还原性强）

2Mg + O₂ --点燃--> 2MgO　（耀眼白光，闪光弹）
3Mg + N₂ --点燃--> Mg₃N₂

② 与 CO₂ 燃烧——重点！

2Mg + CO₂ --点燃--> 2MgO + C

Mg 还原性太强，能把 CO₂ 里的碳都还原出来（黑色 C 附在生成物上）。所以镁着火不能用 CO₂ 灭火器，会越烧越旺；也不能用水（Mg 与水反应）。应用干燥沙土覆盖灭火。

③ 与酸、与水

Mg + 2HCl --> MgCl₂ + H₂↑
Mg + 2H₂O --Δ--> Mg(OH)₂ + H₂↑　（热水中缓慢）

⚠ 高频陷阱：① 认为 CO₂ 一定能灭火——对 Mg 不行！② 认为 Mg 在 CO₂ 中不能燃烧——恰恰能，且生成 C。③ 镁条燃烧不能用玻璃棒/坩埚钳直接观察强光会伤眼。

交互 5 · 镁在 CO₂ 中燃烧

点"点燃"，看镁条在充满 CO₂ 的集气瓶中剧烈燃烧，瓶壁出现黑色碳，说明 CO₂ 也能被 Mg 还原——故不可用 CO₂ 灭镁火。

Mg 条静置在 CO₂ 集气瓶中——点燃看它是否能烧起来。

⑤ 铜 Cu 及其化合物

Cu 是不活泼金属（排在 H 后），冶炼用热还原（2CuO+C→2Cu+CO₂↑）或火法炼铜。它不与稀盐酸/稀硫酸反应（排在 H 后置换不出 H₂），但能与氧化性酸反应。

① 与硝酸（氧化性）

Cu + 4HNO₃(浓) --> Cu(NO₃)₂ + 2NO₂↑ + 2H₂O　（红棕气）
3Cu + 8HNO₃(稀) --> 3Cu(NO₃)₂ + 2NO↑ + 4H₂O　（无色遇空气变红棕）

② 与浓硫酸（加热）

Cu + 2H₂SO₄(浓) --Δ--> CuSO₄ + SO₂↑ + 2H₂O

③ Cu²⁺ 盐与铜绿

Cu²⁺ 在溶液中显蓝色；CuSO₄ 无水粉末白色，遇水变蓝（可检验水）。铜在潮湿空气中生锈生成铜绿（碱式碳酸铜）：

2Cu + O₂ + H₂O + CO₂ --> Cu₂(OH)₂CO₃　（绿色，铜绿/铜锈）
CuSO₄ + 5H₂O = CuSO₄·5H₂O　（蓝色胆矾/晶体）

⚠ Cu 不能与稀 H₂SO₄、稀 HCl 反应放 H₂（排 H 后）；但Cu 能溶于稀 HNO₃、浓 HNO₃、热浓 H₂SO₄（这些是氧化性酸，靠 +5 价 N 或 +6 价 S 氧化 Cu，不是靠 H⁺）。

⑥ 金属的通性与合金

金属通性（物理）

多数金属有金属光泽、导电、导热、延展性（自由电子）。密度、硬度、熔点差别大：最软 Na/K（刀切）、最重 Os、熔点最高 W（灯丝）、熔点最低 Hg（常温液态）、导电最好 Ag>Cu>Au>Al。

金属通性（化学）——还原性

金属都易失电子作还原剂：与 O₂、与酸、与盐溶液（活泼金属置换不活泼金属）反应。活动性越前，还原性越强。

Fe + CuSO₄ --> FeSO₄ + Cu　（湿法炼铜，铁置换铜）

合金

合金＝金属＋（金属或非金属）熔合成的具有金属特性的物质。合金硬度一般更大、熔点一般更低（比成分金属）。常见：生铁/钢（Fe+C）、青铜（Cu+Sn）、黄铜（Cu+Zn）、硬铝（Al+Cu/Mg…）、焊锡（Sn+Pb）。

⚠ 合金熔点低于各成分金属——这是选择常考结论（合金比纯金属易熔）；合金是混合物，不是化合物。

⑧ 易错集（高频翻车点）

1. 只要活泼金属就只能电解法（K、Ca、Na、Mg、Al）——它们不能用 C、CO、H₂ 还原，也不能热分解。

2. 冶炼 Al 电解 Al₂O₃，不电解 AlCl₃——AlCl₃ 是共价化合物，熔融不导电。

3. 电解必须"熔融"，不能用水溶液冶炼活泼金属（水中 H⁺/H₂O 先放电，得不到活泼金属）。

4. Mg 能在 CO₂ 中燃烧（2Mg+CO₂→2MgO+C），故镁火不能用 CO₂ 灭，要用沙土。

5. 冶炼是还原过程（金属离子得电子→单质），不是氧化过程。

6. Cu 排在 H 后，不与稀盐酸/稀硫酸反应放 H₂；但能与硝酸、热浓硫酸（氧化性酸）反应。

7. 铝热反应炼的是比 Al 不活泼的高熔点金属（Fe、Cr…），不能用铝热反应"炼铝"。

8. 合金熔点低于成分金属、硬度一般更大；合金是混合物。

⑨ 尖子生拔高视角

▸ 一条主线看穿全部：冶炼难度＝金属阳离子被还原的难度＝金属活动性。越活泼→越难还原→还原手段越强（电解 > 热还原 > 热分解 > 不用还原）。选方法先问『这金属在活动性表哪一段』。

▸ 还原剂强弱有序：电解阴极（相当于"无限强"还原剂）> 活泼金属（Al 热剂）> H₂/C/CO。所以 Al 能还原 Fe₂O₃，但 CO 还原不了 Al₂O₃。

▸ 用"氧化还原"统一表达：无论哪种方法，金属都是被还原（化合价降低、得电子）。写电子转移时，还原剂 C/CO/H₂/Al 失电子（升价），金属离子得电子（降价至 0）。

▸ 能耗与经济性：电解耗电最贵→只对最活泼金属用；热还原（CO 高炉炼铁）成本低→大规模用于铁；H₂ 还原产物纯但贵、危险，只在需高纯（W、Mo）时用。这是工业流程题的答题逻辑。

▸ 镁铜对照记忆：Mg 活泼（电解冶炼、能烧 CO₂、与酸放 H₂）；Cu 不活泼（热还原冶炼、不与稀酸放 H₂、只被氧化性酸溶）。一活一惰，正好覆盖两端典型。
