遗传两大定律全攻坚

分离定律、自由组合定律、显隐判断、测交、性状分离比、9:3:3:1 及各种变式、致死、概率计算、配子种类数——把孟德尔的两条定律拆成"棋盘格 + 拆分相乘"两招,遗传大题一路碾过去。

🗣️ 两大定律(人话版): 每个性状由一对基因管,这对基因住在一对同源染色体上。分离定律说的是:产生配子时,这一对基因会"分家"——各进一个配子,谁也不粘着谁(所以杂合子 Aa 产生 A 和 a 各一半)。自由组合定律说的是:当你同时管两对性状(如 Aa 和 Bb)、它们住在不同的同源染色体上时,这两对"分家"互不干扰,A/a 和 B/b 随机搭配。
做题的万能口诀:先按每一对基因单独用分离定律算清楚(如 Aa×Aa 后代 3/4 显、1/4 隐),再把几对的结果乘起来——所有 9:3:3:1、致死、概率题都是这么来的。

① 分离定律:一对相对性状

孟德尔用纯种高茎 × 纯种矮茎豌豆做杂交,发现 F1 全高、F1 自交后 F2 高矮之比约 3:1。由此提出显隐性、性状分离和分离定律。

P: 高茎(纯合 DD) × 矮茎(纯合 dd)
F1: 全部高茎(Dd)——显示 D 对 d 为显性
F1 自交 Dd × Dd →
F2: 基因型 1 DD : 2 Dd : 1 dd　表现型 3 高 : 1 矮

四个核心概念

概念含义
显性 / 隐性F1 表现出来的是显性性状(高),被掩盖的是隐性性状(矮)
性状分离杂合 F1 自交,后代同时出现显性和隐性(3:1),叫性状分离
纯合子 / 杂合子DD、dd 为纯合(自交不分离);Dd 为杂合(自交会分离)
等位基因位于一对同源染色体相同位置、控制同一性状的一对基因(D 与 d)

⚠ 3:1 是大量后代的统计比例,样本太少不一定出现。且必须是"一对基因、完全显性、后代成活率相同"才严格 3:1。

① 棋盘格:一对相对性状 → 3:1

选父母双方基因型,看 Punnett 棋盘格自动填格、统计基因型与表现型比。设 D=显性(高)、d=隐性(矮)。

🎯 自编练习题 · 非真题(待替换为真题)

【题】豌豆的高茎对矮茎为显性(受一对等位基因 D、d 控制)。现有一株高茎豌豆,不知其基因型,让其自交,后代共长出 320 株豌豆,其中矮茎 79 株。（1）该高茎豌豆的基因型最可能是什么？（2）若让该高茎豌豆与矮茎豌豆杂交,预期后代表现型及比例是什么？

【解析】(1) 后代出现性状分离（矮茎 79/320 ≈ 1/4，接近 1:3 中的 1），说明该高茎豌豆为杂合子 Dd：Dd 自交 → 1 DD : 2 Dd : 1 dd，表现型 3 高 : 1 矮，与 79:320≈1:4 的矮茎比例吻合。（2）Dd（高茎）× dd（矮茎）为测交：Dd 产生 D、d 两种配子各 1/2，dd 只产生 d 配子，后代 Dd(高茎) : dd(矮茎) = 1 : 1。

② 显隐判断 · 性状分离 · 测交

怎样判断谁是显性?

▸ 杂交法:两纯合亲本杂交,F1 表现出来的那个性状是显性。
▸ 自交/自由交配法:某性状群体内自交,若后代出现性状分离(新性状),则亲本表现的是显性、新出现的是隐性。
「若 A×A 后代出现 a,则 A 显 a 隐」。

测交:验证 F1 的基因型

测交 = 待测个体 × 隐性纯合子(dd)。因为隐性纯合子只能提供 d 配子,后代表现型直接"读出"待测个体产生的配子种类和比例。

若 F1 是 Dd: Dd × dd → 1 Dd(高) : 1 dd(矮) => 1:1
若待测是 DD: DD × dd → 全部 Dd(高) => 全显性

⚠ 测交对象必须是隐性纯合子;后代比 1:1 说明待测为杂合 Dd,后代全显说明待测为纯合 DD。

② 测交演示:F1 × aa → 1:1

选待测个体基因型,与隐性纯合 dd 测交,看后代表现型比例如何"暴露"待测基因型。

🎯 自编练习题 · 非真题(待替换为真题)

【题】番茄的红果对黄果为显性（受等位基因 R、r 控制）。为鉴定一株表现型为红果的番茄的基因型，最简便、最可靠的方法是什么？请写出两种可能的杂交组合及预期结果。

【解析】番茄是自花传粉植物，最简便的方法是让该红果番茄自交（⊗），观察后代是否发生性状分离：①若自交后代全为红果，则该植株为纯合子 RR；②若自交后代出现红果 : 黄果 = 3 : 1（性状分离），则该植株为杂合子 Rr（Rr×Rr → 1RR:2Rr:1rr，表现型 3红:1黄）。也可用测交（该红果 × 黄果 rr）鉴定：若后代红果:黄果=1:1 则为 Rr，若后代全红果则为 RR——但自交操作更简便，故优先选自交法。

③ 分离定律的实质

实质:在减数第一次分裂后期,同源染色体分开进入不同的次级性母细胞;位于这对同源染色体上的等位基因(D 与 d)也随之分开,分别进入不同的配子中。
结果:杂合子 Dd 产生 D、d 两种配子,比例 1:1。

▸ 分离发生的时间:减数第一次分裂后期(同源染色体分离时)。
▸ 分离的主体:一对等位基因随一对同源染色体的分开而分离。
▸ 一对基因、一对同源染色体、一对性状是分离定律研究的"三个一"。

④ 自由组合定律:两对相对性状

孟德尔用两对相对性状(黄圆 × 绿皱)豌豆杂交,F1 全黄圆,F1 自交后 F2 出现 黄圆:黄皱:绿圆:绿皱 = 9:3:3:1。

P: 黄圆(YYRR) × 绿皱(yyrr)
F1: 黄圆(YyRr)
F1 自交 YyRr × YyRr → F2:
9 黄圆(Y_R_) : 3 黄皱(Y_rr) : 3 绿圆(yyR_) : 1 绿皱(yyrr)

拆成两对分离定律来看

黄:绿 = 3:1(Yy×Yy),圆:皱 = 3:1(Rr×Rr)。两对独立,乘起来:
(3黄:1绿) × (3圆:1皱) = 9黄圆 : 3黄皱 : 3绿圆 : 1绿皱。9:3:3:1 就是 (3:1)×(3:1) 的展开。

⚠ 自由组合的前提:两对等位基因位于两对不同的同源染色体上。若在同一对同源染色体上则连锁,不适用自由组合定律。

🎯 自编练习题 · 非真题(待替换为真题)

【题】豌豆的黄色子叶对绿色子叶为显性（Y/y），圆粒对皱粒为显性（R/r），两对基因独立遗传。现让基因型为 YyRr 的豌豆自交，求子代中：（1）黄色皱粒（Y_rr）所占比例；（2）纯合子所占比例；（3）与亲本表现型完全相同（黄色圆粒 Y_R_）所占比例。

【解析】YyRr 自交可拆成两对分离定律分别算再相乘：Yy×Yy → 黄:绿=3:1（Y_=3/4，yy=1/4）；Rr×Rr → 圆:皱=3:1（R_=3/4，rr=1/4）。
（1）黄色皱粒 Y_rr = 3/4 × 1/4 = 3/16。
（2）纯合子（YYRR、YYrr、yyRR、yyrr 四种）：单对纯合概率各为 1/4（YY=1/4 或 yy=1/4），两对都纯合 = 1/4 × 1/4 = 1/16，纯合子共有 4 种基因型，同一株只能是其中一种，其概率之和 = 4 × 1/16 = 1/4（即 YY 或 yy 的概率 1/2，乘以 RR 或 rr 的概率 1/2，1/2×1/2=1/4）。
（3）黄色圆粒 Y_R_ = 3/4 × 3/4 = 9/16，对应经典 9:3:3:1 中的"9"。三问结果互相印证：9/16(Y_R_)+3/16(Y_rr)+3/16(yyR_)+1/16(yyrr)=16/16，比例自洽。

⑤ 自由组合定律的实质

实质:减数第一次分裂后期,同源染色体上的等位基因分离的同时,非同源染色体上的非等位基因自由组合。
结果:YyRr 产生 YR、Yr、yR、yr 四种配子,比例 1:1:1:1。

▸ 分离定律与自由组合定律同时发生、互不矛盾:等位基因(Y/y、R/r)各自分离,非等位基因(Y 与 R、y 与 r 等)之间自由组合。
▸ n 对独立的杂合基因 → 2ⁿ 种配子(YyRr → 2²=4 种)。

⑥ 配子种类与自由组合:n 对杂合 → 2ⁿ 种

选杂合基因对数 n,列出全部配子(树状分叉)并给出 2ⁿ。

⑥ 基因型与表现型 · 常见比例变式

基因型(如 Dd)是控制性状的基因组成;表现型(如高茎)是表现出来的性状。基因型 + 环境 → 表现型。同一表现型可能对应多个基因型(高茎 = DD 或 Dd)。

由 9:3:3:1 演化的常见变式(都来自两对基因)

比例来历(9:3:3:1 的合并)典型机理
1:2:1一对基因,不完全显性Aa 表现中间型(如红:粉:白)
2:13:1 中显性纯合(或某基因型)致死AA 致死 → Aa:aa = 2:1
9:79 : (3+3+1)两对基因均显性才出某表现,其余同一表现
9:6:19 : (3+3) : 1两对显性、单显、双隐三类表现型
15:1(9+3+3) : 1只要有一个显性基因即为一类,双隐另一类
12:3:1(9+3) : 3 : 1一对基因显性上位
13:3(9+3+1) : 3抑制作用

⚠ 见到 9:7 / 9:6:1 / 15:1 / 12:3:1 / 13:3 等,几乎都是 两对基因、9:3:3:1 的某种合并——先还原成 9:3:3:1 再分析。

🎯 自编练习题 · 非真题(待替换为真题)

【题】某植物花色由两对独立遗传的等位基因（A/a、B/b）控制：只有当基因型中同时含有显性基因 A 和 B 时才开紫花，其余基因型均开白花。现用纯合紫花植株与纯合白花植株（aabb）杂交得 F1，F1 自交得 F2。问 F2 中紫花:白花的比例是多少？F2 白花植株中基因型种类有几种？

【解析】纯合紫花必为 AABB（因为要同时含 A、B），AABB × aabb → F1 为 AaBb（紫花）。F1 自交 AaBb×AaBb，按 9:3:3:1 展开：9 A_B_ : 3 A_bb : 3 aaB_ : 1 aabb。因为只有 A_B_（同时含显性 A 和 B）才开紫花，其余 A_bb、aaB_、aabb 均为白花，合并为 3+3+1=7。
所以 F2 紫花:白花 = 9 : 7（典型的两对基因互补作用）。
白花植株对应的基因型：A_bb（AAbb、Aabb）、aaB_（aaBB、aaBb）、aabb 共 5 种基因型（2+2+1=5）。核对：9+3+3+1=16，与 F2 总份数吻合，比例自洽。

⑦ 致死变式计算器 · 概率相乘

某基因型个体在发育中死亡,会改变后代的表现型比例。做法:先按正常写出比例,再划掉致死的那一类,重新归一化。

④ 致死变式计算器(Aa×Aa,某基因型致死 → 重算)

Aa × Aa 正常后代 1 AA : 2 Aa : 1 aa。选一种基因型致死,看存活后代比例如何改变。

概率计算:拆分 → 分别算 → 相乘

核心思想:多对基因决定的表现型概率 = 各对基因单独(用分离定律)算出的概率连乘。
▸ AaBbCc × AaBbCc 求 aabbcc 概率:各对 aa/bb/cc 概率均为 1/4 → (1/4)³ = 1/64。
▸ 求某"至少含一个显性"用加法/减法:1 − (全隐概率)。

P(特定表现型) = P(第1对) × P(第2对) × … × P(第n对)

⑤ 概率相乘计算器(多对基因分别算再乘)

对每一对基因选一对亲本与所求后代表现型,程序按分离定律分别算出概率,再连乘得总概率。

🎯 自编练习题 · 非真题(待替换为真题)

【题】已知玉米的非甜(A)对甜(a)为显性、黄粒(B)对白粒(b)为显性，两对基因独立遗传。现有基因型为 AaBb 的玉米自交得 F1，其中基因型为 AA 的植株因隐性纯合致死基因连锁效应在幼苗期全部死亡（设 AA 致死，其余基因型均正常成活）。求成活的 F1 中：（1）非甜:甜的比例；（2）AaBb×AaBb 后代 aabb（甜、白）的概率。

【解析】（1）先只看 A/a 这一对：Aa×Aa → 正常比例 1 AA : 2 Aa : 1 aa。因 AA 致死，划去该类，存活个体重新归一化为 2 Aa : 1 aa，表现型 非甜:甜 = 2 : 1（这是"显性纯合致死"的经典结果，参见易错集②）。
（2）本问求的是 AaBb×AaBb 后代（未提及致死，按正常自由组合定律计算，两小问相互独立）：aa 的概率 = 1/4（Aa×Aa 隐性纯合概率），bb 的概率 = 1/4（Bb×Bb 隐性纯合概率），两对基因独立，按分离定律各自算出概率后相乘：P(aabb) = 1/4 × 1/4 = 1/16。核对：正常 AaBb 自交后代 9:3:3:1 中 aabb 对应"1"份，1/16 与该展开一致。

🎯 自编练习题 · 非真题(待替换为真题)

【题】人类白化病（受一对等位基因 A、a 控制，基因位于常染色体上）是隐性遗传病。一对表现正常的夫妇生了一个患白化病的孩子。请写出该家庭的遗传图解，并说明这对夫妇再生一个孩子患病的概率是多少。

【解析】父母表现正常，但孩子患病（白化，隐性性状），说明父母都携带隐性基因 a，即父母基因型均为 Aa（杂合子，"无中生有"必为隐性，双亲必为杂合子）。
遗传图解：
P　　　　　正常(Aa)　　×　　正常(Aa)
　　　　　　　　　↓ 杂交
F1　　　　1 AA : 2 Aa : 1 aa
　　　　　表现型　3 正常　:　1 白化
由于 Aa×Aa 属于分离定律的标准杂交，子代患病（aa）概率为 1/4。所以这对夫妇再生一个孩子患白化病的概率是 1/4，正常的概率是 3/4（其中正常又分纯合 AA 1/4 和携带者 Aa 2/4，2/4 相对 3/4 正常个体中携带者占 2/3）。

⑨ 全题型地图

题型 A:显隐性判断

触发给两纯合亲本杂交结果 / 群体自交出现新性状。

动作F1 出现的性状为显性;或"自交出现分离,亲本性状为显性、新性状为隐性"。

例:纯合甲×纯合乙 → F1 全为甲性状,则甲显乙隐。又若一群"甲"自交后代出现"乙",则甲显乙隐。

⚠ 陷阱:若无纯合前提,单看一次杂交不能定显隐,需自交验证。

题型 B:测交验证基因型

触发已知显性个体,要判断其为 DD 还是 Dd。

动作与隐性纯合 dd 测交(或让其自交):后代出现隐性 → 杂合 Dd;后代全显 → 纯合 DD。

例:某高茎豌豆测交,后代高:矮=1:1 → 该高茎为 Dd;若后代全高 → DD。

⚠ 陷阱:自交也能判(后代分离即杂合),但植物才方便自交;动物常用测交。

题型 C:9:3:3:1 及变式识别

触发两对相对性状杂交,F2 出现 16 分之的比例。

动作把观察到的比例还原为 9:3:3:1 的合并(见变式表),判断基因互作类型。

例:F2 = 9:7 → 说明只有 A_B_(双显)才表现某性状,即两对基因互补;9:6:1 → 双显 / 单显 / 双隐三型。

⚠ 陷阱:比例分母未必是 16;先看总份数是否 16 或其约数,判断是几对基因。

题型 D:致死分析

触发后代比例偏离标准(如 2:1、非 3:1),或题干提到某基因型不能存活。

动作先写标准比例,再划去致死类,重新归一化。

例:Aa×Aa,AA 致死 → 存活 2 Aa : 1 aa = 2:1。若两对基因各有一显性纯合致死,比例会更复杂,逐对处理再相乘。

⚠ 陷阱:配子致死 vs 合子(个体)致死不同;显性纯合致死常导致 2:1。

题型 E:概率计算(分离定律各对分开算再乘)

触发求 AaBbCc 类多对基因杂交后代某特定表现型/基因型概率。

动作拆成单对,用分离定律各算一个概率(如 A_ =3/4、aa=1/4),再相乘;"或"的情形用加法。

例:AaBbCc×AaBbCc,求 A_bbC_ 概率 = 3/4 × 1/4 × 3/4 = 9/64。

⚠ 陷阱:表现型概率用 3/4 与 1/4;基因型概率(如 Aa=1/2、AA=1/4)别混淆。

题型 F:配子种类数与自由组合

触发问某基因型个体能产生几种配子 / 后代基因型种类。

动作数杂合的对数 n,配子种类 = 2ⁿ;后代基因型种类 = 各对种类数相乘。

例:AaBbCC 有 2 对杂合(Aa、Bb)→ 2²=4 种配子(ABC/AbC/aBC/abC)。

⚠ 陷阱:纯合对(AA、CC)不增加配子种类;只数杂合对数。

⑩ 易错集

① 自由组合题先拆成单对,各用分离定律算概率,再相乘;不要直接背 9:3:3:1 硬套。

② 致死改变比例:先写标准比,再划掉致死类重新归一。显性纯合致死常见 2:1。

③ F2 的 3:1 / 9:3:3:1 是大量后代的统计比例,需足够多的个体才准;子代少时不能强求。

④ 正反交判断:正反交结果相同 → 基因在常染色体;正反交结果不同、后代随母本 → 细胞质遗传;与性别相关联 → 基因在性染色体(X)。

⑤ 表现型概率(3/4、1/4)与基因型概率(1/4、1/2、1/4)是两套,别用错。

⑥ 测交对象必须是隐性纯合子;后代 1:1 说明待测杂合,全显说明待测纯合。

⑦ 自由组合的前提是两对基因在不同的同源染色体上;同一对同源染色体上不适用。

⑪ 尖子生拔高视角

▸ "一对一乘"通法:任何多对基因遗传题,先把每一对基因当作独立的分离定律小问题算清(概率或比例),最后用乘法原理合并。9:3:3:1、致死、多基因概率全是这个套路的展开。

▸ 变式反推基因数:看 F2 总份数——若为 16 或其合并(9+7=16、12+3+1=16…)则两对基因;若为 4(3+1)则一对。总份数不是 4 的幂,可能有致死或连锁。

▸ 配子/合子致死的区分:配子致死影响的是产生配子的比例(改变输入);合子(个体)致死影响的是后代成活比例(划掉一格)。审题看"什么阶段死"。

▸ 棋盘格 vs 分支法:两对以内用棋盘格直观;三对以上用"分支(概率相乘)法"更快,避免画 64 格。高手默认用分支法。

▸ 自交与自由交配的区别:自交是同基因型个体相交(各自 ⊗);自由交配是群体内随机交配,需先算配子(基因)频率再组合,二者后代比例可能不同。
