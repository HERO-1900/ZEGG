硫及其化合物

硫单质 → SO₂ → SO₃ → 硫酸,一条清晰的价态升级链(−2 → 0 → +4 → +6)。这一章的分,全靠三根柱子:浓硫酸三大特性、SO₂ 的漂白与还原之分、SO₄²⁻ 检验排干扰。把每个反应讲成人话、再动手拨一拨,选择与实验题就稳了。

🗣️ 把硫想成一个"会升级的角色": 它最低是 −2 价(在 H₂S、金属硫化物里),中间是 0 价(硫单质),再往上 +4 价(SO₂、亚硫酸盐),最高 +6 价(SO₃、硫酸、硫酸盐)。
价态越低越想被氧化(还原性),价态越高越想氧化别人(氧化性),中间的 +4 价既能上又能下(两面派)。 考试就考:给你一个含硫物质,它这次是当"还原剂"往上升,还是当"氧化剂"往下降。抓住"低价还原、高价氧化、中价两头跑"这条主线,再背熟浓硫酸三性、SO₂ 漂白 vs 还原、SO₄²⁻ 检验三步,这一章就拿下了。
本包导航: ① 硫单质 S → ② 二氧化硫 SO₂(漂白性 + 还原性) → ③ 三氧化硫 SO₃ 与硫酸 → ④ 浓硫酸三大特性 → ⑤ SO₂ 与 CO₂ 鉴别 → ⑥ SO₄²⁻ 检验排干扰 → ⑦ 价态转化与酸雨。每节都配可拨动的可视化。

① 硫单质 S(0 价 · 两头都能跑)

俗称硫黄。淡黄色固体,不溶于水,微溶于酒精,易溶于 CS₂(二硫化碳)。硫处于 0 价,既有氧化性也有还原性。

核心反应(记住产物里硫是几价)

反应方程式说明
与铁(变价金属)Fe + S →(Δ) FeS只得 +2 价 FeS,不是 +3 价 → 说明硫氧化性较弱
与铜2Cu + S →(Δ) Cu₂S只得 +1 价 Cu₂S(而非 CuS)→ 再证硫氧化性弱
与氧气(燃烧)S + O₂ →(点燃) SO₂发出蓝紫色火焰,硫被氧化到 +4 价(还原性)
与氢气H₂ + S ⇌(Δ) H₂S硫作氧化剂,降到 −2 价

⚠ 硫的氧化性较弱:与变价金属反应只把金属氧化到低价(Fe→FeS 而非 Fe₂S₃,Cu→Cu₂S 而非 CuS)。对比氯气(强氧化性):Fe + Cl₂ → FeCl₃(直接到 +3 价)。这是常见考点。

📌 硫的清洗: 试管壁上的硫可用 CS₂ 或热的 NaOH 溶液洗去:3S + 6NaOH →(Δ) 2Na₂S + Na₂SO₃ + 3H₂O(硫的歧化)。

② 二氧化硫 SO₂(+4 价 · 明星考点)

物理性质: 无色、有刺激性气味、有毒气体,密度比空气大,易液化,易溶于水(1体积水溶约 40 体积 SO₂)。

(1) 酸性氧化物 — 与水生成亚硫酸

SO₂ + H₂O ⇌ H₂SO₃(中强酸,可逆)

与碱反应:SO₂ + 2NaOH → Na₂SO₃ + H₂O(碱足量);SO₂(过量) + NaOH → NaHSO₃。

(2) 漂白性 — 与"氯水漂白"本质不同(核心区分)

SO₂ 能与某些有色物质(如品红)结合生成不稳定的无色物质,使其褪色。这种漂白是可逆的:加热又恢复原来的颜色。

⚠ SO₂ 漂白 不能漂白酸碱指示剂(它使石蕊变红但不褪色);SO₂ 漂白具有选择性和可逆性。而氯水靠强氧化性把色素彻底破坏,是永久漂白、加热不恢复。二者原理完全不同。

(3) 还原性 — 使溴水 / 酸性 KMnO₄ 褪色(是氧化还原,不是漂白!)

+4 价硫想升到 +6 价,遇强氧化剂被氧化:

SO₂ + Br₂ + 2H₂O → H₂SO₄ + 2HBr　(使溴水褪色)
5SO₂ + 2KMnO₄ + 2H₂O → K₂SO₄ + 2MnSO₄ + 2H₂SO₄　(使酸性高锰酸钾褪色)

⚠ SO₂ 使溴水 / 酸性 KMnO₄ 褪色,是还原性(被氧化),不是漂白!这两者褪色不可逆、加热不恢复。别把"还原褪色"当成"漂白褪色"。

(4) 弱氧化性

遇更强还原剂时 SO₂ 也可作氧化剂:SO₂ + 2H₂S → 3S↓ + 2H₂O(硫从 +4 与 −2 归中为 0)。

(5) 检验 SO₂

用品红溶液:通入 SO₂ 后品红褪色,加热又恢复红色 → 证明是 SO₂。

交互 1 · SO₂ 漂白(可逆)vs 氯水漂白(永久)

给两杯品红溶液分别通 SO₂ 与氯水,都褪色。点"加热":只有 SO₂ 那杯恢复红色,氯水那杯依旧无色——这就是"可逆 vs 永久"的命门。

两杯都是红色品红溶液。分别通气使其褪色,再加热看谁恢复。
交互 3 · SO₂ 还原性:使溴水 / KMnO₄ 褪色(氧化还原,非漂白)

选一种氧化剂,持续通入 SO₂。看颜色随反应逐渐消退——这是 +4 价硫被氧化成 +6 价(SO₄²⁻),不是漂白,加热也不会恢复。

反应进度:0%

③ 三氧化硫 SO₃ 与硫酸

三氧化硫 SO₃(+6 价 · 硫酸的酸酐)

常温下为无色固体(熔点低,标准状况下也可视作液态/固态),是硫酸的酸酐。与水剧烈反应放大量热:SO₃ + H₂O → H₂SO₄。

2SO₂ + O₂ ⇌ 2SO₃(催化剂、Δ,工业接触法)

稀硫酸 — 酸的通性

强酸,完全电离出 H⁺,具备:使指示剂变红、与活泼金属放 H₂、与碱中和、与碱性氧化物、与某些盐反应。此时起作用的是 H⁺,硫本身不变价。

如:Fe + H₂SO₄(稀) → FeSO₄ + H₂↑(生成 +2 价、放氢气);Zn + H₂SO₄(稀) → ZnSO₄ + H₂↑。

④ 浓硫酸三大特性(实验题高频)

🗣️ 浓硫酸三兄弟:吸水、脱水、强氧化。 吸水是"抢现成的水分子"(当干燥剂);脱水是"把化合物里的 H、O 按 2:1 抢出来变成水"(把有机物烧黑碳化);强氧化是"+6 价硫亲自出马抢电子"(与铜、碳反应放 SO₂)。三者别混。

特性一 · 吸水性(物理过程,当干燥剂)

吸收气体或物质中的游离水分子,自身不变。可干燥 H₂、O₂、CO₂、SO₂、HCl、Cl₂ 等,不能干燥碱性气体 NH₃ 和还原性气体 H₂S(会反应)。

特性二 · 脱水性(化学过程,碳化)

按 H:O = 2:1 从有机物里"夺"出氢氧变成水,留下黑色的碳。经典现象:滴到蔗糖上变黑膨胀成"黑面包"、使纸张/木条炭化变黑。

C₁₂H₂₂O₁₁ →(浓H₂SO₄) 12C + 11H₂O　(蔗糖脱水碳化)

特性三 · 强氧化性(+6 价硫抢电子)

① 与铜加热(经典方程):

Cu + 2H₂SO₄(浓) →(Δ) CuSO₄ + SO₂↑ + 2H₂O

② 与碳加热:C + 2H₂SO₄(浓) →(Δ) CO₂↑ + 2SO₂↑ + 2H₂O。

③ 常温使 Fe、Al 钝化: 冷的浓硫酸使铁、铝表面生成致密氧化膜而停止反应。钝化是化学变化,不是"不反应"。所以可用铁/铝罐车装运冷浓硫酸。

⚠ 稀硫酸与金属反应靠 H⁺ 放 H₂;浓硫酸与金属(如铜)靠 +6 价硫的强氧化性放 SO₂,且不放氢气。别把浓、稀硫酸的产物搞混。加热到反应后期硫酸变稀,反应会停止。

⑤ SO₂ 与 CO₂ 的鉴别

两者都是无色酸性氧化物、都能使澄清石灰水变浑浊,所以不能只用石灰水区分。关键抓 SO₂ 独有的漂白性和还原性:

试剂SO₂CO₂能否鉴别
品红溶液褪色,加热恢复不褪色✅ 首选
溴水 / 酸性 KMnO₄褪色(被氧化)不褪色✅ 可(证还原性)
澄清石灰水变浑浊变浑浊❌ 不能区分

⚠ 若气体中同时含 SO₂ 和 CO₂,要检验 CO₂ 需先除尽 SO₂(用酸性 KMnO₄ 或溴水),再用品红验证 SO₂ 已除净,最后才用石灰水检 CO₂。顺序不能乱。

⑥ SO₄²⁻ 的检验(排干扰三步走)

标准步骤: 取待测液,先加足量稀盐酸酸化(观察无沉淀、无气体),再滴加 BaCl₂ 溶液,若产生白色沉淀(BaSO₄,不溶于稀硝酸/稀盐酸)→ 含 SO₄²⁻。

SO₄²⁻ + Ba²⁺ → BaSO₄↓(白色,不溶于稀盐酸/稀硝酸)

为什么第一步一定要先加盐酸?(排三类干扰)

干扰离子不排除的后果盐酸如何排除
CO₃²⁻ / SO₃²⁻也与 Ba²⁺ 生成白色沉淀(BaCO₃/BaSO₃)它们溶于盐酸放气,先加盐酸即除去
Ag⁺与 Cl⁻ 生成白色 AgCl 沉淀先加盐酸使 Ag⁺ 提前沉淀被排除/识别

⚠ 为什么不用稀硝酸 + Ba(NO₃)₂?因为若溶液含 SO₃²⁻,硝酸的强氧化性会把 SO₃²⁻ 氧化成 SO₄²⁻,造成"假阳性"误判!所以规范做法用盐酸 + BaCl₂。也不能只加 BaCl₂ 不加盐酸(排不掉 CO₃²⁻/SO₃²⁻/Ag⁺)。

⑦ 硫及化合物的价态转化 · 酸雨

硫元素价态转化主线(必背)

−2 (H₂S/FeS) ⇌ 0 (S) ⇌ +4 (SO₂/H₂SO₃) ⇌ +6 (SO₃/H₂SO₄)

向上升(失电子、还原性):S→SO₂、SO₂→SO₃、H₂S→S。向下降(得电子、氧化性):浓硫酸→SO₂、SO₂→S。+4 价的 SO₂ 是十字路口,既能被氧化上到 +6,也能被还原下到 0。

硫酸型酸雨的形成

燃煤/含硫燃料排放 SO₂ 进入大气,经两条途径变成硫酸,随雨水降下(pH < 5.6 即为酸雨):

途径一:SO₂ + H₂O → H₂SO₃,　2H₂SO₃ + O₂ → 2H₂SO₄
途径二:2SO₂ + O₂ →(粉尘催化) 2SO₃,　SO₃ + H₂O → H₂SO₄

危害: 腐蚀建筑与金属、酸化土壤与水体、危害植被。防治: 燃煤脱硫(加生石灰/石灰石)、开发清洁能源、尾气处理。

⚠ 新降酸雨的 pH 会随放置时间逐渐减小(更酸),因为 H₂SO₃ 被空气中 O₂ 继续氧化成更强的 H₂SO₄。

⑨ 易错集(高频翻车点)

1. SO₂ 漂白是可逆的(加热恢复),且不能漂白酸碱指示剂(只使石蕊变红不褪色)。氯水漂白是永久的。

2. SO₂ 使溴水 / 酸性 KMnO₄ 褪色是还原性(被氧化),不是漂白!加热不恢复。

3. 浓硫酸使 Fe、Al 钝化是化学变化(生成致密氧化膜),不是"不反应"。加热则会剧烈反应。

4. 检验 SO₄²⁻ 先加足量稀盐酸排除 CO₃²⁻ / SO₃²⁻ / Ag⁺ 干扰,再加 BaCl₂。不用硝酸(会把 SO₃²⁻ 氧化成 SO₄²⁻ 误判)。

5. 稀硫酸与金属放 H₂(靠 H⁺);浓硫酸与铜放 SO₂(靠 +6 价硫强氧化性),二者产物别混。

6. 硫氧化性较弱:Fe + S → FeS(+2),不是 Fe₂S₃;Cu + S → Cu₂S(+1),不是 CuS。

7. SO₂ 与足量石灰水都变浑浊,不能用石灰水区分 SO₂ 和 CO₂;要用品红。

8. 浓硫酸不能干燥 NH₃(碱性,会反应)和 H₂S(还原性,会被氧化)。

⑩ 尖子生拔高视角

▸ 用"价态-性质"一张表统摄全章: 低价(−2)只还原、高价(+6)只氧化、中价(0/+4)两头跑。任何含硫反应先标价态升降,氧化剂/还原剂立刻现形,配平也用电子守恒秒解。

▸ "漂白"要分四类机制: ① 吸附漂白(活性炭,物理);② 氧化漂白(氯水/H₂O₂/O₃/Na₂O₂,永久);③ 化合漂白(SO₂,可逆);④ 还原褪色(SO₂ 使溴水/KMnO₄ 褪色,本质是氧化还原,不算漂白)。看清 SO₂ 一物身兼③④两种"褪色"是拉开差距的点。

▸ 浓硫酸与金属反应的动态视角: 随反应进行硫酸被消耗、变稀,后期铜与稀硫酸不再反应,故铜不能被浓硫酸"吃干净"。这解释了很多定量实验里"剩余金属/气体量小于理论"的现象。

▸ SO₄²⁻ 检验的逻辑闭环: 目的是排除一切"会与 Ba²⁺ 生成白沉淀"或"会被氧化生成 SO₄²⁻"的干扰。先盐酸(排 CO₃²⁻/SO₃²⁻/Ag⁺、且不引入氧化性)是这条逻辑的最优解。理解"为什么不用硝酸"比记住步骤更重要。

▸ 尾气处理与绿色化学: 含 SO₂ 尾气常用 NaOH 溶液吸收(SO₂ + 2NaOH → Na₂SO₃ + H₂O),既防污染又回收。把"环境-资源"视角带进实验设计题能拿全分。
