古代诗歌鉴赏 · 表达技巧

「表达技巧」是古诗鉴赏里最大的一块失分区——因为它其实是四家人合住一个屋檐:修辞手法、表达方式、表现手法、结构技巧。再加上小而精的炼字炼句。这一页把四大类分门别类,给每一型配上「怎么认→怎么答→带解例题→哪里踩坑」的固定套路。答题永远是一条链:指出手法 → 结合诗句 → 分析效果与情感。

🗣️ 表达技巧(人话版): 这类题问的是「诗人用了什么手法,好在哪里」。别一上来就背名词。先分清是哪一家:修辞(比喻/拟人/夸张…,句子层面的花样)、表达方式(描写/抒情…,写作的大方式)、表现手法(衬托/对比/象征…,构思的巧劲)、结构技巧(照应/伏笔/以景结情…,谋篇布局)。答题=准确点出手法名+引用/概括诗句+说清效果和抒发的情感。三步缺一不可,只报手法名=只拿一半分。
一条万能答题链: 手法 + 诗句 + 效果 + 情感。① 先亮出手法名(用了什么);② 结合具体诗句说它怎么用的;③ 分析它带来的表达效果(生动/含蓄/画面感/对比强烈…);④ 落到诗人的思想感情。任何表达技巧题,把这四格填满,分就稳了。

① 修辞手法:句子层面的花样

修辞是最小、最好认的一类,着眼于一句或几句的语言加工。高考高频八种:

修辞特征 / 效果诗例
比喻以彼物比此物,化抽象为具体、生动形象「问君能有几多愁,恰似一江春水向东流」
拟人把物当人写,赋予情态、亲切生动「感时花溅泪,恨别鸟惊心」
夸张故意放大/缩小,突出强调、抒发强烈情感「白发三千丈,缘愁似个长」
对偶结构对称、音韵和谐、凝练工整「无边落木萧萧下,不尽长江滚滚来」
借代以局部/特征代整体,含蓄、形象「知否知否,应是绿肥红瘦」(红=花,绿=叶)
双关一语双意(谐音/语义),含蓄委婉「东边日出西边雨,道是无晴却有晴」(晴/情)
用典引前人事/语,以少胜多、含蓄深沉「廉颇老矣,尚能饭否」(辛弃疾)
互文上下文义互相补充、渗透,凝练含蓄「秦时明月汉时关」(=秦汉时的明月和关)

⚠ 比喻和拟人最易混:比喻是「A像B」(有本体喻体);拟人是「让物做人做的事」(有人的动作情态)。「花溅泪」是花在流泪(拟人),不是把花比作泪。

② 表达方式:写作的四种大方式

表达方式只有四种:记叙、描写、抒情、议论。古诗里最重要的是描写和抒情,考点几乎都在这两块。

描写:怎么写景状物

描写角度要领诗例
动静结合以动衬静 / 动静相映,画面有生气「明月松间照,清泉石上流」
虚实结合眼前实景 + 想象/回忆/梦境的虚景「今宵酒醒何处?杨柳岸,晓风残月」
远近高低视角变换,层次分明、立体开阔「两个黄鹂鸣翠柳,一行白鹭上青天」
视听结合色彩(视)+声音(听),多感官立体「明月别枝惊鹊,清风半夜鸣蝉」
细节描写抓典型细节,以小见真情「临行密密缝,意恐迟迟归」

抒情:怎么表达情感

抒情方式要领诗例
直接抒情(直抒胸臆)不加掩饰,直接喊出情感「安得广厦千万间,大庇天下寒士俱欢颜」
借景抒情(寓情于景)情融于景,景中含情「枯藤老树昏鸦…断肠人在天涯」
托物言志借某物象征人格/志向「零落成泥碾作尘,只有香如故」(咏梅)
用典抒情借典故含蓄地寄托情志「怀旧空吟闻笛赋,到乡翻似烂柯人」

⚠ 「借景抒情」是情借景发(景是手段,情是主);「托物言志」是物即人格化身(物贯穿全篇,象征志向)。写景抒情的多是即时之情,咏物言志的多是长久之志。

③ 表现手法:构思上的巧劲

表现手法是着眼于全篇或整体构思的技巧,比修辞更「大」。常考:

表现手法要领 / 效果诗例
衬托(正衬/反衬)以次要事物托主要事物(乐景衬哀、以动衬静)「蝉噪林逾静,鸟鸣山更幽」(以动衬静)
对比两相对照,反差突出、爱憎分明「朱门酒肉臭,路有冻死骨」
象征以具体物象征抽象品格/情思「大雪压青松,青松挺且直」(松=坚贞)
联想想象由此及彼、由实入虚,拓展意境「遥知兄弟登高处,遍插茱萸少一人」
欲扬先抑先贬后褒(或反),波澜起伏、印象深「闺中少妇不知愁…悔教夫婿觅封侯」
以小见大由细小事物折射宏大主题「一叶落而知天下秋」式的取景
白描不加渲染,寥寥数笔勾勒,质朴传神「枯藤老树昏鸦,小桥流水人家」

⚠ 衬托是「陪衬」(有主次,方向可同可反);对比是「对照」(无主次,双方并列反差)。「以动衬静」是衬托,「朱门酒肉臭 vs 路有冻死骨」是对比。

④ 结构技巧:谋篇布局

结构技巧关注诗篇的开头、过渡、结尾、前后勾连,常和「作用」题一起考。

结构技巧位置 / 作用诗例
首尾照应首尾呼应,结构完整、主旨强化「少小离家老大回」与「笑问客从何处来」
开门见山开篇直入主题,起势有力「国破山河在,城春草木深」
卒章显志结尾点明主旨、升华情感「安得广厦千万间…」(篇末言志)
以景结情结句写景,含蓄悠远、余味无穷「曲终人不见,江上数峰青」
铺垫伏笔前文蓄势/埋线,后文照应,顺理成章层层写景为末句抒情蓄势

⚠ 「以景结情」专指结尾用景收束(手段是结构+抒情双重);「借景抒情」可在任何位置。看到「末句写景有何妙处」,优先想以景结情。

⑤ 炼字:一个字的重量

炼字题问「某字用得好在哪」「能否换成别的字」。核心是词的表达效果:

字类常见效果诗例(题眼字)
动词化静为动、生动传神、一字活全句「红杏枝头春意闹」「云破月来花弄影」
形容词绘色状态、渲染氛围、化平为奇「大漠孤烟直,长河落日圆」
叠词音韵和谐、强化情态、增添韵律「寻寻觅觅,冷冷清清,凄凄惨惨戚戚」
数量词以少胜多、精准传情、凸显意境「前村深雪里,昨夜一枝开」
炼字答题三步:释义(该字什么意思/什么用法) → 描景(带入诗句描绘画面) → 情感效果(表达了什么、妙在哪)。

⚠ 炼字不能只夸「生动形象」四个字。必须落到具体画面和具体情感。「闹」字妙,妙在写出了杏花繁盛、春意盎然的热闹景象和喜悦之情——这才叫分析。

⑥ 炼句:赏析名句

炼句(名句赏析)是「表达技巧的综合题」。赏析一个句子,通常要从多角度切入,哪个明显答哪个:

切入角度看什么
修辞有没有比喻、拟人、对偶、夸张…
描写角度动静、虚实、远近、视听、色彩…
炼字句中有没有传神的动词/形容词
情感/意境营造了怎样的画面、寄托了什么情
炼句答题:点手法/角度 → 结合句子分析 → 描绘画面/意境 → 落到情感主旨。

示范:赏析「大漠孤烟直,长河落日圆」。① 视觉描写,「直」「圆」两个形容词炼字精准;② 写出大漠孤烟劲拔、黄河落日浑圆的壮阔画面,线条简净;③ 意境雄浑苍茫,寄寓诗人开阔的胸襟与漂泊边塞的孤寂。

⚠ 炼句最忌「只翻译不赏析」。把句子用白话说一遍不是赏析;必须点出用了什么、好在何处、抒了什么情。

⑪ 全题型地图

题型A — 修辞手法赏析题

触发信号:「运用了什么修辞手法」「从修辞角度赏析」「这句妙在何处(且句中有明显修辞)」。

标准动作:手法 + 诗句 + 效果 + 情感。准确点出修辞名 → 结合诗句说怎么用 → 分析效果(生动/含蓄/强烈) → 落到情感。

示例(带解):赏析「问君能有几多愁,恰似一江春水向东流」。答:运用比喻(设问兼比喻),将抽象的「愁」比作滚滚东流的一江春水,化无形为有形。效果是把愁绪之多、之绵长、之不可遏止写得可感可触。情感上抒发了亡国之君李煜绵绵不尽、深重难解的哀愁。

陷阱:只写「用了比喻」不分析;把比喻误判成拟人;漏掉情感落点。

题型B — 表达方式(描写/抒情)分析题

触发信号:「从描写角度分析」「这几句是怎样写景/抒情的」「情景关系如何」。

标准动作:先判断是描写还是抒情;描写→点动静/虚实/远近/视听;抒情→点直抒/借景/托物;再结合诗句析效果与情。

示例(带解):分析「明月松间照,清泉石上流」的写景手法。答:动静结合,上句明月照松是静景,下句清泉流石是动景,以动衬静。视听结合,月色是视觉,泉声是听觉。效果是勾勒出雨后山林清幽明净的画面。情感上寄托了诗人对山居生活的喜爱与超然物外的闲适。

陷阱:把「描写」当成「表现手法」硬套;只答一个角度(其实动静、视听可叠加);不落情感。

题型C — 表现手法(衬托/对比/虚实)题

触发信号:「运用了什么表现手法」「这首诗在构思上有何特点」「主要艺术手法是什么」。

标准动作:先在衬托/对比/象征/欲扬先抑/以小见大里定位 → 说清谁衬谁、谁对比谁 → 析反差/突出的效果 → 落情。

示例(带解):分析「朱门酒肉臭,路有冻死骨」的手法。答:运用对比,把权贵门内酒肉腐臭的奢靡,与门外路旁冻饿而死的白骨并置。效果是贫富悬殊、阶级对立的惨状被推到极致,触目惊心。情感上表达了杜甫对统治者的强烈愤慨和对民生疾苦的深切同情。

陷阱:衬托与对比混判;「以动衬静」误当对比;只报手法名不说「谁衬/对比谁」。

题型D — 炼字题

触发信号:「某字用得好,好在哪」「诗眼是哪个字,为什么」「能否把 X 换成 Y」。

标准动作:三步——释义(该字含义/活用) → 描景(带入句中描绘画面) → 情感效果(表达了什么、妙在何处)。换字题还要答「换了就失去了什么」。

示例(带解):「春风又绿江南岸」的「绿」好在哪?答:①「绿」本是形容词,此处用作动词(使动),意为「吹绿」。② 带入诗句,写出春风一到,江南两岸转眼草木新绿、生机盎然的动态画面。③ 一个「绿」字化静为动、以色显春,既写活了春的到来,又暗含诗人对故乡的思念与归情——若换「到/过/入」皆无此画面与情味。

陷阱:只夸「生动形象」不落画面情感;不点出词性活用;换字题不说「换了失去什么」。

题型E — 炼句 / 名句赏析题

触发信号:「赏析某一联/某一句」「这句历来为人称道,请分析」「结合全诗赏析尾句」。

标准动作:多角度切入(修辞/描写角度/炼字任选其显),点手法 → 结合句子析 → 描画面意境 → 落情感主旨。

示例(带解):赏析「无边落木萧萧下,不尽长江滚滚来」。答:① 对偶工整,「无边」对「不尽」,「萧萧」对「滚滚」,气势磅礴。② 视听结合,落叶纷飞是视觉,萧萧之声是听觉。③ 描绘出秋日夔州萧瑟辽阔、江流奔涌的壮阔画面。④ 情感上以壮景写悲情,抒发杜甫年老多病、漂泊无依的深沉悲慨与时光流逝之叹。

陷阱:只翻译不赏析;角度单一;没结合全诗情感(炼句要和主旨挂钩)。

题型F — 结构技巧题

触发信号:「这句在结构上有什么作用」「末句写景有何妙处」「开头/结尾有何特点」。

标准动作:定位结构技巧(首尾照应/开门见山/卒章显志/以景结情/铺垫伏笔) → 说其结构作用 → 兼顾内容/情感效果。

示例(带解):「曲终人不见,江上数峰青」作结的妙处?答:运用以景结情。曲已终、人已散,诗人不直抒惆怅,而以「江上数峰青」的空寂之景收束。结构上戛然而止、含蓄蕴藉;效果上留白悠远,把曲终人散的怅惘化入青山静水。情感上余味无穷,言有尽而意无穷。

陷阱:以景结情与借景抒情混;只答内容不答「结构作用」;忽略「含蓄/余味」这类结构性效果。

⑫ 易错集

① 比喻 vs 拟人:比喻是「A像B」(有本体喻体);拟人是「物做人的事」(有人的动作情态)。「感时花溅泪」是拟人不是比喻。

② 衬托 vs 对比:衬托有主次(陪衬主体,可正衬反衬);对比无主次(双方并列反差)。「以动衬静」是衬托,「朱门/路骨」是对比。

③ 只答手法不析效果:写「用了比喻」就停笔=只拿一半分。必须补「怎么用+什么效果+什么情感」。

④ 炼字不联系情感:光说「生动形象」是空话。要落到具体画面和诗人的具体情思。

⑤ 虚实/动静判断错:眼前所见为实,想象、回忆、梦境、未来为虚;有位移/声响变化的为动,凝止的为静。别把「想象的将来」当实景。

⑥ 借景抒情 vs 托物言志:前者情借景发(景为手段);后者物即人格(物贯全篇象征志向)。咏物诗多是托物言志。

⑦ 以景结情 vs 借景抒情:以景结情专指结尾用景收束;借景抒情可在任何位置。

⑧ 四大类张冠李戴:把「描写」(表达方式)答成「衬托」(表现手法),或把「对偶」(修辞)答成结构技巧。先归位,再选名。

⑨ 炼句只翻译:把句子用白话说一遍不是赏析。必须点手法、描意境、落情感。

② 炼字答题三步(背下来当模板): 释义(这个字什么意思、有无活用) → 描景(把字带进诗句,描绘它营造的画面) → 情感效果(它带来什么表达效果、寄托什么情感、妙在何处)。换字题在三步末尾再加一句「若换成 X 则失去了…」。

③ 判断优先级: 一句诗可能同时含多种手法。答题时把最突出、最典型的放前面,次要的补充。宁可少答而准,不要罗列一堆名词却都不分析。

④ 「效果词」库要熟: 生动形象、化抽象为具体、含蓄委婉、耐人寻味、画面感强、动静相映、意境开阔、以少胜多、余味无穷、反差强烈……这些是分析效果时的「零件」,但每个都要接上具体诗句,不能空贴。

⑤ 一切归于情感: 无论哪种表达技巧,最后一步永远是「抒发了/寄托了什么情感」。手法是船,情感是岸——答题没落到情感,等于船没靠岸。

⑭ 自测(先想,再点开)
