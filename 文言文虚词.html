文言文虚词

实词管「意思」,虚词管「关系和语气」。18 个高频虚词(之、而、以、于、其、为、则、乃、焉、者、所、因、且、若、所以……)本身几乎不表实义,却决定句子怎么连、语气怎么走。它们一词多用,考的从来不是「背释义」,而是看位置 + 看词性 + 看语气三步在具体语境里辨析。这一页把每个虚词的多种用法拆开、给例句、配可点的辨析工具。

🗣️ 文言虚词(人话版): 一个虚词像一个「多面插头」——同一个字,插在不同位置就是不同用法。辨析虚词不是猜,而是三看:①看位置(它前面后面是什么词?名词、动词、还是句末?)②看词性(它充当代词、助词、介词、还是连词?)③看语气(疑问、感叹、揣测、还是陈述?)。三看之后代入检验:把你判断的释义放回句子里,句子通不通、逻辑顺不顺,一试便知。
一句话方法: 虚词辨析 = 看位置 + 看词性 + 看语气 + 代入检验。永远先问「它前后各是什么成分」,再定它是代词/助词/介词/连词/语气词,最后把释义代回原句验证。「意义和用法是否相同」题的答法:意义(译成现代汉语一样吗)和用法(词性+语法作用一样吗)两者都相同才算相同。

① 高频虚词全景:一字多用是常态

虚词 → 看位置定词性 → 定用法 → 代入译句检验

虚词几乎不表实义,它的价值在「语法关系」和「语气」。同一个字在不同句子里可以是代词、助词、介词、连词、语气词。下表是最该先拿下的几个「多面手」:

虚词常见词性核心用法速记
之代词 / 助词 / 动词代词(他/它/这)、助词「的」、取独、宾语前置标志、凑音节、动词「到」
而连词 / 代词并列、承接、转折、修饰、递进(六种关系全靠语境)
以介词 / 连词介词(用/把/因为/凭借)、连词(来/因而/而)
于介词在、向、到、比、被(全看后面的成分)
其代词 / 语气副词代词(他的/其中)、语气副词(大概/难道/还是)
为动词 / 介词 / 语气词动词(是/做/当)、介词(替/被/因为)、句末语气词(呢)

⚠ 千万别「一个字只记一个意思」。看到「之」就译「的」、看到「以」就译「用」,是虚词题的头号死法。永远先看它在句中的位置和前后词性。

② 核心虚词精讲(附辨析方法)

「之」的六种用法

用法标志例句(带译)
代词动词后作宾语「爱之能勿劳乎」——之=他,作宾语
结构助词「的」名词/名词短语之间「小大之狱」——大大小小的案件
取消句子独立性主谓之间,不译「师道之不传也久矣」——主谓间取独
宾语前置标志宾语提到动词前「何陋之有」=有何陋,之是提宾标志
凑音节助词时间/动词后,无义「顷之」「久之」——只补足音节
动词「到、往」后接处所「辍耕之垄上」——之=到

「而」的五种关系

关系译法例句
并列又…又/不译「蟹六跪而二螯」——并列
承接就/接着「登高而招」——先登后招
转折却/但是「青取之于蓝而青于蓝」——却
修饰地/着(不译)「吾尝跂而望矣」——踮着脚,修饰
递进而且/并且「君子博学而日参省乎己」——递进

「以」「于」「其」「为」「则」「乃」「焉」「者」「所」要点

以:介词(用/拿/把:「以刀劈狼」;因为:「不以物喜」;凭借/按照:「以勇气闻于诸侯」);连词(来/用来:「属予作文以记之」;因而:「不宜妄自菲薄,以塞忠谏之路」)。辨介连:后面接名词/代词多为介词,接动词/分句多为连词。

于:介词。在(「战于长勺」)、向/对(「告之于帝」)、比(「苛政猛于虎」,形容词后表比较)、被(「受制于人」,动词后表被动)、从(「青取之于蓝」)。看后面成分定译法。

其:代词(他/他的/其中:「择其善者而从之」);语气副词——揣测「大概」(「其皆出于此乎」)、反问「难道」(「其孰能讥之乎」)、婉商「还是」(「吾其还也」)。句首多作语气副词。

为:动词(是/做/当:「中峨冠而多髯者为东坡」);介词(替/给:「为之奈何」;被:「身死人手,为天下笑」;因为/为了);句末语气词(呢:「何辞为」)。

则:连词。就(「学而不思则罔」)、却(表转折:「于其身也,则耻师焉」)、是/就是(表判断:「此则岳阳楼之大观也」)。

乃:于是/才(「乃悟前狼假寐」)、竟然(「乃不知有汉」)、是/就是(判断:「当立者乃公子扶苏」)。

焉:兼词「于此/于之」(「三人行必有我师焉」=于其中);代词(「以俟夫观人风者得焉」=之);句末语气词(「万钟于我何加焉」)；疑问代词「哪里」(「且焉置土石」)。

者:「……的人/……的事物」(「求人可使报秦者」);表停顿,引出判断/解释(「陈胜者,阳城人也」);放数词后=「个/样」(「二者不可得兼」)。

所:「所」字结构(所+动词=「所……的」:「道之所存」);「为……所……」表被动(「为国者无使为积威之所劫」)。

因、且、若、所以:因——介词(趁/凭借/通过:「因利乘便」)、连词(于是/因此)、动词(沿袭);且——「将要」「暂且」「况且」「一边…一边」;若——「你」「像」「如果」;所以——「用来…的」(「师者,所以传道受业解惑也」)、「……的原因」(「所以遣将守关者」)。

⚠ 辨析三法:①看位置(句首?句末?主谓间?动词前后?)②看前后词性(后接名词还是动词、分句)③代入检验(把释义放回原句,译得通才对)。

③ 「之」的六面插头(逐一带例)

「之」是虚词题的一号常客,六种用法必须条件反射。判断的钥匙永远是看它前后各是什么:

动词/介词 + 之 → 多为代词(它/他)
名词 + 之 + 名词 → 结构助词「的」
主语 + 之 + 谓语 → 取消句子独立性(不译)
宾语 + 之 + 动词(疑问句) → 宾语前置标志(如「何……之有」)

剩两种: 时间词/不及物动词后凑音节(「久之」不译);后接处所名词则是动词「到」(「之垄上」)。

⚠ 最易混:助词「的」 vs 取独。诀窍——「的」两边是名词性成分(可翻译成「……的……」);取独两边是「主语+谓语」(是个句子,被「之」取消了独立性,不能译成「的」)。

④ 「而」的五种关系判断

「而」永远连接前后两部分,考的是前后两件事是什么逻辑关系。五选一,靠代入试译:

关系判断口诀代入检验词
并列前后地位平等、可互换「又…又」「和」
承接前后有先后顺序「就」「接着」「然后」
转折前后语意相反、有落差「却」「但是」
修饰前一动作是后一动作的状态/方式「地」「着」(前修饰后)
递进后比前意思更进一层「而且」「并且」

⚠ 承接 vs 修饰是最大难点!承接:两个动作有先后、都是「实打实做出的动作」(登→招);修饰:前一个是后一个的「姿态/方式」,不能单独成为一步(跂[踮脚]着望——踮脚是望的姿态)。试着在中间加「地」,通=修饰;加「就」,通=承接。

⑤ 「以」:介词还是连词?

「以」的辨析核心是分介词和连词,方法极简单:看「以」后面接什么。

以 + 名词 / 代词(宾语) → 介词(用/拿/把/凭借/因为)
以 + 动词 / 分句 → 连词(来/用来/因而)

介词例:「以刀劈狼」(以+刀[名词]=用刀);「不以物喜」(以+物=因为外物)。连词例:「属予作文以记之」(以+记之[动词]=来);「不宜……以塞忠谏之路」(以=因而)。

再细分介词义:后接工具→「用/拿」;后接原因→「因为」;后接凭借/条件→「凭/靠/按照」。

⚠ 陷阱:「以」后省略了宾语,看起来后面直接接动词,其实是介词(「(公)以(之)相如功大,拜为上卿」)。判断时要还原省略成分。另:「是以」=「以是」=「因此」,固定结构。

⚖️ 互动 4 — 「以」介词 vs 连词判断器:读句子，判断词性并看理由

给一个含「以」的句子，你判它是介词还是连词，工具告诉你对错，并指出「以」后面接的是名词还是动词/分句——这就是判据。

⑥ 综合练习:意义和用法是否相同

高考最经典的题型:给两个句子,同一个虚词,问「意义和用法是否相同」。答题铁律——意义(译成现代汉语)和用法(词性+语法作用)必须都相同才算「相同」,只要一项不同就是「不同」。

第1步:分别判断两句中该虚词的词性 + 用法
第2步:分别把它译成现代汉语
第3步:词性/用法同 && 意义同 → 相同;否则 → 不同

示例:「作《师说》以贻之」的「以」(连词,来) vs 「以勇气闻于诸侯」的「以」(介词,凭借)——词性不同,故不同。

⑦ 全题型地图

题型A — 「之」的用法辨析

触发信号:划线句含「之」,问「之」的意义和用法;或要求解释加点的「之」。

标准动作:看「之」前后成分——动词后作宾语=代词;名词间=助词「的」;主谓间=取独;疑问句宾语提前=提宾标志;时间/动词后无义=凑音节;后接处所=动词「到」。

示例(带解):「师道之不传也久矣」——「师道」是主语,「不传」是谓语,「之」夹在主谓之间→取消句子独立性,不译。若换成「小大之狱」,「之」在两个名词性成分间→助词「的」。

陷阱:把取独当成「的」硬译;把凑音节的「之」当代词去找它代什么。

题型B — 「而/以/于」辨析

触发信号:句中含「而」「以」「于」,问其意义和用法。

标准动作:「而」看前后逻辑(并列/承接/转折/修饰/递进);「以」看后接词性(接名词=介词,接动词/分句=连词);「于」看后面成分定译(在/向/比/被/从)。

示例(带解):「青取之于蓝而青于蓝」——第一个「于」(取之于蓝)=「从」;「而」前后语意相反(取自蓝草却比蓝草青)=转折,却;第二个「于」(青于蓝)在形容词「青」后=比。

陷阱:「而」的承接与修饰混淆;「以」后省略宾语看漏,把介词当连词;「于」表被动漏判。

题型C — 「其/为/则」辨析

触发信号:句中含「其」「为」「则」,尤其出现在句首或句末,问用法。

标准动作:「其」在句首+疑问/感叹语气→语气副词(大概/难道/还是),作定语/主语→代词;「为」看词性(动词是/做,介词替/被,句末呢);「则」看逻辑(就/却/是)。

示例(带解):「其皆出于此乎」——句首、句末有「乎」表揣测→「其」是语气副词=大概;而「其孰能讥之乎」句末反问→「其」=难道。同字不同气。

陷阱:把句首表语气的「其」当代词;「则」表判断(是)漏判;「为」的被动义(为……所)看不出。

题型D — 意义和用法比较(两句同虚词是否相同)

触发信号:四个选项各给两个句子,同一虚词,问「意义和用法相同的一组」。

标准动作:逐项判断两句中该虚词的词性+译法,两者全同才算相同,一项不同即排除。善用课内例句迁移(有一句常出自课本)。

示例(带解):「作《师说》以贻之」(连词,来) vs 「以勇气闻于诸侯」(介词,凭借)——词性、译法都不同→不相同。判「不同」只需找一处差异即可。

陷阱:只对了「意义」没核「用法」(或反之);忽略课内那一句其实自己学过。

题型E — 语气词辨析(焉/也/矣/乎/哉/耳/耶)

触发信号:句末虚词,问它表什么语气,或「焉」是兼词、代词还是语气词。

标准动作:看句子语气(陈述/疑问/感叹/揣测)和位置。「焉」尤其要分:句中/句末表处所可还原「于之」=兼词;作宾语=代词;纯收尾无实义=语气词。

示例(带解):「三人行必有我师焉」——「焉」=「于此(在其中)」,可还原为介词短语→兼词;而「万钟于我何加焉」句末表反问语气→语气词(呢)。

陷阱:凡「焉」都当语气词;「也」既可句末(判断/陈述)也可句中(表停顿),漏判句中「也」。

⑧ 易错集

① 「之」作助词「的」还是代词?——助词「的」两边是名词性成分;代词多在动词/介词后作宾语,可指代具体对象。别一律译「的」。

② 「之」的「的」 vs 取独:两边是「主语+谓语」(一个句子)就是取独,不译;两边是「定语+中心语」才是「的」。

③ 「以」是介词还是连词?——后接名词/代词=介词;后接动词/分句=连词。注意省略宾语的介词「以」。

④ 「而」的承接与修饰:能加「就」=承接(两个先后动作);能加「地/着」=修饰(前是后的姿态方式)。这是失分重灾区。

⑤ 「其」表推测还是反问?——揣测配「乎/也」译「大概」;反问配「乎/哉」译「难道」;婉商译「还是」。看句末语气词和上下文。

⑥ 「于」的「被」义:动词后的「于」常表被动(「受制于人」);形容词后的「于」常表比较(「猛于虎」)。

⑦ 「焉」是兼词不是语气词:句末能还原为「于此/于之」的是兼词,别一律当语气词。

⑧ 「意义和用法是否相同」:必须词性、语法作用、译法都相同才算相同,漏核任何一项都会判错。

⑨ 「为」的被动:「为……所……」和单独「为」(为天下笑)都可表被动,别只译成「做/是」。

⑨ 尖子生拔高视角

① 虚词辨析的「三看一代」通用公式: 看位置(句首/句末/主谓间/动词前后)→ 看前后词性(名词?动词?分句?)→ 看语气(疑问/感叹/揣测/陈述)→ 代入检验(译回原句通不通)。任何一个虚词、任何一道题,都能套这套流程,不必死记零散释义。

② 18 虚词速查卡(核心义提炼): 之(代/的/取独/提宾/凑音节/到)、而(并列/承接/转折/修饰/递进)、以(用/因为/凭借/来/因而)、于(在/向/比/被/从)、其(他的/大概/难道/还是)、为(是做/替被/呢)、则(就/却/是)、乃(才于是/竟然/是)、焉(兼词于此/代词/语气词/哪里)、者(……的/表停顿)、所(所字结构/为所被动)、因(趁凭借/于是)、且(将要/暂且/况且)、若(你/像/如果)、所以(用来/的原因)、与(和/给/参与/欤)、且夫/然则(句首关联)。把它们默成一张卡,考场随取随用。

③ 「意义和用法是否相同」的满分答法: 拆成两个独立判断——先判「用法」(词性 + 在句中的语法作用是否一致),再判「意义」(译成现代汉语是否一致)。写答案时两条都要写到:「都是……(词性),都作……,都译为……,故相同」;或「前者是……,后者是……,词性/译法不同,故不同」。只答一条会扣分。

④ 课内例句是暗线: 比较题里几乎总有一句来自课本(《劝学》《师说》《六国论》《赤壁赋》等)。先把课内那句的用法钉死,再拿它去比课外那句——用你确定的锚定不确定,正确率飙升。所以背课文时要连虚词用法一起记。

⑤ 虚词与句式联动: 虚词常是特殊句式的「路标」:「之/是」提宾(何……之有)、「于/为……所/见」表被动、「者……也」表判断、「所」构成所字结构。看到这些虚词,先反应「这可能是个特殊句式」,辨析就有了方向,翻译也不会漏。
