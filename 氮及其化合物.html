氮及其化合物

从空气里最"懒"的 N₂,到会变色的 NO/NO₂、极易溶的 NH₃、强氧化的 HNO₃——一条完整的"氮价态链"。把每种物质讲成人话、动手拨一拨颜色与产物,喷泉、铜与硝酸、铵盐检验这些高频题就再也不慌。

🗣️ 把氮元素想成一个"会换脸的演员": 它能演 −3 价(NH₃、NH₄⁺)、0 价(N₂)、+2 价(NO)、+4 价(NO₂)、+5 价(HNO₃)。
N₂ 因为叁键太牢,是全场最"躺平"的;可一旦被放电或高温逼着反应,就沿着价态阶梯一级级往上爬:N₂ → NO → NO₂ → HNO₃。反过来,+5 价的硝酸"氧化性最强",遇到金属就把自己降下来变回 NO 或 NO₂。
考试就抓三条:颜色(NO 无色、NO₂ 红棕)、溶解性(NH₃ 极易溶做喷泉)、氧化性(硝酸与金属不放 H₂、稀出 NO 浓出 NO₂)。
本包导航: ① 氮气 N₂ 与氮的固定 → ② NO / NO₂ 的颜色与转化 → ③ 氨气 NH₃ 与喷泉实验 → ④ 铵盐与 NH₄⁺ 检验 → ⑤ 硝酸 HNO₃ 的强氧化性 → ⑥ 氨的催化氧化与工业制硝酸 → ⑦ 氮的价态循环与污染。每节都配可拨动的可视化。

① 氮气 N₂ 与氮的固定

空气里约 4/5 是 N₂。它无色无味、化学性质极稳定——因为一个 N≡N 叁键把两个氮原子锁得死死的,常温下几乎"油盐不进"。

:N≡N:   (叁键 · 键能大 · 极稳定)

N₂ 的两个关键反应(都要"逼"它)

放电 / 高温:N₂ + O₂ ⇌ 2NO　(雷雨发庄稼的第一步)
高温高压催化剂:N₂ + 3H₂ ⇌ 2NH₃　(合成氨,工业固氮)

什么是"氮的固定"?

类型方式把 N₂ 变成
自然固氮雷电放电 / 豆科植物根瘤菌NO(再到硝酸盐)/ 含氮化合物
人工固氮合成氨 N₂+3H₂⇌2NH₃NH₃(再制化肥、硝酸)

📌 氮的固定 = 把游离态的 N₂ 转化为化合态的氮化合物。这是氮循环的入口。

⚠ N₂ 与 O₂ 生成的是 NO(不是 NO₂!),因为放电条件只能爬到 +2 价。NO 再遇空气中的 O₂ 才变 NO₂。

② 一氧化氮 NO 与二氧化氮 NO₂

性质NO(+2 价)NO₂(+4 价)
颜色气味无色、无味红棕色、有刺激性气味
毒性有毒(与血红蛋白结合)有毒
溶于水不溶于水与水反应
收集方法排水法(不能排空气,会被 O₂ 氧化)向上排空气法(不能排水,会与水反应)

两条命门反应

2NO + O₂ → 2NO₂　(无色瞬间变红棕,不需点燃)
3NO₂ + H₂O → 2HNO₃ + NO　(NO₂ 溶水歧化,又吐回 NO)

📌 工业尾气吸收常"通入 O₂ 循环",让 NO 反复被氧化、再溶水,直到全部变 HNO₃:总反应 4NO₂ + O₂ + 2H₂O → 4HNO₃、4NO + 3O₂ + 2H₂O → 4HNO₃。

⚠ NO 只能用排水法!把 NO 排空气收集,它一出瓶口就遇 O₂ 变红棕的 NO₂,收不到纯 NO。这是选择题的经典坑。

③ 氨气 NH₃ 与喷泉实验

物理性质: 无色、有刺激性气味、密度比空气小、极易溶于水(1 体积水溶约 700 体积氨)。正是"极易溶"造就了喷泉实验。

化学性质

溶于水:NH₃ + H₂O ⇌ NH₃·H₂O ⇌ NH₄⁺ + OH⁻　(弱碱性,使湿润红色石蕊变蓝)
与酸:NH₃ + HCl → NH₄Cl　(蘸浓盐酸的玻璃棒靠近,冒白烟——检验氨)
催化氧化:4NH₃ + 5O₂ →(催化剂/Δ) 4NO + 6H₂O　(工业制硝酸的基础!)

实验室制法

2NH₄Cl + Ca(OH)₂ →(Δ) CaCl₂ + 2NH₃↑ + 2H₂O

📌 装置:固+固加热型(与制 O₂ 类似,试管口略向下防倒流)。向下排空气法收集(氨比空气轻)。干燥用碱石灰,不能用浓硫酸/无水 CaCl₂(会吸收氨)。

⚠ 检验氨用湿润的红色石蕊试纸——变蓝。也可用蘸浓盐酸的玻璃棒靠近冒白烟。别用蓝色石蕊(碱性看不出变化)。

交互 2 · 喷泉实验原理动画

圆底烧瓶里充满 NH₃。点『挤入水滴』:少量水进瓶,NH₃ 极易溶被"吸掉",瓶内压强骤降,外面大气压把烧杯里的水压着喷上来形成喷泉(滴入酚酞会显红,证明碱性)。

瓶内压强:101 kPa
原理:气体极易溶于液体 → 瓶内压强骤减 → 大气压把液体压入瓶中喷射。

④ 铵盐与 NH₄⁺ 的检验

铵盐通性: 都是易溶于水的白色晶体;受热易分解;与碱反应放出氨气。

受热分解:NH₄Cl →(Δ) NH₃↑ + HCl↑　(冷却又重新结合,升华现象)
与碱:NH₄⁺ + OH⁻ →(Δ) NH₃↑ + H₂O　(检验 NH₄⁺ 的核心离子方程)

NH₄⁺ 的检验(必背四步)

步骤操作
① 取样取少量待测固体或溶液于试管
② 加碱加入 NaOH 溶液(强碱)
③ 加热微微加热,放出气体
④ 检气用湿润的红色石蕊试纸靠近管口,试纸变蓝 → 证明含 NH₄⁺

⚠ 三个关键词缺一不可:加碱、加热、湿润红色石蕊变蓝。不加热氨放得慢难观察;试纸不湿润氨没法溶解显碱;红色石蕊才能"变蓝"给出正结果。

交互 3 · NH₄⁺ 的检验(加碱加热 → 石蕊变蓝)

按顺序点亮四步。到最后一步,湿润的红色石蕊试纸遇到逸出的 NH₃ 变蓝——这就是阳性结果。

第 0 步:准备好试管里的铵盐样品。

⑤ 硝酸 HNO₃ 的强氧化性

🗣️ 硝酸是"三重身份":强酸 + 不稳定 + 强氧化剂。 起氧化作用的不是 H⁺,而是 +5 价的氮(NO₃⁻)。所以硝酸与金属反应不放 H₂,而是把自己还原成 NO(稀)或 NO₂(浓)。

不稳定性

见光 / 受热:4HNO₃ →(光/Δ) 4NO₂↑ + O₂↑ + 2H₂O　(浓硝酸发黄=溶了 NO₂)

强氧化性:与铜反应(稀 / 浓 产物不同)

稀硝酸:8HNO₃(稀) + 3Cu → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O　(无色 NO)
浓硝酸:4HNO₃(浓) + Cu → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O　(红棕 NO₂)

📌 与非金属:C + 4HNO₃(浓) →(Δ) CO₂↑ + 4NO₂↑ + 2H₂O(浓硝酸能氧化碳)。

📌 常温钝化: 冷的浓硝酸(或浓硫酸)使 Fe、Al 表面生成致密氧化膜而"钝化",反应停住。所以可用铁罐/铝罐运输浓硝酸。

📌 王水(浓硝酸 : 浓盐酸 = 1 : 3 体积比)氧化性极强,能溶解金、铂。

⚠ 硝酸与金属反应永远不放 H₂!产物是 NO(稀)或 NO₂(浓)。稀出 NO 无色、浓出 NO₂ 红棕——记牢"稀弱浓强、稀低浓高"。

交互 4 · 硝酸与铜:稀 / 浓 产物对比

左瓶稀硝酸、右瓶浓硝酸,都放铜片。点『反应』:稀瓶冒无色 NO(升到瓶口遇空气才变红棕)、浓瓶直接冒红棕 NO₂;两边溶液都变蓝(生成 Cu²⁺)。

稀硝酸产物 NO(无色)、浓硝酸产物 NO₂(红棕);两者都不放 H₂,溶液变蓝(Cu²⁺)。

⑥ 氨的催化氧化与工业制硝酸

工业制硝酸(氨氧化法)三步走,把 −3 价的 N 一路氧化到 +5 价:

①催化氧化:4NH₃ + 5O₂ →(催化剂/Δ) 4NO + 6H₂O
②氧化:2NO + O₂ → 2NO₂
③吸收:3NO₂ + H₂O → 2HNO₃ + NO　(NO 循环回②)

📌 总账(把循环并进去):4NH₃ + 8O₂ → 4HNO₃ + 4H₂O(尾气 NO 补氧循环吸收)。原料是氨、空气(O₂)、水。

⚠ 第①步生成的是 NO 不是 NO₂;催化氧化的氮从 −3 升到 +2,是关键"起跳"。别把催化氧化产物写成 N₂ 或 NO₂。

交互 5 · 氨催化氧化制硝酸流程

点『开始流水线』,看 NH₃ 沿三步反应器一路走:催化氧化→氧化→水吸收,价态从 −3 爬到 +5,最终流出 HNO₃。NO 会循环回收。

当前工序:待机
原料:NH₃ + 空气(O₂) + H₂O → 产品:HNO₃。

⑦ 氮的价态循环与污染

把含氮物按化合价从低到高排成一条链,任何"某物→某物"的转化题都能在链上定位:

−3(NH₃/NH₄⁺) → 0(N₂) → +2(NO) → +4(NO₂) → +5(HNO₃/NO₃⁻)

污染面(考点)

污染元凶成因
酸雨(硝酸型)NOxNO/NO₂ 溶水生成硝酸,随雨落下
光化学烟雾NOx + 碳氢化合物汽车尾气在阳光下发生光化学反应,生成刺激性物质
富营养化硝酸盐/铵盐过量氮肥入水体,藻类疯长

📌 治理思路:汽车尾气用催化转化器把 NO 还原成 N₂;工厂尾气用碱液(或氨)吸收 NOx。

⚠ 别混:硫酸型酸雨来自 SO₂,硝酸型酸雨来自 NOx。光化学烟雾是 NOx 的"招牌罪状"。

⑨ 易错集(高频翻车点)

1. NO 用排水法收集,不能用排空气法——NO 一遇 O₂ 立即变红棕 NO₂,收不到纯品。NO₂ 反过来只能向上排空气(遇水会反应)。

2. 稀硝酸与铜产物是 NO(无色),浓硝酸是 NO₂(红棕)。搞反是送分题变送命题。

3. 硝酸(以及浓硫酸)与金属反应不放 H₂,是 +5 价 N(或 +6 价 S)起氧化作用,产物 NO/NO₂。

4. 常温下浓硝酸/浓硫酸使 Fe、Al 钝化(不是不反应,是生成致密膜阻止继续反应),故可用铁/铝制容器盛运浓硝酸;但加热或稀酸就会剧烈反应。

5. NH₄⁺ 检验必须"加碱 + 加热 + 湿润红色石蕊变蓝"三要素齐全,红色石蕊而非蓝色。

6. N₂ + O₂ 放电生成的是 NO(+2),不是 NO₂;氨催化氧化 4NH₃+5O₂→4NO+6H₂O 产物也是 NO,不是 N₂/NO₂。

7. 3NO₂+H₂O→2HNO₃+NO 里 NO₂ 既是氧化剂又是还原剂(歧化),生成的 NO 不能再直接溶水,要补 O₂ 循环。

8. 氨极易溶于水做喷泉,原理是"瓶内压强骤降";凡是极易溶(或极易反应)的气体+吸收液都能做喷泉(如 HCl+水、CO₂+浓 NaOH)。

⑩ 尖子生拔高视角

▸ 用价态链统一记反应方向: 一切含氮反应无非在 −3 → 0 → +2 → +4 → +5 这条链上"升"或"降"。氧化剂让 N 升价、还原剂让 N 降价,配平先定这两端。

▸ 电子守恒秒杀含氮计算: Cu 失电子总数 = N 得电子总数。稀硝酸每个 N(+5→+2)得 3e⁻,浓硝酸每个 N(+5→+4)得 1e⁻;混合气体先设 NO、NO₂ 物质的量,再列"电子守恒 + 原子守恒"两方程。

▸ 硝酸的双重角色: 与金属反应时,一部分 HNO₃ 起酸性(变成 NO₃⁻ 进盐),一部分起氧化性(变成 NO/NO₂)。算消耗硝酸别只算气体那部分。

▸ "喷泉"本质是压强差: 判断某气体能否做喷泉,只看它在吸收液里"溶得/反应得够不够快够不够多",让瓶内压强瞬间显著下降即可,不限于 NH₃。

▸ 尾气循环观: NO/NO₂ + O₂ + H₂O 最终都能写成生成 HNO₃ 的守恒式(4NO+3O₂+2H₂O=4HNO₃、4NO₂+O₂+2H₂O=4HNO₃),混合气吸收题用守恒一步到位。
