水溶液离子平衡

离子浓度比较看着千变万化,其实只有两步:先写守恒,再判主次。把这两步练成条件反射,这一类题就从"凭感觉"变成"按步走"。

总钥匙(两步走):
① 先写出能写的三大守恒(电荷守恒、物料守恒、质子守恒)——它们永远成立,是铁律;
② 再判断这溶液是电离占优还是水解占优,定离子大小的主次。
几乎所有"离子浓度比较"都被这两步拿下。

① 三大守恒:永远成立的铁律

电荷守恒

溶液呈电中性:所有阳离子电荷总数 = 所有阴离子电荷总数。(多价离子要乘价数,如 2c(CO₃²⁻))

物料守恒

某元素的原子总量不变:它分散成的各种微粒浓度加起来 = 投入的总量。

质子守恒

水电离出的 H⁺ 和 OH⁻ 总量相等。由前两式相减即可得到,不用硬背。

② 谁电离、谁水解:定主次

两条口诀先记死:

盐类水解:谁弱谁水解,越弱越水解,谁强显谁性。
水的电离:加酸/碱 → 抑制;加能水解的盐 → 促进。常温 c(H⁺)·c(OH⁻)=10⁻¹⁴。

⚠ 弱酸 + 其盐等浓混合(如醋酸+醋酸钠):醋酸电离>水解 → 显酸性;但 HCN+NaCN 等浓则水解>电离 → 显碱性。强弱不同,主次相反。

③ 实例库:点一个,看守恒 + 排序

④ 题型卡(收进题型地图)

触发信号

题目给一个盐溶液 / 酸碱盐混合液,让你比较离子浓度大小,或判断某个守恒等式对不对。

标准动作

① 判类型:单一盐?弱酸/弱碱?等浓混合?反应后过量谁?
② 写三守恒(电荷必写,物料按元素写,质子=两式相减)。
③ 判主次:水解显谁性 / 电离vs水解谁强 → 定 H⁺、OH⁻ 谁大。
④ 排序:一般"不水解/不电离的主离子最大 → 次之 → 显性离子 → 另一个"。

易错 / 不适用

多元弱酸盐分步水解(第一步远大于第二步),别漏 2× 的系数;混合液先判"谁过量、生成什么"再守恒;比较的是浓度 c 不是个数。

⑤ 自测(先想,再点开)
