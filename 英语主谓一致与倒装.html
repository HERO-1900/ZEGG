主谓一致与倒装

这两个考点常一起考,共同的命门都是「谓语动词怎么摆」。主谓一致管的是谓语动词的单复数(跟对主语);倒装管的是谓语(或助动词)的位置(要不要提到主语前)。看懂主语的「真身」、认出倒装的「触发词」,这两类题就能稳拿分。

🗣️ 说人话总览: 主谓一致 = 谓语动词的单复数要「跟对主语」——主语是单数就用单数动词(is/does/has),主语是复数就用复数动词(are/do/have),关键是先找到真正的主语别被中间的修饰语骗了。倒装 = 为了强调或语法需要,把谓语(或助动词)提到主语前面。看到某些「触发词」开头(否定词、only、地点副词),就得把语序调过来。
两条主线: ① 主谓一致三原则——语法一致(主语单复数决定谓语)、意义一致(the police/family 看意义)、就近一致(there be / or / either...or 就近)。② 倒装两类——完全倒装(整个谓语提前:here/there/地点副词开头)、部分倒装(只把助动词提前:否定词/only+状语/so/neither/省 if 的虚拟)。

① 主谓一致三大原则

语法一致　·　意义一致　·　就近一致

语法一致(默认): 主语在语法形式上是单数,谓语用单数;主语是复数,谓语用复数。这是最基本的规则。

The book on the desk is mine.　(主语 The book 单数 → is)
The books on the desk are mine.　(主语 The books 复数 → are)

意义一致: 有些主语看形式判断不了,得看意义。集合名词(family/team/class/police)指整体用单数,指成员个体用复数。

The family is large.(整体) / The family are watching TV.(成员各看)

就近一致: there be、or、either...or、neither...nor、not only...but also 连接的并列主语,谓语跟最靠近它的那个名词一致。

Neither you nor he is right.　(靠 he → is)

⚠ 三原则有优先级:能靠语法就靠语法;the police 这类看意义;there be / or 连接才用就近。别一上来就用就近。

② 特殊主语的一致

主语类型谓语单复数例
each / every / one of / either / neither(单独作主语)单数Each of us has a book.
a number of + 复数名词复数(许多)A number of students are here.
the number of + 复数名词单数(数目)The number of students is 50.
分数 / 百分比 of + 名词看 of 后名词Two-thirds of the water is gone. / Half of them are gone.
集合名词 family/team/class整体→单;成员→复The team is strong. / The team are arguing.
不可数名词 / 单数动名词短语单数Reading books is useful.
the + 形容词(表一类人)复数The rich are not always happy.

⚠ 头号陷阱:a number of(许多,复数)和 the number of(...的数目,单数)一字之差,谓语完全相反。牢记 a=多=复,the=数=单。

③ 完全倒装:整个谓语提前

完全倒装是把整个谓语动词(不是助动词)移到主语前面。触发条件是句首放了地点/方位副词或介词短语,或 here/there 开头,且主语是名词(不是代词)。

Here comes the bus.　(here 开头 → 谓语 comes 提前)
In the room sat an old man.　(地点短语开头 → 谓语 sat 提前)
On the wall hangs a famous painting.　(方位开头 → hangs 提前)

关键限制: 主语若是代词,不倒装!"Here it comes." / "Here he comes." 保持正常语序。

⚠ 完全倒装无需助动词(不用 do/does/did);主语是代词时坚决不倒。别写成 "Here comes it."(错)。

④ 部分倒装:只把助动词提前

部分倒装是把助动词/情态动词/be 动词提到主语前,主谓其余部分不动(像疑问句语序)。若句中没有助动词,要借 do/does/did。四大触发:

(1) 否定/半否定词开头

never, hardly, seldom, rarely, little, no sooner...than, hardly...when, not only, by no means, not until 等放句首 → 部分倒装。

Never have I seen such a scene.　(Never 开头 → have 提前)
Not until he left did I know the truth.　(借 did)

(2) only + 状语开头

Only then did I realize my mistake.　(only+时间状语 → 借 did)

注意:only 后是主语时不倒装。"Only Tom knows it."(不倒)

(3) so / neither / nor 承接

He likes coffee. So do I.　(我也是 → So + do + 主语)
She can't swim. Neither can I.　(我也不 → Neither + can + 主语)

(4) 省略 if 的虚拟条件

虚拟条件句省去 if 时,把 had / were / should 提到主语前。

Had I known it, I would have told you.　(= If I had known)
Were I you, I would go.　(= If I were you)

⚠ so 承接前后主语不同才倒装(表「也」);若 so 表「的确如此」(前后同一主语)则不倒装:"He is clever. So he is."(他确实聪明)。

⑤ 部分倒装还原:变回正常语序

做题和自查的利器:把倒装句还原成正常语序,主谓一致和句意就一目了然。步骤:

第1步:找到被提前的助动词(have/did/can/had/were...)。
第2步:把助动词放回主语后面,恢复「主语 + 助动词 + 其余」。
第3步:否定词/only 状语留在原位(它们本就是状语),句子含义不变。

例: Never have I seen such a thing. → I have never seen such a thing.(还原后 never 回到动词前)

⚠ 还原只动助动词位置,不改否定词或状语的意思。借来的 do/does/did 还原后要合并回实义动词的时态:"did I know" → "I knew"。

⑧ 全题型地图

题型A — 主谓一致(语法/意义/就近)型

触发信号:横线在主语后、谓语动词处,考 is/are、has/have、does/do 的单复数;或主语被 of 短语、定语从句隔开。

标准动作:先划掉修饰语找真主语 → 语法一致优先;集合名词看意义;there be/or/either...or 用就近。

示例(带解):The teacher, together with his students, is going to the park. → 真主语 The teacher 单数,together with 短语不影响 → is。(with/along with/as well as 连接时看前面的主语)

陷阱:被 together with / as well as / of 短语中的名词带偏;the police/people 恒复数误当单数。

题型B — 特殊主语一致型

触发信号:主语含 each/every/one of/a number of/the number of/分数/百分比/the+形容词。

标准动作:each/every/one of → 单;a number of → 复,the number of → 单;分数/百分比看 of 后名词;the rich/poor → 复。

示例(带解):Two-thirds of the students are boys.(of 后 students 复数 → are) / Two-thirds of the money is spent.(money 不可数 → is)

陷阱:a number of 误用单数;every 后接 and 并列仍用单数(Every boy and girl is...)。

题型C — 否定词开头部分倒装型

触发信号:句首是 never/hardly/seldom/little/not only/no sooner/not until/by no means 等否定或半否定词。

标准动作:句首否定词 → 部分倒装:把助动词/情态/be 提前;没有助动词借 do/does/did,时态照原句。

示例(带解):Not only did he pass the exam, but he also got an A.(没有助动词 → 借 did,he 后动词还原为原形 pass)。

陷阱:忘了借 do/did;借了 did 却没把实义动词变回原形;no sooner...than 与 hardly...when 时态(过去完成+一般过去)搭配错。

题型D — only + 状语倒装型

触发信号:句首 only + 副词/介词短语/状语从句(only then, only in this way, only when...)。

标准动作:only+状语开头 → 部分倒装(主句倒装,状语从句本身不倒);only+主语不倒装。

示例(带解):Only when the war ended could they return home.(从句 when the war ended 不倒,主句 could 提前)。

陷阱:把从句部分也倒装;分不清 only 后是状语(倒)还是主语(不倒)。

题型E — so/neither...承接倒装型

触发信号:前句说完,后句用 so/neither/nor 表示「另一人也这样/也不这样」。

标准动作:肯定承接用 So + 助动词 + 主语(也是);否定承接用 Neither/Nor + 助动词 + 主语(也不是)。助动词与前句时态一致。

示例(带解):She has finished. So have I.(前句 has → 承接用 have + I) / He didn't go. Neither did I.

陷阱:误把 So + 主语 + 助动词(表「的确如此」)当成承接倒装;助动词时态与前句不符。

题型F — 完全倒装(地点状语)型

触发信号:句首是地点/方位副词或介词短语(here/there/in the room/on the hill),后面是名词主语。

标准动作:把整个谓语提到主语前(不借助动词);主语是代词时不倒装。

示例(带解):Under the tree stood a little girl.(地点开头,主语 a little girl 是名词 → 谓语 stood 提前)。

陷阱:主语是代词还倒装(Here comes it. 错);误用助动词(Here does come the bus. 错)。

⑨ 易错集

① a number of(许多→复数 are)vs the number of(数目→单数 is):一字之差,谓语相反。

② not only...but also 放句首,前半句要部分倒装(Not only did he...),后半句正常语序。

③ so vs such:so + 形容词/副词(so beautiful),such + (a/an) + 名词(such a beautiful girl);so...that 与 such...that 结构别混。

④ 就近原则:there be / or / either...or / neither...nor / not only...but also 连接主语,谓语跟最近的名词(Neither you nor he is right)。

⑤ 集合名词 family/team/class/police:整体用单数,强调成员用复数;the police/people 恒复数。

⑥ with/together with/as well as/along with 连接的不是并列主语,谓语看前面的主语(The teacher, with his students, is...)。

⑦ 完全倒装主语是代词时不倒:Here it comes.(对)/ Here comes it.(错)。

⑧ 借来的 do/does/did 后面实义动词要变回原形:Not until then did I know(不是 knew)。

⑨ only + 主语不倒装(Only Tom knows);only + 状语才倒装(Only then did...)。

⑩ 尖子生拔高视角

① 一切主谓一致题,先做「找真主语」这一步: 划掉 of 短语、with/as well as 短语、定语从句、插入语,剩下的核心名词才决定谓语。80% 的一致陷阱都靠这一刀解决——修饰语再长也只是干扰项。

② 倒装的本质是「触发词把助动词吸到句首」: 部分倒装 = 借疑问句语序(助动词+主语);没有现成助动词就借 do/does/did。看到否定词/only状语/so/neither开头,大脑直接切成「疑问句语序」,又快又准。

③ 完全倒装 vs 部分倒装,一句话区分: 完全倒装动的是整个谓语(地点副词开头,stood/hangs/comes 整体提前);部分倒装只动助动词(否定词开头,did/have/can 提前)。前者不借助动词、代词主语不倒;后者必用助动词。

④ 虚拟省 if 的倒装是「高分标志」: Had I..., Were I..., Should you... 三种句型直接对应 if 从句的三种时态(过去完成/过去/将来)。写作里主动用这类倒装,句式立刻上档次,也是完形/语法高频考点。

⑤ 就近一致是「例外中的规律」: there be、or、either...or、neither...nor、not only...but also——这些「二选一/接力式」主语,谓语只认最后一个。把它们当一组记死,遇到就切「就近模式」,避免误用语法一致。
