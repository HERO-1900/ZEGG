构词法(词根 · 词缀)

背单词最亏的方式,是把每个词当成毫无规律的字母串死记。其实绝大多数长词都是拼出来的:一个词根管核心意思,前面加前缀改方向或正反,后面加后缀改词性。看懂这套拼法,生词能猜、拼写不慌、语法填空的「词性转换」直接秒杀。这一页教你把「记单词」变成「拆单词」。

🗣️ 构词法(人话版): 一个单词像乐高:词根是中间那块主板(定意思),前缀贴在前面(改意思:un- 变否定、re- 变「再」、ex- 变「向外」),后缀贴在后面(改词性:-tion 变名词、-ful 变形容词、-ly 变副词、-ize 变动词)。所以看到一个没见过的长词,别慌——先切三刀:掐头(前缀)、去尾(后缀)、留中间(词根),意思和词性八九不离十就出来了。语法填空里让你「用括号里的词的适当形式填空」,考的就是这一招。
一句话口诀: 前缀改意思,后缀改词性,词根是核心。 拆词三刀:前缀(un-/dis-/in-/re-/pre-/mis-…) + 词根(spect 看 / port 拿 / dict 说) + 后缀(-tion/-ment 名词、-ful/-able 形容词、-ly 副词、-ize/-en 动词)。语法填空第一问永远是:这个空该填什么词性? 看句子结构定词性,再挑对应后缀。

① 前缀:改意思,不改词性

前缀加在词根前面,只改变意思(否定、方向、程度、时间),一般不改变词性。 happy(形容词)加 un- 还是 unhappy(形容词),只是意思反过来。

前缀意思例词
un-否定「不」(最常用,加形容词/分词)happy→unhappy, able→unable
dis-相反 / 否定(常加动词、形容词)agree→disagree, appear→disappear
in-/im-/il-/ir-否定「不」(拼写随后一个字母变)correct→incorrect, possible→impossible
re-再 / 重新 / 回write→rewrite, turn→return
pre-前 / 预先view→preview, pay→prepay
mis-错误地understand→misunderstand
ex-向外 / 前任port→export, ex-boss
en-/em-使…(少数前缀会「变词性」→ 动词)large→enlarge, rich→enrich

⚠ 大多数前缀不改词性,但 en-/em- 是例外(large 形容词 → enlarge 动词)。别把「加了前缀词性一定不变」当死规矩。

🔧 互动 1 — 构词拆解器:输入或点一个词,自动切成「前缀+词根+后缀」并判词性

颜色对应:蓝=前缀(改意思) · 绿=词根(核心) · 橙=后缀(定词性)。词库里没有的词会提示,先点上面的示例词感受一下切法。

② 后缀:改词性,是语法填空的命门

后缀加在词根后面,主要改变词性。 看到空要填什么词性,就挑对应后缀——这是语法填空「词性转换」题的核心动作。

后缀变成例词
-tion/-sion/-ment/-ness/-ity名词 n.act→action, move→movement, happy→happiness
-er/-or/-ist/-ee名词(表人)teach→teacher, act→actor, art→artist
-ful/-less/-able/-ible/-al/-ous/-y形容词 adj.care→careful/careless, comfort→comfortable
-ly副词 adv.(加在形容词后)quick→quickly, careful→carefully
-ize/-ise/-en/-ify动词 v.modern→modernize, wide→widen, beauty→beautify

判词性三步: ①看空在句中的位置 → ②确定要名词/动词/形容词/副词 → ③把括号里的词加对应后缀(注意拼写变化)。

⚠ -ly 不一定是副词!friend→friendly(友好的,形容词)、love→lovely(可爱的,形容词)是「名词+ly=形容词」。判词性要看句子,不是死记后缀。

🎡 互动 2 — 后缀转盘:点一个后缀,指针转到它,告诉你「变成什么词性」+ 例词

同一颜色 = 同一词性:绿=名词 · 橙=形容词 · 紫=副词 · 红=动词。点任意扇区看它的转换规律。

③ 词根:核心意思的来源,一根带一串

词根是单词的「意思内核」。 记住一个词根,就能顺藤摸瓜猜出一整串同族词——这是「构词法猜词」题的底层能力。

词根意思一串同族词
spect看 (look)inspect 检查 · respect 尊敬 · spectator 观众 · prospect 前景
port拿 / 搬运 (carry)import 进口 · export 出口 · transport 运输 · portable 便携的
dict说 (say)predict 预言 · dictionary 词典 · dictate 口述 · contradict 反驳
vis/vid看 (see)visible 可见的 · television 电视 · video 视频
ject扔 (throw)reject 拒绝 · inject 注射 · project 投射/项目

猜词法: 遇到生词 inspector → in-(向内)+ spect(看)+ -or(表人)→「往里看的人」=检查员/督察。八九不离十。

⚠ 词根猜词是「毛估方向」,不是精确翻译。考场上用它缩小选项范围,再结合上下文定死,别硬拿字面意思去套具体语境。

🌳 互动 4 — 词根家族树:点一个词根,长出一整串同族词(spect / port / dict)

中间是词根,四周是它派生出的词。同一个词根反复出现在不同词里——记根,一次记一串。

④ 否定前缀怎么选:in / im / il / ir 的拼写规则

否定前缀 in- 会为了发音顺口而变形,规则很硬:

用哪个规则例词
il-后一个字母是 llegal→illegal, legible→illegible
ir-后一个字母是 rregular→irregular, responsible→irresponsible
im-后一个字母是 m / p / bpossible→impossible, polite→impolite
in-其他情况(默认)correct→incorrect, complete→incomplete
un-最通用,多加形容词/分词happy→unhappy, expected→unexpected
dis-表「相反」,常加动词/形容词agree→disagree, honest→dishonest

记法: l 前用 il、r 前用 ir、m/p/b 前用 im、其余用 in。这是「同化」(字母跟着下一个字母走)。

⚠ 别所有否定都套 un-。possible 的否定是 impossible 不是 unpossible;regular 是 irregular 不是 unregular。具体词有固定搭配,拼写规则帮你排除大部分错误。

⑤ 派生词的词性判断(语法填空核心技能)

语法填空最常考:给你 (explain),要你根据句子填 explanation / explain / clearly…。先判空要什么词性,再挑后缀。

空的位置线索要填的词性常见后缀
冠词后 / 介词后 / 作主语宾语名词 n.-tion -ment -ness -ity
be 动词后 / 名词前 / 系动词后形容词 adj.-ful -less -able -al -ous
修饰动词 / 修饰形容词 / 句首副词 adv.-ly
作谓语 / 情态动词后 / to 后动词 v.-ize -en -ify(或原形)

⚠ 判词性靠句子结构,不是靠感觉。「the ___ of…」中间必是名词;「be ___」多半形容词;「V + ___」多半副词。定死词性再变形,拼写变化(y→i、去 e 加后缀)别丢分。

⑥ 全题型地图

题型A — 语法填空 · 词性转换型

触发信号:括号里给一个词的原形,如 (explain)(care)(possible),要「用适当形式填空」。

标准动作:①看空在句中位置定词性(名/动/形/副)→ ②加对应后缀 → ③处理拼写变化(y→i、去e、双写)→ ④否定语境加 un-/im- 等前缀。

示例(带解):He gave a clear ___ (explain) of the rule. 空在冠词 a + 形容词 clear 之后,作宾语 → 要名词 → explain + -ation → explanation。

陷阱:只加后缀忘了拼写变化(happy→happiness 不是 happyness);语境要否定却漏掉前缀。

题型B — 完形填空 · 派生词辨析型

触发信号:四个选项是同根不同缀的词,如 care / careful / careless / carefully。

标准动作:先用「空的词性」淘汰一半(要副词就只留 -ly),再用「意思正反」(careful 认真 vs careless 粗心)在剩下的里定死。

示例(带解):She drove very ___. 空修饰动词 drove → 要副词 → 只能 carefully / carelessly;由 very(小心地) 语境 → carefully。

陷阱:光看意思不看词性,选了 careful(形容词)去修饰动词。先定词性,再挑意思。

题型C — 阅读 · 构词法猜词型

触发信号:文中出现生词,题目问 the underlined word probably means…,或直接影响理解。

标准动作:切三刀:前缀(方向/正反)+ 词根(核心)+ 后缀(词性),拼出大致意思,再回上下文验证。

示例(带解):生词 irreplaceable → ir-(否定)+ replace(替换)+ -able(能…的)→「不能被替换的」= 不可替代的。上下文若讲珍贵之物,吻合。

陷阱:只靠词根字面硬翻,忽略上下文。构词法给方向,上下文定终稿。

题型D — 前缀否定型(填空/选择)

触发信号:句子语义明显是「相反/否定」,要给一个词加否定前缀。

标准动作:先判是否要否定(看 but/however/not 等信号)→ 再按拼写规则选前缀:l前il、r前ir、m/p/b前im、其余in,或固定搭配 un-/dis-。

示例(带解):It is ___ (possible) to finish in one second. 语义否定 + possible 以 p 开头 → im- → impossible。

陷阱:一律套 un-(写出 unpossible / unregular 这类错词);possible→impossible、regular→irregular 是固定的。

题型E — 合成词型

触发信号:两个独立单词拼成一个新词,如 blackboard, well-known, sunrise, part-time。

标准动作:拆成两个词分别理解,再合起来。名词+名词=名词(blackboard);副词/形容词+分词=形容词(well-known 出名的、good-looking 好看的)。

示例(带解):a well-known writer → well(好地)+ known(被知道的)→「广为人知的」作定语修饰 writer。连字符合成词常作形容词。

陷阱:合成形容词作定语时中间的分词/形容词不再单独变化(a five-year-old boy,不是 five-years-old)。

⑦ 易错集

① 词性转换拼写:happy→happiness(y→i)、write→writing(去e)、begin→beginning(双写)。加对了后缀却拼错,照样扣分。

② 否定前缀选错:possible→impossible(不是 unpossible)、regular→irregular、legal→illegal。按 l/r/m-p-b 规则 + 固定搭配。

③ -ible vs -able:多数是 -able(comfortable, acceptable);少数固定 -ible(possible, responsible, visible, terrible)。-ible 的记牢,其余默认 -able。

④ -ly 不都是副词:friendly / lovely / lonely / lively 是形容词(名词+ly)。判词性看句子,不看后缀长相。

⑤ 前缀一般不改词性,但 en-/em-(enlarge, empower)会把形容词/名词变动词——别忽略这个例外。

⑥ 词根猜词只给方向,不是精确翻译。选择题用它排除,最后一定回上下文定死。

⑦ 形容词变副词:careful→carefully(直接+ly)、true→truly(去e)、happy→happily(y→i再+ly)。别漏拼写变化。

⑧ 名词后缀混淆:同一动词可能有固定名词形式,explain→explanation(不是 explaination)、pronounce→pronunciation。这类高频词要背准。

⑧ 尖子生拔高视角

① 把「背单词」升级成「拆单词」: 每记一个长词,顺手拆出前缀/词根/后缀,再联想同根的三五个词。记 spect 一次,inspect/respect/spectator/prospect 一起进账——同样时间,词汇量翻几倍,而且拼写和词性一并稳固。

② 词性转换的底层是「句法先行」: 高手拿到语法填空,先不看括号里的词,而是分析空在句中的语法位置(主语?定语?状语?),定死要什么词性,再回头给单词穿对应后缀的衣服。词性由句子结构决定,不由单词本身决定——这是不失分的关键顺序。

③ 否定前缀是「发音同化」不是「随机」: in- 在 l/r/m/p/b 前变 il/ir/im 纯粹为了好读(舌头不打架)。理解这层,你不再死背,而是「一读就知道该用哪个」,考场上零犹豫。

④ 猜词=构词法×上下文的双保险: 只靠构词法可能偏(prospect 字面「向前看」实指「前景」),只靠上下文可能猜不到范围。两者叠加:构词法圈定语义大方向,上下文把它钉死到具体义项——这是阅读生词题的满分解法。

⑤ 建立自己的「词根—词缀」小地图: 把高频词根(spect/port/dict/vis/ject…)和高频后缀(-tion/-ment/-able/-ize…)各列一张表,新词往上挂。三个月后你看长词像看拼装积木,写作用词也更精准——这是从「够用」到「尖子」的分水岭。
