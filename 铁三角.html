铁三角:Fe / Fe²⁺ / Fe³⁺ 的转化与检验

铁的三种身份 Fe、Fe²⁺、Fe³⁺ 围成一个"三角循环"。谁能变谁、要用什么试剂、怎么检验、怎么保存除杂——本质都是一条"氧化还原强弱顺序"在指挥。把这条链讲成人话、动手拨一拨,选择与实验题就能稳拿。

🗣️ 把铁想成一个人的三种状态: Fe 是"满血 0 价"(还原性最强),Fe²⁺ 是"中间态 +2"(可上可下),Fe³⁺ 是"缺电子 +3"(氧化性最强)。
氧化剂往上推,还原剂往下拉: 遇到强氧化剂(Cl₂、HNO₃、O₂)就被夺电子往 +3 爬;遇到还原剂(Fe、Cu、I⁻)就被还给电子往 0 或 +2 掉。关键一句:Fe³⁺ 氧化性排在 Cu²⁺ 之上、I₂ 之下——记住这条尺子,能氧化谁、不能氧化谁、谁能把谁还原,全都一目了然。
本包导航: ① 铁三角总原理与转化方程 → ② 氧化性还原性强弱顺序(这把尺子统一一切) → ③ Fe²⁺/Fe³⁺ 的检验 → ④ 保存与除杂 → ⑤ 离子共存 → ⑥ 全题型地图 → ⑦ 易错集 → ⑧ 尖子生拔高 → ⑨ 自测。每节都配可拨动的可视化。

① 铁三角:三者互相转化的条件与方程

三个顶点、六条边。向上(升价)靠氧化剂,向下(降价)靠还原剂。先看清每条边要用什么。

转化方向常用试剂典型方程(离子式)
Fe → Fe²⁺升价(弱氧化)非氧化性酸(稀盐酸/稀硫酸)、弱氧化剂、Fe³⁺Fe+2H⁺=Fe²⁺+H₂↑ ; Fe+2Fe³⁺=3Fe²⁺
Fe → Fe³⁺升价(强氧化)Cl₂、稀HNO₃(过量)、热浓H₂SO₄2Fe+3Cl₂ =(点燃)= 2FeCl₃
Fe²⁺ → Fe³⁺升价(被氧化)Cl₂、HNO₃、H₂O₂、KMnO₄(H⁺)、O₂2Fe²⁺+Cl₂=2Fe³⁺+2Cl⁻
Fe³⁺ → Fe²⁺降价(被还原)Fe、Cu、I⁻、SO₂、H₂S2Fe³⁺+Cu=2Fe²⁺+Cu²⁺
Fe³⁺ → Fe深度还原铝热反应、CO、H₂(高温)2Al+Fe₂O₃ =(高温)= 2Fe+Al₂O₃
Fe²⁺ → Fe降价(析出)更活泼金属(Zn)、电解Zn+Fe²⁺=Zn²⁺+Fe

⚠ Fe 与足量稀 HNO₃ / Cl₂ / 热浓硫酸 → 得 Fe³⁺;Fe 过量时(或加非氧化性酸)→ 只得 Fe²⁺。"过量还是不足"决定最终价态,是最爱考的分岔口。

⚠ 常温下 Fe 遇冷浓 H₂SO₄ / 冷浓 HNO₃ 会钝化(表面生成致密氧化膜而反应停止),不能一路氧化到 Fe³⁺。

② 氧化性 / 还原性强弱顺序(统一一切的尺子)

🗣️ 一条铁律:强氧化剂 + 强还原剂 → 弱氧化剂 + 弱还原剂。 反应总是"强的吃掉弱的"。把常见氧化剂按强弱排成一条轴,谁在上面谁就能氧化下面的还原剂对应物。

氧化性由强到弱(必背这一段)

Cl₂ > Br₂ > Fe³⁺ > I₂ > ...　　对应还原性:Cl⁻ < Br⁻ < Fe²⁺ < I⁻

同时记住金属离子氧化性次序里:Fe³⁺ 排在 Cu²⁺ 之上——所以 Fe³⁺ 能氧化 Cu(2Fe³⁺+Cu=2Fe²⁺+Cu²⁺)。

Fe³⁺ 氧化性的"精确定位":在 Cu²⁺ 与 I₂ 之间

Fe³⁺ 能否氧化?结论依据
Fe³⁺ + Cu能,生成 Fe²⁺、Cu²⁺Fe³⁺ 氧化性 > Cu²⁺
Fe³⁺ + I⁻能,生成 Fe²⁺、I₂Fe³⁺ 氧化性 > I₂
Fe³⁺ + Br⁻不能Br₂ 氧化性 > Fe³⁺,故 Fe³⁺ 拉不动 Br⁻
口诀:2Fe³⁺ + 2I⁻ = 2Fe²⁺ + I₂　(能) ；　Fe³⁺ 遇 Br⁻ 不反应(不能)

⚠ 向 FeBr₂ 溶液中通 Cl₂:因还原性 Fe²⁺ > Br⁻,先氧化 Fe²⁺ 再氧化 Br⁻。少量 Cl₂ 只把 Fe²⁺ 变 Fe³⁺;足量 Cl₂ 才连 Br⁻ 一起氧化。顺序题的经典陷阱。

③ Fe²⁺ 与 Fe³⁺ 的检验

三大检验试剂速查

试剂加入 Fe²⁺加入 Fe³⁺结论
KSCN 溶液不变红(需先氧化)立即变红(血红色)变红即证有 Fe³⁺
NaOH 溶液白色→灰绿→红棕(渐变)红棕色沉淀(直接)颜色变化过程验 Fe²⁺
K₃[Fe(CN)₆](铁氰化钾)蓝色沉淀(滕氏蓝)无蓝色沉淀生成蓝色沉淀即证有 Fe²⁺

关键方程与现象

Fe³⁺ + 3SCN⁻ = Fe(SCN)₃(血红,可逆,非沉淀)
Fe²⁺+2OH⁻=Fe(OH)₂↓(白) → 4Fe(OH)₂+O₂+2H₂O=4Fe(OH)₃(红棕)
Fe³⁺+3OH⁻=Fe(OH)₃↓(红棕)

严谨检验 Fe²⁺ 的方案(排除 Fe³⁺ 干扰): 取样,先加 KSCN 不变红(证原来无 Fe³⁺),再加氯水(或 H₂O₂),变红(证 Fe²⁺ 被氧化成 Fe³⁺)——两步才严谨。

⚠ 检验 Fe²⁺ 时若直接加 KSCN 不变红只能说明"此刻无 Fe³⁺",不能直接证明有 Fe²⁺;必须再用氧化剂氧化后变红。用 K₃[Fe(CN)₆] 出蓝色沉淀则可直接证 Fe²⁺。

④ Fe²⁺ 的保存 与 铁盐除杂

Fe²⁺ 易被氧化 → 如何保存

Fe²⁺ 在空气中易被 O₂ 氧化成 Fe³⁺(4Fe²⁺+O₂+4H⁺=4Fe³⁺+2H₂O)。保存 FeSO₄ / FeCl₂ 溶液的两招:

① 加入少量铁钉(Fe):把生成的 Fe³⁺ 随时还原回去(Fe+2Fe³⁺=3Fe²⁺)
② 加入少量对应的酸(如 H₂SO₄):抑制 Fe²⁺ 水解,并造酸性环境

除杂:加什么、通什么

目标(主)杂质方法原理
FeCl₂混有 FeCl₃加过量铁粉后过滤Fe+2Fe³⁺=3Fe²⁺,把杂质 Fe³⁺ 变成主体 Fe²⁺,不引入新杂质
FeCl₃混有 FeCl₂通入适量 Cl₂2Fe²⁺+Cl₂=2Fe³⁺+2Cl⁻,把杂质 Fe²⁺ 氧化成主体 Fe³⁺

⚠ 除杂"不减主、不增新":FeCl₂ 除 Fe³⁺ 只能用 Fe(不能用 Cu,会引入 Cu²⁺);FeCl₃ 除 Fe²⁺ 只能通 Cl₂(不能加氯水以外的会引杂的氧化剂,也不能过量以免… Cl₂ 恰好即可,多余 Cl₂ 逸出不留杂)。

⚠ 不能用硝酸把 FeCl₂ 里的 Fe²⁺ 氧化——会引入 NO₃⁻ 杂质;也不能用双氧水后残留问题需具体分析。首选 Cl₂ 因为它只带入 Cl⁻(与主体一致)。

⑤ 离子共存判别

Fe³⁺ 是"氧化性强 + 易水解 + 会络合"的离子,一堆离子不能与它共存:

与 Fe³⁺ 不共存原因
I⁻被氧化:2Fe³⁺+2I⁻=2Fe²⁺+I₂(氧化还原)
S²⁻、HS⁻被氧化:2Fe³⁺+S²⁻=2Fe²⁺+S↓(氧化还原)
SCN⁻络合变红:Fe³⁺+3SCN⁻=Fe(SCN)₃(生成络合物)
CO₃²⁻、HCO₃⁻、AlO₂⁻双水解:2Fe³⁺+3CO₃²⁻+3H₂O=2Fe(OH)₃↓+3CO₂↑
OH⁻生成沉淀 Fe(OH)₃↓

Fe²⁺ 的共存要点: Fe²⁺ 有还原性,不能与强氧化性离子(如酸性 MnO₄⁻、NO₃⁻/H⁺、ClO⁻)共存;也不与 OH⁻、S²⁻ 共存(生成沉淀)。

⚠ Fe²⁺ 与 NO₃⁻ 在酸性条件下不共存(NO₃⁻/H⁺ 相当于稀硝酸,会氧化 Fe²⁺);中性时 Fe²⁺ 与 NO₃⁻ 可共存。加不加 H⁺ 是关键分岔。

⑦ 易错集(高频翻车点)

1. Fe 与 Cl₂ / 足量稀 HNO₃ / 热浓硫酸 → Fe³⁺;而 Fe 与盐酸、稀硫酸 → 只到 Fe²⁺。别把非氧化性酸也氧化到 +3。

2. Fe 过量时(如 Fe 与稀 HNO₃、Fe 与 FeCl₃),最终产物是 Fe²⁺,不是 Fe³⁺。"过量还是不足"决定价态。

3. 常温下 Fe 遇冷浓硫酸 / 冷浓硝酸钝化,不是不反应,而是生成致密氧化膜后停止。

4. 检验 Fe²⁺ 直接加 KSCN 不变红只能证"无 Fe³⁺",不能证"有 Fe²⁺";要再加氧化剂变红才严谨,或用 K₃[Fe(CN)₆] 出蓝色沉淀。

5. Fe(OH)₂ 白色只是瞬间,题目问现象要写"白→灰绿→红棕"完整过程;方程 4Fe(OH)₂+O₂+2H₂O=4Fe(OH)₃ 别漏 H₂O。

6. FeBr₂ 通 Cl₂:还原性 Fe²⁺>Br⁻,少量 Cl₂ 先氧化 Fe²⁺,足量才氧化 Br⁻。别一上来就写 Br⁻ 被氧化。

7. Fe³⁺ 能氧化 Cu、I⁻,但不能氧化 Br⁻、Cl⁻。Fe³⁺ 卡在 Cu²⁺ 与 I₂ 之间。

8. 除杂 FeCl₂ 中的 Fe³⁺ 用铁粉不用铜;FeCl₃ 中的 Fe²⁺ 用 Cl₂ 不用硝酸(否则引入新杂质离子)。

9. Fe²⁺ 与 NO₃⁻ 共存要看酸碱性:加 H⁺ 则相当于稀硝酸会氧化 Fe²⁺,不共存;不加 H⁺ 可共存。

10. 蒸干灼烧 FeCl₃ 溶液得 Fe₂O₃(水解 + 灼烧),不是 FeCl₃。

⑧ 尖子生拔高视角:一把尺子统一一切

▸ 所有铁三角问题 = 在强弱顺序轴上找位置。 记住这条氧化性序:Cl₂ > Br₂ > Fe³⁺ > I₂ >(Cu²⁺ 在 Fe³⁺ 之下)。任何"能不能反应""谁先反应""生成什么价态",都在这条轴上一眼看穿。

▸ "强吃弱"定所有产物: 氧化剂过量→还原剂被氧化到最高可及价态(Fe→Fe³⁺);还原剂过量→氧化剂被还原、且多余还原剂把已生成的高价拉回(Fe 过量→Fe²⁺)。判断产物先问"谁过量"。

▸ 多种还原剂共存,按还原性从强到弱依次被氧化: FeBr₂ 通 Cl₂,先 Fe²⁺ 后 Br⁻;FeI₂ 通 Cl₂,先 I⁻(还原性 I⁻>Fe²⁺)后 Fe²⁺——顺序会因 I⁻/Br⁻ 而不同,这是压轴分。

▸ 得失电子守恒秒杀计算: 氧化剂得电子总数 = 还原剂失电子总数。不必写全方程,数电子即可。例:n 个 Fe³⁺ 被还原成 Fe²⁺ 得 n 个电子,对应 Cu 失电子数 = n/2 × 2… 直接列等式。

▸ 检验先排干扰: 严谨方案永远是"先证无干扰离子、再制造目标现象"。检验 Fe²⁺ 先加 KSCN 证无 Fe³⁺,再氧化变红——这套"先排除后确认"的思路适用于所有离子检验。
