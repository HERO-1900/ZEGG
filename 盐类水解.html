盐类水解

盐溶到水里,为什么有的显酸、有的显碱、有的还会"打起来"彻底分解?一句"谁弱谁水解、谁强显谁性"背后,是弱离子悄悄抢走水电离出的 H⁺/OH⁻。把这条链拨明白,酸碱判断、方程书写、蒸干产物、浓度比较就都通了。

🗣️ 把盐溶液想成一场"抢水"游戏: 水自己会电离出极少的 H⁺ 和 OH⁻(手拉手的一对)。盐溶进去后拆成阴阳离子——如果某个离子来自弱酸或弱碱,它就"贪心"地把水里的 H⁺(或 OH⁻)抢过去结合成弱电解质分子。
被抢走一方就"落单"变多:抢走 H⁺ → 剩 OH⁻ 多 → 显碱性;抢走 OH⁻ → 剩 H⁺ 多 → 显酸性。抢的过程促进了水继续电离。记住三句口诀:有弱才水解、越弱越水解、谁强显谁性,再加"升温、稀释都促进水解",这一章的骨架就立起来了。
本包导航: ① 水解本质 → ② 谁弱谁水解·谁强显谁性 → ③ 越弱越水解 → ④ 水解方程书写 → ⑤ 影响因素(温度/浓度/酸碱) → ⑥ 双水解 → ⑦ 电离与水解竞争 → ⑧ 离子浓度大小比较 → ⑨ 全题型地图 → ⑩ 易错集 → ⑪ 尖子生视角 → ⑫ 自测。每节都配可拨动的可视化。

① 盐类水解的本质

定义: 盐电离出的弱离子(弱酸的酸根 / 弱碱的阳离子)与水电离出的 H⁺/OH⁻ 结合生成弱电解质,从而破坏了水的电离平衡、促进水继续电离,使溶液显酸或碱性。

Ac⁻ + H₂O ⇌ HAc + OH⁻　　(抢走 H⁺、剩 OH⁻ → 碱性)
NH₄⁺ + H₂O ⇌ NH₃·H₂O + H⁺　　(抢走 OH⁻、剩 H⁺ → 酸性)

三个"本质"要点

① 水解是中和反应的逆反应:酸+碱⇌盐+水,倒过来就是盐+水⇌酸+碱,故水解吸热。

② 水解促进水的电离(与加酸加碱"抑制"正相反),盐溶液中由水电离出的 c(H⁺) 或 c(OH⁻) 大于 10⁻⁷。

③ 水解程度一般很微弱:c(盐)远大于水解产物,方程用可逆号 ⇌、不标 ↑↓。

⚠ 只有含弱离子的盐才水解。强酸强碱盐(NaCl、KNO₃、Na₂SO₄)全是"强"离子,不水解,显中性。

交互 1 · 水解平衡粒子动画

选一种弱离子,看它不断从水分子那里"抢"H⁺/OH⁻,结合成弱电解质;落单的另一半离子逐渐变多,溶液显色。这是一个动态平衡:表面不变,底下一直在换人。

② 谁弱谁水解 · 谁强显谁性

🗣️ 把盐拆成"阳离子"和"阴离子"两半,分别看它们的"出身":来自强酸/强碱的离子老实,不抢水;来自弱酸/弱碱的离子会水解。最后谁强就显谁的性质。

四类盐速判表

盐的类型例子谁水解溶液性质
强酸强碱盐NaCl、KNO₃、Na₂SO₄都不水解中性
强碱弱酸盐NaAc、Na₂CO₃、NaF阴离子水解碱性(显强碱的性)
强酸弱碱盐NH₄Cl、AlCl₃、FeCl₃、CuSO₄阳离子水解酸性(显强酸的性)
弱酸弱碱盐NH₄Ac、(NH₄)₂CO₃双水解(阴阳都水解)看 Ka、Kb 谁大

⚠ "谁强显谁性"针对强弱搭配的盐:强碱弱酸盐显碱性,强酸弱碱盐显酸性。弱酸弱碱盐两边都弱,要比 Ka 与 Kb:如 NH₄Ac 中 Ka(HAc)≈Kb(NH₃·H₂O),故近中性。

③ 越弱越水解(强弱定量)

同浓度的钠盐,对应的酸越弱(Ka 越小),其酸根水解程度越大,溶液碱性越强、pH 越高。

水解常数 Kh = Kw / Ka　→　Ka 越小,Kh 越大,水解越彻底

典型强弱顺序(酸性由强到弱)

H₂SO₃ > H₃PO₄ > HF > HAc > H₂CO₃ > HClO > HCO₃⁻ > HSiO₃⁻ ……对应钠盐碱性反过来:酸越弱、盐越碱。

同浓度钠盐对应弱酸 Ka水解程度碱性
NaAcKa≈1.8×10⁻⁵(较大)小弱碱
NaHCO₃Ka₂(H₂CO₃)≈4.7×10⁻¹¹较大较碱
NaClOKa(HClO)≈3×10⁻⁸较大较碱
Na₂CO₃Ka₂ 很小大强碱

⚠ 判"同浓度盐溶液谁碱性强":先找对应的酸,酸越弱盐越碱。这是比较 pH 大小题的核心工具。

④ 水解离子方程式书写

四条铁律

① 一般水解程度小,用可逆号 ⇌,一般不标 ↑ ↓(产物量太少不足以逸出或沉淀)。

② 多元弱酸根分步水解,以第一步为主,分步写:

CO₃²⁻ + H₂O ⇌ HCO₃⁻ + OH⁻　(主)　；　HCO₃⁻ + H₂O ⇌ H₂CO₃ + OH⁻(次)

③ 多元弱碱阳离子(如 Fe³⁺、Al³⁺)水解一步写到底:

Fe³⁺ + 3H₂O ⇌ Fe(OH)₃ + 3H⁺

④ 双水解(彻底)用 = 且标 ↑ ↓(互相促进、进行到底):

2Al³⁺ + 3CO₃²⁻ + 3H₂O = 2Al(OH)₃↓ + 3CO₂↑

⚠ 别把水解写成电离:HCO₃⁻ 水解是 HCO₃⁻+H₂O⇌H₂CO₃+OH⁻;HCO₃⁻ 电离是 HCO₃⁻⇌H⁺+CO₃²⁻。两个方向别写反。别把一般水解标 ↑↓,也别把双水解写成可逆。

⑤ 影响水解的因素

水解平衡也遵守勒夏特列原理。以 Ac⁻+H₂O⇌HAc+OH⁻ 为例:

操作水解平衡水解程度说明
升高温度正向(促进)增大 ↑水解吸热,升温永远促进水解
加水稀释正向(促进)增大 ↑越稀越水解,但离子浓度都变小
加酸(H⁺)逆向(抑制)对碱性盐抑制阴离子水解H⁺ 中和 OH⁻;但会促进阳离子盐? 见下
加碱(OH⁻)抑制阳离子水解、促进阴离子水解视情况OH⁻ 中和 H⁺
加同种可水解盐看是否互促—互促盐一起加会加剧双水解

口诀: 升温、稀释一律促进水解;加"抑制离子"(酸性盐加酸、碱性盐加碱)则抑制水解;加"能中和产物"的物质则促进水解。

⚠ 配 FeCl₃、AlCl₃、SnCl₂ 等易水解盐溶液时,要加对应的酸(盐酸)抑制水解防浑浊;稀释虽促进程度,但离子浓度总体下降。

⑥ 双水解:彻底 vs 部分

🗣️ 一般水解各玩各的、彼此不理;双水解是阳离子水解生 H⁺、阴离子水解生 OH⁻,两者互相中和、互相促进,像两个人拆台越拆越狠,直到彻底分解。

能发生"彻底双水解"的组合(必记)

Al³⁺ 与 CO₃²⁻ / HCO₃⁻ / AlO₂⁻ / S²⁻ / HS⁻ / ClO⁻；Fe³⁺ 与 CO₃²⁻ / HCO₃⁻ / AlO₂⁻。

2Al³⁺ + 3S²⁻ + 6H₂O = 2Al(OH)₃↓ + 3H₂S↑
Al³⁺ + 3HCO₃⁻ = Al(OH)₃↓ + 3CO₂↑　(泡沫灭火器原理)

应用: ① 明矾 KAl(SO₄)₂ 净水:Al³⁺ 水解生成 Al(OH)₃ 胶体吸附杂质;② 泡沫灭火器:Al₂(SO₄)₃ 与 NaHCO₃ 双水解快速放 CO₂。

⚠ 不是所有阴阳离子相遇都双水解到底。NH₄⁺ 与 CO₃²⁻ 只是相互促进但不彻底(仍写可逆);Al³⁺ 与 HCO₃⁻ 才彻底(用=并标↑↓)。判断关键:是否生成气体/沉淀离开体系。

⑦ 电离 ⚖ 水解 的竞争

对酸式盐的酸根(如 HCO₃⁻、HSO₃⁻、H₂PO₄⁻),它既能电离(HCO₃⁻⇌H⁺+CO₃²⁻,显酸)又能水解(HCO₃⁻+H₂O⇌H₂CO₃+OH⁻,显碱)。谁强就显谁性。

酸式酸根电离 vs 水解溶液性质
HCO₃⁻水解 > 电离碱性
HS⁻水解 > 电离碱性
HSO₃⁻电离 > 水解酸性
H₂PO₄⁻电离 > 水解酸性
HPO₄²⁻水解 > 电离碱性

⚠ NaHCO₃ 显碱性(水解占优),NaHSO₃ 显酸性(电离占优)——这是经典对比考点,别背反了。判据:看该酸根对应弱酸的 Ka 与 Kh 谁大。

⑧ 盐溶液离子浓度大小比较

三步法: ① 先定溶液酸碱性(定 H⁺/OH⁻ 谁多);② 判主要离子:未水解的"完整"离子最多,水解产物次之,深度水解最少;③ 用三大守恒收尾验证。

典型排序(牢记)

0.1mol/L NaAc:c(Na⁺) > c(Ac⁻) > c(OH⁻) > c(H⁺)
0.1mol/L NH₄Cl:c(Cl⁻) > c(NH₄⁺) > c(H⁺) > c(OH⁻)
0.1mol/L Na₂CO₃:c(Na⁺) > c(CO₃²⁻) > c(OH⁻) > c(HCO₃⁻) > c(H⁺)

三大守恒(以 NaHCO₃ 为例):
电荷守恒:c(Na⁺)+c(H⁺)=c(HCO₃⁻)+c(OH⁻)+2c(CO₃²⁻)
物料守恒:c(Na⁺)=c(HCO₃⁻)+c(CO₃²⁻)+c(H₂CO₃)
质子守恒:c(H⁺)+c(H₂CO₃)=c(OH⁻)+c(CO₃²⁻)

⚠ 写电荷守恒别漏乘电荷数(CO₃²⁻ 前面 ×2);比较不同盐时先判酸碱性再排序。

⑩ 易错集(高频翻车点)

1. 把水解方程标 ↑↓ 或写"="。一般水解微弱,用 ⇌、不标 ↑↓;只有双水解彻底才用=并标↑↓。

2. 多元弱酸根一步写完。CO₃²⁻ 水解必须分步,以第一步为主,不能写成 CO₃²⁻+2H₂O⇌H₂CO₃+2OH⁻。

3. 把 NaHSO₃ 当碱性、NaHCO₃ 当酸性——记反了。NaHCO₃ 水解占优显碱性,NaHSO₃ 电离占优显酸性。

4. 蒸干 AlCl₃、FeCl₃、MgCl₂ 溶液以为得原盐。挥发性酸的弱碱盐蒸干灼烧得氧化物(Al₂O₃、Fe₂O₃、MgO)。

5. 认为所有阴阳离子相遇都双水解。只有 Al³⁺/Fe³⁺ 与 CO₃²⁻/HCO₃⁻/AlO₂⁻/S²⁻ 等才彻底;NH₄⁺ 与 CO₃²⁻ 不彻底。

6. 电荷守恒漏乘电荷数:CO₃²⁻、SO₄²⁻、Ca²⁺ 等二价离子系数要 ×2。

7. 误以为水解使盐溶液离子总数减少。水解程度小,主体仍是原盐离子,c(盐)≫c(水解产物)。

8. 把"升温促进水解"和"升温使溶液更碱/更酸"搞混:碱性盐升温更碱,酸性盐升温更酸,方向随盐而定。

⑪ 尖子生拔高视角

▸ 统一用 K 量化: Kh=Kw/Ka(阴离子水解)或 Kw/Kb(阳离子水解)。"越弱越水解"= Ka 越小 → Kh 越大。判酸式酸根显性:比较 Ka(电离常数)与 Kh(水解常数),Kh>Ka 显碱性。

▸ 守恒是不变量: 无论怎样水解、混合,电荷守恒与物料守恒永远成立。压轴浓度比较先写守恒,把未知消去,常能秒杀。质子守恒=电荷守恒−物料守恒。

▸ 水解 vs 电离的对称美: 加酸加碱"抑制"水的电离;可水解盐"促进"水的电离。两者是同一 Kw 平衡被推的两个方向。

▸ 双水解的判据在"离开体系": 只要生成难溶物或气体离开平衡,反应就被拉到底。这也解释了泡沫灭火器为何反应快而彻底。

▸ 蒸干产物统一逻辑: 阴离子挥发(Cl⁻源自 HCl、NO₃⁻)+ 阳离子水解 → 得氧化物;阴离子不挥发(SO₄²⁻)→ 得原盐。抓"酸挥不挥发"这一条即可秒判。
