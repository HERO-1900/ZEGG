常见气体的实验室制备与净化收集

发生装置怎么选、除杂用什么、干燥剂配哪种、气体怎么收、尾气怎么处理——这是实验大题里最"套路化"也最容易丢分的一块。把每一步讲成人话、再动手拨一拨,连接顺序一气呵成,你就能在实验题里稳稳拿满分。

🗣️ 把制气想成一条流水线: 原料在发生装置里反应放出气体 → 气体一路"过关"被洗气瓶除杂 → 再进干燥管抽走水汽 → 到收集瓶装起来 → 剩下的有毒尾气用碱液吸收。
整条线只有一个顺序不能乱:制备 → 除杂 → 干燥 → 收集 → 尾气。而每一节又各有一句口诀:发生装置看『反应物状态+要不要加热』,收集方法看『溶不溶于水、比空气重不重』,干燥剂看『酸配酸、碱配碱、氨气最娇气』。抓住这几句,半张实验卷就稳了。
本包导航: ① 三种发生装置 → ② 常见气体制法归纳 → ③ 除杂净化(洗气) → ④ 干燥剂选择 → ⑤ 收集方法 → ⑥ 验满验纯与尾气处理 → ⑦ 装置连接总顺序。每节都配可动手的可视化,最后是全题型地图 + 易错集 + 自测。

① 气体发生装置的三种类型

选装置只看两件事:反应物是固体还是液体、要不要加热。三张脸对号入座:

① 固 + 固,加热型("试管口略向下")

两种固体混合、需要酒精灯加热。大试管固定在铁架台,试管口要略向下倾斜——防止反应生成的水冷凝后倒流回热的管底,把试管炸裂。

2KClO₃ → 2KCl + 3O₂↑　(MnO₂ 催化, Δ)
2NH₄Cl + Ca(OH)₂ → CaCl₂ + 2NH₃↑ + 2H₂O　(Δ)

⚠ 口诀:固固加热、口向下。这是"制 O₂(氯酸钾法)"和"制 NH₃(铵盐+碱)"的标志装置。

② 固 + 液 / 液 + 液,不加热型(启普发生器 / 简易)

固体与液体常温反应即可放气,不需加热。可用启普发生器(随开随用、随关随停),或"锥形瓶+分液漏斗"简易装置。要求固体是块状不溶(如大理石、锌粒),反应放热不剧烈。

Zn + H₂SO₄(稀) → ZnSO₄ + H₂↑
CaCO₃ + 2HCl → CaCl₂ + H₂O + CO₂↑

⚠ 启普发生器只适用于"块状固体+液体、不加热、气体不溶于水或微溶"的反应;粉末状固体、需加热、气体太易溶都不能用。

③ 固/液 + 液,加热型

固体(或浓溶液)与液体反应,并且需要加热才能进行。用圆底烧瓶+分液漏斗+酒精灯,常配石棉网。

MnO₂ + 4HCl(浓) → MnCl₂ + Cl₂↑ + 2H₂O　(Δ)

⚠ 稀盐酸不行,必须浓盐酸;而且随反应进行盐酸变稀,反应会停止。这是"制 Cl₂"的标志装置。

② 常见气体制法归纳

气体主要原理发生装置干燥剂收集

NH₃2NH₄Cl+Ca(OH)₂→CaCl₂+2NH₃↑+2H₂O(Δ)①固固加热碱石灰(不可浓硫酸/CaCl₂)向下排空气
SO₂Na₂SO₃+H₂SO₄→Na₂SO₄+H₂O+SO₂↑②固液不加热浓硫酸(不可碱石灰)向上排空气

③ 除杂净化(洗气)

刚制出的气体常带杂质(挥发的酸、水汽等)。用洗气瓶(装液体试剂)或干燥管(装固体)让气体通过,把杂质"吸掉"。

选除杂试剂的两条铁律

① 只除杂、不吸主气: 试剂能吸收杂质,却不能和要收集的目标气体反应。

② 不能引入新杂质: 除完不能带进别的气体。

目标气体要除的杂质选用试剂为什么不用别的
Cl₂HCl饱和食盐水不用水/NaOH:会溶解或反应掉 Cl₂
CO₂HCl饱和 NaHCO₃ 溶液不用 Na₂CO₃/NaOH:会吸收 CO₂
SO₂HCl饱和 NaHSO₃ 溶液不用 NaOH:会吸收 SO₂
CO₂SO₂(混在里面)饱和 NaHCO₃ 溶液NaHCO₃ 吸 SO₂ 却放出 CO₂,不损失目标气

⚠ 记住:除 Cl₂ 中 HCl 用饱和食盐水(既压 HCl 挥发,又不溶 Cl₂);除 CO₂ 中 HCl 用饱和 NaHCO₃(它吸 HCl 不吸 CO₂)。选错试剂把目标气也吸走,是本节头号翻车点。

交互 3 · 洗气瓶进气方向(长进短出)

洗气瓶必须『长管进、短管出』:气体从长导管插入液面下鼓泡通过,才能充分接触试剂;从短管排出。点按钮看正确气流,以及接反了会怎样。

④ 干燥剂选择

🗣️ 酸配酸、碱配碱,酸碱互不沾;氨气最娇气,浓硫酸、无水氯化钙都不能碰。 干燥剂本质是"抢水",但抢水的同时不能和目标气体反应。
干燥剂酸碱性能干燥不能干燥
浓硫酸 / P₂O₅酸性酸性 & 中性气体:CO₂、SO₂、Cl₂、H₂、O₂、HCl、NO₂等NH₃(碱性,会被吸收/反应)
碱石灰(NaOH+CaO)碱性碱性 & 中性气体:NH₃、H₂、O₂、N₂等CO₂、SO₂、Cl₂、HCl(酸性气,会被吸收)
无水 CaCl₂中性大多数中性/酸性气体NH₃(会生成 CaCl₂·8NH₃)

⚠ 三条高频错点:NH₃ 不能用浓硫酸、也不能用无水 CaCl₂,只能用碱石灰;CO₂/SO₂/Cl₂ 不能用碱石灰(碱性干燥剂吸走它们);要干燥的气体和干燥剂"酸碱相反"就一定错。

⑤ 收集方法

收集只看两条性质:溶不溶于水、密度比空气大还是小(空气平均摩尔质量 ≈ 29 g/mol)。

收集法适用条件典型气体
排水法不溶 / 难溶于水,且不与水反应O₂、H₂、NO、N₂
向上排空气法(瓶口朝上)密度 大于空气(M > 29)Cl₂(71)、CO₂(44)、SO₂(64)、NO₂(46)、HCl(36.5)
向下排空气法(瓶口朝下)密度 小于空气(M < 29)H₂(2)、NH₃(17)

能排水法尽量排水法(纯度高),但易溶于水的(NH₃、HCl、SO₂、CO₂部分)只能排空气;有色/有毒又不溶的(如 NO)常用排水。

⚠ 三个"钉子":NH₃ 极易溶于水,只能向下排空气;NO 只能排水法(见空气变 NO₂);NO₂ 不能排水(与水反应),只能向上排空气。H₂ 既可排水也可向下排空气。

⑥ 验满 / 验纯 与尾气处理

验满(检验瓶子是否收满)

气体验满方法
O₂带火星的木条放瓶口,复燃则已满
CO₂燃着的木条放瓶口,熄灭则已满
Cl₂湿润的淀粉碘化钾试纸放瓶口,变蓝则已满
NH₃湿润的红色石蕊试纸放瓶口,变蓝则已满

验纯(点燃前防爆——针对可燃气)

H₂ 验纯: 收一小试管气体点燃,声音尖锐刺耳=不纯(混空气),声音很轻"噗"=较纯。可燃气(H₂、CO 等)点燃或加热前必须先验纯,否则易爆炸。

尾气处理(有毒气体不能直排)

尾气处理方法
Cl₂、SO₂通入 NaOH 溶液吸收(碱液)
NO₂、NONaOH 溶液吸收(NO 常需与 O₂ 配合)
NH₃、HCl(极易溶)用水吸收,需防倒吸(倒扣漏斗/干燥管)
CO点燃烧掉(可燃,尾气收集或燃烧)

⚠ 极易溶于水的气体(NH₃、HCl)用水或碱液吸收时,导管不能直接插入液面下,否则液体倒吸。要用倒扣漏斗或防倒吸装置。

⑦ 装置连接总顺序

制备(发生) → 除杂(洗气) → 干燥 → 收集 → 尾气处理

为什么是这个顺序? ① 先除杂再干燥——因为很多除杂用的是溶液(如饱和食盐水、NaHCO₃),气体过完液体会重新带上水汽,所以干燥必须放在洗气之后,否则前面干了后面又湿了,白干。

② 收集放在干燥之后——要收的是又纯又干的气体。

③ 尾气处理放最后——把没收满、逸出的有毒气兜住。

⚠ 头号错点:先干燥后除杂是错的!洗气(溶液)会把气体重新润湿,干燥一定在洗气之后。这一条几乎年年考。

⑨ 易错集(高频翻车点)

1. NH₃ 不能用浓硫酸、也不能用无水 CaCl₂ 干燥(都会吸收/反应),只能用碱石灰。

2. CO₂、SO₂、Cl₂、HCl 等酸性气体不能用碱石灰干燥(碱性干燥剂会把它们吸走)。

3. 顺序错:必须"先除杂再干燥"。洗气用的是溶液,会重新带上水汽,干燥要放在洗气之后。

4. 洗气瓶要"长进短出":长导管伸入液面下鼓泡,短导管排气。接反(短进长出)起不到洗气作用甚至把液体压出。

5. 固固加热型(制 O₂、NH₃)试管口要略向下倾斜,防冷凝水倒流炸裂试管。

6. NO 只能排水法(见空气变 NO₂);NO₂ 不能排水(与水反应生成 HNO₃ 和 NO),只能向上排空气。

7. 除 Cl₂ 中 HCl 用饱和食盐水(不能用水或 NaOH,会损失 Cl₂);除 CO₂ 中 HCl 用饱和 NaHCO₃(不能用 NaOH/Na₂CO₃)。

8. 制 Cl₂ 必须用浓盐酸且加热;稀盐酸不行,盐酸变稀后反应停止。

9. 极易溶气体(NH₃、HCl)尾气吸收要防倒吸(倒扣漏斗)。

10. 可燃气(H₂)点燃前必须验纯,防爆炸。

⑩ 尖子生拔高视角

▸ 一条主线统摄全局: 任何制气题都能拆成"发生—除杂—干燥—收集—尾气—验纯/验满"六格。拿到题先在脑中列这六格,逐格填,漏项、错序自然浮现。

▸ "酸碱相斥"的普适判据: 干燥剂、除杂剂、尾气吸收剂,本质都在问"目标气体是酸性/碱性/中性?试剂会不会和它反应?"。把气体分类(酸性:CO₂/SO₂/Cl₂/HCl/NO₂;碱性:NH₃;中性:H₂/O₂/NO/N₂),再套"同性不吸、异性吸收",一切迎刃而解。

▸ 装置气密性检验是隐藏得分点: 组装后先检气密性(如微热法看气泡/液柱)。大题常在此设问,别忽略。

▸ 启普发生器的三条件同时满足: 块状固体 + 不加热 + 气体不溶(或微溶)。任缺一条就不能用——这是判断题的常客(如制 O₂ 需加热、制 CO₂ 用粉末都不行)。

▸ 浓稀之辨: 铜与浓硝酸放 NO₂、与稀硝酸放 NO;MnO₂ 只与浓盐酸反应放 Cl₂。"浓/稀"是很多制气题的分水岭,读题先圈浓度。
