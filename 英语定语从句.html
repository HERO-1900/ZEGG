定语从句

定语从句就是用一整个句子当形容词,去修饰前面的一个名词(叫「先行词」)。所有难点都收敛成两个动作:①先看先行词是人还是物;②再看关系词在从句里当什么成分(主语/宾语/状语)。关系代词、关系副词、介词+which、as/which 的所有花样,全都是这两步的延伸。

🗣️ 定语从句(人话版): 你想说「那本书」,又想补一句「(那本书)是我昨天买的」——英语不重复说 the book,而是塞一个关系词 that/which 把两句缝成一句:the book that I bought yesterday。核心永远是两步:①找先行词是人还是物(人→who/whom/whose,物→which/that);②看关系词在从句里缺的是主语、宾语还是状语(缺主/宾→关系代词,缺状语→关系副词 when/where/why 或介词+which)。
一句话公式: 先行词(人/物) + 关系词 + 从句(把关系词在里面的位置留空)。关系词在从句里当什么成分,决定用哪个词:当主语/宾语→关系代词(who/which/that/whose);当状语(时间/地点/原因)→关系副词(when/where/why)= 介词+which。永远先补全从句的完整意思,再看缺什么。

① 先行词与关系词:两条主线

先行词(被修饰的名词) ← 关系词 + 从句

关系词分两大类。关系代词在从句里作主语/宾语/定语(替代一个名词);关系副词在从句里作状语(替代「介词+名词」这种地点/时间/原因短语)。

关系词先行词在从句里作
who人主语(有时口语作宾语)
whom人宾语(尤其介词后)
which物主语 / 宾语
that人 或 物主语 / 宾语(仅限制性)
whose人或物定语(所属关系,+名词)
when时间状语 = in/on/at which
where地点状语 = in/at/on which
whyreason状语 = for which

⚠ 判断代词还是副词,只看一件事:从句里缺不缺「名词性」成分。缺主语/宾语(是名词)→关系代词;不缺主宾、只是缺个地点/时间/原因(是状语)→关系副词。

🧩 互动 1 — 成分分析器:点句子,高亮先行词 + 关系词 + 从句里被替代的成分

红=先行词,蓝=关系词,绿=从句里那个「被留空」的位置。看清关系词到底顶替了从句中的哪个成分,选词就不会错。

② 关系词在从句中作什么成分:决定用哪个

核心方法:把定语从句拆回一个完整的独立句子。 关系词其实替代了原句里的某个词,那个词在原句里当什么成分,就决定用哪个关系词。

The man [who lives next door] is a doctor. → He lives next door.（who = 主语 he）
The book [which I bought] is new. → I bought it.（which = 宾语 it，可省）
The city [where I was born] is small. → I was born there/in it.（where = 状语，不能省）

关系代词作宾语时可以省略(限制性从句中):the book (that) I bought。关系副词、以及作主语的关系代词不能省。

⚠ 别只看先行词就选词!先行词是地点(city/place),从句里若缺的是宾语,仍要用 which/that,不能用 where。where 只在从句缺「状语」时才用。

🌳 互动 2 — 关系词决策树:先行词人/物 → 从句缺主/宾/状 → 自动给出该用的词

两个下拉一改,决策树立刻走到对应叶子,亮出该用的关系词和一句例证。这正是考场上你脑子里该跑的流程。

③ 限制性 vs 非限制性:那个逗号改变一切

限制性(不加逗号): 从句是先行词不可缺的「身份证」,去掉后意思就残缺/变了。可用 that,作宾语时关系词可省。

非限制性(加逗号): 从句只是「顺便补充一句」,去掉后主句意思仍完整。不能用 that,不能省关系词;人用 who/whom,物用 which。

限制性(无逗号)非限制性(有逗号)
作用限定、区分(缺它意思变)补充说明(缺它意思全)
能用 that 吗能不能(只用 which/who)
关系词可省吗作宾语时可省不可省
翻译「…的」(直接修饰)「,而/其中…」(另起一句)

⚠ 一个逗号,意思天差地别。My brother who lives in Paris = 我有好几个哥哥,住巴黎那个;My brother, who lives in Paris, = 我只有一个哥哥,顺便说他住巴黎。专有名词(唯一的人/物)后面几乎都用非限制性。

🔀 互动 3 — 限制 vs 非限制对比:开关那个逗号,看意思如何反转

同一句 "My sister(,) who is a teacher(,) lives in Beijing.",逗号在=补充说明(只有一个姐姐);逗号不在=区分(有多个姐姐,当老师那个)。

④ 介词 + 关系代词:where/when 的「原形」

当关系词在从句里作「介词的宾语」时,可以把介词提到关系词前面,构成 介词 + which/whom。这也是关系副词的「本来面目」。

where = in / at / on which　　when = in / on / at which　　why = for which
the house in which I live = the house where I live（我住的房子）
the day on which we met = the day when we met（我们见面的那天）
the reason for which he left = the reason why he left（他离开的原因）

介词后面只能用 which/whom,绝不能用 that/who。 且此时关系词不能省。介词也可以留在从句末尾:the man whom I spoke to = the man to whom I spoke。

⚠ "介词+that" 和 "介词+who" 都是错的(in that ✗ / to who ✗)。要么用「介词+which/whom」,要么把介词丢回句末再用 that/省略。

🔧 互动 4 — 介词+which 还原器:演示 where = in which,来回切换两种写法

左边关系副词写法,右边「介词+which」写法,两者完全等价。看清 where 其实就是 in which 的缩写,你就再也不会在两者间纠结。

⑤ as 与 which 的特殊用法

which 指代整个主句(只用 which,不用 that)

非限制性从句里,which 可以指代前面整个主句的内容,译作「这一点/这件事」。

He passed the exam, which surprised everyone.（他通过了考试，这件事让大家吃惊）

as 引导非限制性从句(as we all know 型)

as 也能指代整个主句,常见于固定表达:as we all know / as is reported / as is often the case。as 引导的从句可放句首、句中、句末,which 只能放后面。

As is known to all, the earth is round.（= The earth is round, which is known to all.）
aswhich
位置句首/句中/句末都行只能句末(主句之后)
含义倾向「正如…」(常与已知/客观事实)中性,补充说明
常见搭配as we know, as is said—

⚠ 想把从句放到主句前面时,只能用 as,不能用 which。which 指代主句时永远跟在主句后面。

⑥ that 与 which 的辨析

只能用 that、不用 which 的情况

① 先行词被 all / every / any / no / little / much / few 修饰;② 先行词是 不定代词(all, everything, nothing, something...);③ 先行词有最高级或序数词(the best, the first...);④ 先行词是 the only / the very / the same;⑤ 先行词既有人又有物;⑥ 句中已有 which 作疑问,避免重复。

This is the best film that I have ever seen.（最高级 → 用 that）
All that glitters is not gold.（先行词 all → 用 that）

只能用 which、不用 that 的情况

① 非限制性从句(有逗号);② 介词 + which(介词后);③ which 指代整个主句。

The book, which is red, is mine.（有逗号 → 用 which）

⚠ 记口诀:「最高序数不特代,人物混合 all 无(no) → 只用 that」;「逗号、介词、指全句 → 只用 which」。

⑦ 全题型地图

题型A — 关系代词作主语/宾语

触发信号:先行词是普通人/物,横线后紧跟动词(缺主语)或紧跟主语+动词(缺宾语)。

标准动作:人+缺主语→who/that;人+缺宾语→whom/that/(省);物+缺主/宾→which/that/(宾时可省)。

示例(带解):The girl ___ won the prize is my friend. 从句缺主语、先行词是人 → who/that:The girl who won the prize... The novel ___ he wrote sold well. 缺宾语、物 → which/that/省:The novel (that) he wrote...

陷阱:作主语的关系词不能省;把「缺宾语」误判成「缺主语」而漏了从句的真正主语。

题型B — whose(所属关系)

触发信号:横线后跟一个名词,表示「先行词的某物」(the …'s …),人或物都可以。

标准动作:whose + 名词,= of which / of whom。物也能用 whose(the tree whose leaves = the tree the leaves of which)。

示例(带解):I know a boy ___ father is a pilot. → 「男孩的父亲」= whose father。The house ___ windows face south is warm. → whose windows(= the windows of which)。

陷阱:whose 后直接跟名词,别加 the;别把「物的所属」错以为不能用 whose。

题型C — 关系副词 when/where/why

触发信号:先行词是时间/地点/reason,且横线后的从句成分完整(不缺主也不缺宾)。

标准动作:时间→when;地点→where;reason→why。判据:从句拆开后缺的是「状语(in/on/at + 名词)」。

示例(带解):This is the school ___ I studied. 从句 "I studied" 完整、缺地点状语 → where。对比 the school ___ I visited(缺宾语)→ which/that。

陷阱:先行词是地点≠一定用 where!要看从句缺状语(用 where)还是缺宾语(用 which/that)。

题型D — 介词 + which / whom

触发信号:横线前有介词(in/on/at/for/to/with…),或从句里动词/形容词需要一个介词宾语。

标准动作:介词 + which(物)/ whom(人)。可等价换成关系副词或把介词丢句末。介词后不用 that/who、不可省。

示例(带解):The lady ___ ___ I talked is a teacher. → to whom(= whom I talked to)。The tool ___ ___ he fixed it is here. → with which。

陷阱:in that/to who 全错;介词提前后关系词绝不能省;别忘了 depend on / look after 这类短语动词的介词。

题型E — 非限制性定语从句

触发信号:横线前有逗号,或先行词是专有名词/唯一事物,从句只是补充说明。

标准动作:人→who/whom;物→which;整句→which/as。一律不用 that,不可省。

示例(带解):Beijing, ___ is the capital of China, is beautiful. → which。Mr. Li, ___ taught us English, has retired. → who。

陷阱:逗号后写 that = 送分题写崩;非限制性从句关系词永远不能省。

题型F — as vs which(指代主句)

触发信号:从句指代整个主句,或出现 as we know / as is reported 这类固定表达,或从句在主句前面。

标准动作:从句在主句后 → which 或 as 都行;从句在主句前 → 只能 as;固定习语 → as。

示例(带解):___ is known to all, he is a hero.(句首)→ As。He was late, ___ made the teacher angry.(句末,指全句)→ which(也可 as)。

陷阱:句首指代主句只能用 as,不能用 which;which 指代主句必须紧跟主句之后。

题型G — that 与 which 辨析

触发信号:先行词带最高级/序数/all/every/no/the only/不定代词,或有逗号/介词。

标准动作:「最高序数、all/no/only、人物混合、不定代词」→ that;「逗号、介词后、指全句」→ which。

示例(带解):This is the first thing ___ we should do. → that(序数)。The car, ___ is new, runs fast. → which(逗号)。

陷阱:两条规则一冲突先看有没有逗号/介词(那必须 which);否则再看有没有 that 的强制条件。

⑧ 易错集

① 关系副词 vs 关系代词:唯一判据是「从句缺不缺主/宾」。缺主宾(名词性)→代词 which/that;不缺、只缺状语→副词 where/when/why。地点先行词不等于一定 where。

② 限制 vs 非限制:有逗号=补充(不用 that、不可省);无逗号=限定(可用 that、宾语可省)。专有名词后一般非限制。

③ 介词后不用 that / who:in that ✗、to who ✗。介词后只能 which/whom,且不可省;或把介词移回句末。

④ whose 与 of which:whose+名词(直接);of which 要接在名词后(the leaves of which)。物也能用 whose。

⑤ 作主语的关系词不能省;只有「限制性从句里作宾语」的关系代词才能省。关系副词一律不能省。

⑥ that 的强制场景:最高级、序数、all/every/no/the only、不定代词、人物混合 → 只用 that。

⑦ which 的强制场景:非限制性(逗号)、介词后、指代整个主句 → 只用 which。

⑧ as vs which 指主句:句首只能 as;句末两者皆可;as we know 是固定表达。

⑨ 尖子生拔高视角

① 一切定语从句都能「拆回两句话」: 遇到任何空,先把从句还原成一个独立句子,关系词替代的那个词在原句里当什么(主/宾/所属/状语),就选对应的关系词。这个「拆句法」比背规则可靠得多——规则会打架,句子结构不会。

② 关系副词 = 介词+which 的缩写,这是理解一切的钥匙: where=in which,when=on/at which,why=for which。凡是纠结「该用 which 还是 where」,就把从句补全:若缺的位置能填「介词+名词」(状语)→用副词/介词+which;若缺的是光秃秃的名词(主/宾)→用 which/that。

③ 逗号是语义开关,不是标点小事: 限制性=「哪一个」(区分),非限制性=「顺便说」(补充)。翻译时前者并进「…的名词」,后者拆成「,而它…」。看到专有名词/独一无二的先行词,先默认非限制性。

④ that 与 which 的规则本质是「that 更朴素、which 更正式」: that 不能进非限制性、不能跟在介词后、不能指代整句——因为这些都是「正式/独立」的语境;反过来最高级、all/no、不定代词这些「限定到极致」的先行词偏爱朴素的 that。理解这层气质,规则就不用死记。

⑤ as 与 which 指主句的差别在「视角」: as 自带「正如、如同」的评述口吻,能灵活地放句首(先亮观点再陈述);which 是事后补充,只能跟在主句后。写作时想强调「众所周知/正如所料」,主动用 as 开头,立刻上一个档次。
