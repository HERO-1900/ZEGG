实用类文本阅读(非连续性文本)

这一页讲的是高考语文里最「理科」的一块阅读:几则材料 + 图表拼在一起(非连续性文本)。它不考文学赏析,考的是快速筛选信息、跨材料比对、读懂图表、分点概括。文本来自新闻、通讯、传记、科普、访谈、调查报告。核心一句话:求准、求全、有条理——找对句子、答全要点、分点作答,分就到手。

🗣️ 实用类文本(人话版): 试卷上给你材料一、材料二、材料三,往往还配一张图表,它们围绕同一个话题(比如「乡村振兴」「人工智能」「传统文化传承」)从不同角度说事。考你的不是「写得美不美」,而是「你能不能在一堆信息里快速定位、跨材料比对、看懂图表趋势、把要点分条说清楚」。客观题跟论述类一样防「曲解/以偏概全/无中生有」的坑;主观题就一个铁律:分点 + 关键词 + 结合材料。别写文学腔的排比抒情——这里要的是准确、全面、有条理。
一句话方法: 定位 → 比对 → 概括 → 分点。看到题干先圈出关键词(时间、对象、角度),回到材料定位对应句,再把散落的信息提炼成短语,最后分①②③点写出来。图表题额外看三样:趋势(升/降)、极值(最高/最低)、比较(谁大谁小)。

① 文本类型:先认出这是「哪一种」实用文

实用类文本不是散文小说,它讲究真实、准确、传递信息。高考常见五类,先认出类型,才知道它想让你抓什么:

类型特点阅读要抓什么
新闻/消息时效性强,倒金字塔结构标题、导语(第一句)、时间地点、核心事件
通讯比新闻详细,有细节和评价人物事迹、典型事例、作者态度
传记写人物生平、成就、精神关键经历、性格品质、评价语句
科普解释科学概念/现象核心概念、原理、说明顺序、数据
访谈/调查报告问答体 / 数据+分析受访者观点、问题指向、数据结论

⚠ 别把实用类当文学类读。实用类不问「表达了怎样的情感」,问的是「概括了什么信息、图表说明了什么趋势、几则材料的异同」。答题语言要朴实、精准、分点,不要堆砌华丽词句。

🗂️ 互动 1 — 非连续性文本结构图:材料一/二/三 + 图表 → 共同话题(点开看每则材料的角度)

非连续性文本 = 几则材料围绕同一话题,从不同角度/文体切入。点每块看它的角度。做题第一步:先判断每则材料各说了什么,它们共同指向什么话题。

② 信息筛选与整合:定位关键句是命根

核心动作:题干关键词 → 回材料找对应句 → 提炼。 实用类主观题的分,几乎都藏在材料的某几句话里。你的任务是把它们找出来、翻译成要点,而不是自己发挥。

筛选三步

第1步:审题干,圈关键词(问「原因」还是「措施」?问「特点」还是「意义」?限定哪则材料?)
第2步:回材料定位(找与关键词相关的句子,常在段首、段尾、转折词「但」「然而」「因此」后)
第3步:整合提炼(把找到的句子压缩成短语,同类合并,分点罗列)

定位信号词: 「因此、所以、由此可见」后接结论;「首先、其次、此外、另外」提示并列要点;「但是、然而、相反」提示转折重点;数据、百分比、年份往往是命题点。

⚠ 「筛选」不是「照抄整段」。要提炼关键词,把长句压成短语。也不是「凭印象编」——每个要点都必须在材料里有依据,能指回原句。

🔍 互动 3 — 信息筛选定位器:选题干关键词,材料里的对应句自动高亮

同一段材料,题干问的点不同,该定位的句子就不同。点按钮,看关键词把哪些句子「点亮」。这就是带着问题回材料找句子的过程。

③ 图表解读:读趋势 / 极值 / 比较

图表(柱状图、折线图、饼图、表格)是非连续性文本的常客。读图不是把数字念一遍,而是看出它在说什么规律。抓三样:

看什么怎么读答题措辞
趋势整体是升、降、还是先升后降「呈上升/下降趋势」「逐年增长」「先增后减」
极值最高点、最低点在哪一年/哪一项「X 年达到峰值」「Y 项最低」
比较项与项、组与组谁多谁少、差多少「A 明显高于 B」「城市是农村的两倍」

读图口诀:先看标题和坐标(横轴是什么、纵轴是什么、单位),再看整体趋势,再找极值,最后做比较。 一定要结合数据说话——不能只说「增长了」,要说「从 X 增长到 Y」。

⚠ 别看错趋势(把下降看成上升);别漏单位(是「万人」还是「亿元」);别脱离数据空谈。概括图表时,要点 + 数据支撑缺一不可。

📊 互动 2 — 图表解读演示:切换柱状/折线,看趋势(升降)、极值(最高最低)、比较(谁大谁小)

同一组数据(某市 2019–2023 年数字阅读人数,单位万人)。切换视图:柱状适合比大小,折线适合看趋势,再点极值找最高/最低点。答题时把这三点连成一句话。

④ 跨材料比较:找异同是高频主观题

非连续性文本天生要「比」。常见问法:「几则材料侧重点有何不同?」「材料一和材料三的观点有何异同?」答题=先找共同点(同),再找差异点(异),分材料分点说。

比较三问

同:它们共同关注/围绕的话题、共识是什么?
异:各自的角度不同(如:一则谈成就、一则谈问题、一则谈对策)?文体不同?立场不同?
答:分点写「材料一……;材料二……;材料三……」,或「相同点是……;不同点是……」。

常见「异」的维度: 角度(现象/原因/对策)、文体(新闻/评论/访谈)、对象(城市/农村)、态度(肯定/担忧)、详略(概述/深入)。

⚠ 比较题最忌只答一则或只说「都很好」这类空话。要逐则对应、点出差异维度。问「异同」就必须既写同、又写异,漏一半扣一半。

⚖️ 互动 4 — 主观概括题四步流程:审题 → 定位 → 分点 → 提炼(一步步走完一道真题)

以一道「概括乡村数字化带来的变化」为例,一步步演示答主观概括题的四步法。每步都告诉你手上该做什么、心里在想什么。走完就知道满分答案是怎么「攒」出来的。

⑤ 概括归纳:分点作答、条理清晰

主观题的最终产品是一份分点答案。阅卷是「踩点给分」——你答到一个得分点给一分,答不到不给。所以宁可多分几点,也别把要点糊成一大段。

答题规范(答主观题的「格式」)

① 分点:用 ①②③ 或 (1)(2)(3) 标清,一点一个意思。
② 关键词:每点先给概括性短语(如「政策扶持」「技术下乡」),再简要展开。
③ 结合材料:要点须有材料依据,可点「据材料二」,不能空谈。

要点从哪来? 从材料的段落、图表数据、转折句里来。几则材料的要点要合并同类项(同一意思只算一点),再补全遗漏(不同材料的不同角度都要覆盖)。

⚠ 三大失分:不分点(一大段,阅卷找不到点)、概括不全(只抄一则材料,漏了其他材料的角度)、照抄原句不提炼(字数超了还没答到关键词)。

⑥ 客观题:设错手法与论述类同源

实用类客观题(选择题)的设错套路,跟论述类文本一模一样:选项对材料信息动手脚。识别它们,靠「选项 vs 原文」逐字比对。

设错手法怎么识别
曲解文意选项改了原文的意思(偷换概念、张冠李戴)
以偏概全把「部分/某些」说成「全部/所有」,「可能」说成「一定」
无中生有选项内容原文根本没有,是编造/推不出的
因果混乱强加因果,或把因果关系颠倒
混淆时态把「将要/计划」说成「已经」,已然未然不分

比对铁律: 每个选项都要回原文定位,把选项和原句放一起,盯住范围词(所有/部分)、程度词(一定/可能)、逻辑词(因为/所以)、时间词(已经/将)。这几类词最容易被做手脚。

⚠ 别凭「感觉对」就选。选项读着顺、听着有道理,恰恰是设错高手的伪装。没在原文找到依据,一律存疑。

🎯 互动 5 — 客观题设错识别:读原句,给每个选项做诊断(对 / 曲解 / 以偏概全 / 无中生有)
点选项旁的按钮,判断它错在哪。

先读原文句,再逐个判断四个选项。每个选项点「揭晓」看诊断。练的就是「选项 vs 原文」逐字比对的手感。

⑦ 全题型地图

题型A — 客观信息比对题(选择题)

触发信号:「下列对材料相关内容的理解和分析,正确/不正确的一项是」。四个选项,逐条对材料。

标准动作:每个选项回原文定位 → 逐字比对 → 盯范围词/程度词/因果/时态 → 找出被改动处。

示例(带解):原文:「据调查,约六成受访青年经常使用数字阅读。」选项:「所有青年都经常使用数字阅读。」→ 错(以偏概全):把「约六成受访青年」偷换成「所有青年」,范围扩大。正确选项应保留「约六成/部分」的限定。

陷阱:被「读着顺」骗;不回原文;忽略「所有/一定/已经」等限定词的改动。

题型B — 图表数据解读题

触发信号:材料含柱状图/折线图/表格,问「图表反映了什么」「概括图表信息」。

标准动作:看标题坐标单位 → 说整体趋势 → 指极值 → 做比较,每点结合数据,分点写。

参考答案示例:某市数字阅读人数 2019–2023 年(单位万人):120、150、210、260、300。答:①总体呈逐年上升趋势,五年从 120 万增至 300 万,增长 1.5 倍;②增速加快,2020→2021 增幅最大(增 60 万);③2023 年达到峰值 300 万。(趋势+极值+数据支撑)

陷阱:只念数字不说规律;看错升降;漏单位;不给数据支撑。

题型C — 跨材料异同比较题

触发信号:「几则材料的侧重点/角度有何不同」「材料一与材料三观点异同」。

标准动作:先找同(共同话题/共识),再找异(角度、文体、态度、对象),逐则对应分点。

参考答案示例:问三则材料侧重点。答:相同点:三则都关注乡村数字化这一话题。不同点:①材料一(新闻)侧重报道成效现象(电商助农、直播带货);②材料二(图表)用数据呈现增长趋势;③材料三(访谈)侧重分析问题与对策(人才短缺、基础设施)。

陷阱:只答一则;只写「都很好」的空话;问「异同」却只写「异」漏了「同」。

题型D — 概括原因/措施/特点题(主观分点)

触发信号:「概括……的原因」「请归纳……的措施/特点」「结合材料谈……」。

标准动作:审题定「问的是原因还是措施」→ 跨材料定位相关句 → 提炼短语 → 合并同类 → 分①②③点,每点带材料依据。

参考答案示例:问「乡村数字化带来哪些变化」。答:①拓宽销路——农产品经电商、直播走向全国(材料一);②提升收入——农户增收明显(材料二数据);③改变生活方式——村民线上办事、学习(材料三);④倒逼基础设施升级——网络、物流下乡(材料三)。

陷阱:不分点;只抄一则材料导致概括不全;答成议论文而非要点。

题型E — 内容理解推断题

触发信号:「根据材料,下列推断正确/不正确的一项」「可以推出」。考在原文基础上的合理推断。

标准动作:判断选项是否由原文合理推出——原文有依据且逻辑成立才对;绝对化、强加因果、无中生有则错。

示例(带解):原文:「数字阅读人数逐年增长。」选项:「未来纸质书将彻底消失。」→ 错(推断过度/无中生有):数字阅读增长推不出纸质书「彻底消失」,属绝对化的过度推断。

陷阱:把「可能」推成「必然」;强加因果;脱离原文过度联想。

⑧ 易错集

① 主观题不分点:糊成一大段,阅卷「踩点给分」找不到点。务必 ①②③ 分条。

② 概括不全:只抄材料一,漏了材料二、三的角度。跨材料题必须覆盖各则材料。

③ 图表看错趋势:把下降看成上升、漏单位、只念数字不说规律。要「趋势+极值+数据」。

④ 客观题不回原文:凭感觉选。设错就藏在范围词/程度词/因果/时态里,必须逐字比对。

⑤ 照抄原句不提炼:整段搬,字数爆了还没答到关键词。要压成概括性短语。

⑥ 比较题只写一半:问「异同」只答「异」或只答「同」,直接丢一半分。

⑦ 把实用类当文学类:堆华丽辞藻、写情感赏析。实用类要朴实、精准、分点。

⑧ 推断过度:把「可能」推成「必然」,「增长」推成「彻底取代」,越界即错。

⑨ 尖子生拔高视角

① 主观概括题的四步法(审题→定位→分点→提炼),背下来当肌肉记忆: 拿到题先审——问的是原因/措施/特点/意义?限定哪几则材料?圈准关键词。再定位——带着关键词回材料,找相关句(段首段尾、转折词后、数据处)。再分点——把找到的句子按「不同意思」拆成①②③。最后提炼——每点压成「概括短语+简要展开」。这套流程走顺了,任何主观题都不慌。

② 「合并同类 + 补全遗漏」是拿满分的关键: 几则材料难免有重复,同一意思只算一个得分点(合并);但不同材料的不同角度都是新得分点,不能漏(补全)。拿到多则材料,脑子里要有张「要点清单」,逐则勾选,既不重复也不遗漏。

③ 图表题「趋势—极值—比较」三连,一句话都不能省: 顶尖答案不是罗列数字,而是用数据说规律:先定性(总体上升/下降),再定量(从 X 到 Y,增长几倍),再指极值(某年峰值),再做比较(城乡差距)。每一句话背后都有数据撑腰,阅卷老师一眼就看到你「读懂了图」。

④ 客观题「选项 vs 原文」的六个雷区词: 范围词(所有/部分/都/仅)、程度词(一定/可能/必然)、逻辑词(因为/所以/从而)、时间词(已经/将/正在)、指代词(此/这)、绝对词(彻底/完全/永远)。见到选项里出现这些词,立刻回原文核对——十有八九的坑就设在这里。

⑤ 实用类的底层逻辑是「信息的搬运与重组」,不是「情感的体会」: 想清楚这一点,你的答题语言就会自动切换到「客观、准确、分点」模式。材料给了什么、图表说明什么、几则之间什么关系——你只是把信息准确地搬运并重组,不加个人抒情,不玩文学修辞。这就是实用类和文学类最本质的分野。
