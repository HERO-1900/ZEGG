铝及其化合物(含两性与图像)

铝是高考金属部分的"两面派":既能和强酸反应,又能和强碱反应,一路带出 Al₂O₃、Al(OH)₃ 的两性,以及那道让无数人翻车的"滴加顺序不同、图像不同"的经典题。把两性讲成人话、把图像拨给你看,压轴计算也能稳稳拿下。

🗣️ 把铝想成一个"两面讨好"的老好人: 遇到强酸(HCl)它反应放氢气,遇到强碱(NaOH)它也反应放氢气——酸碱通吃,这就是两性的直觉。
它的氧化物 Al₂O₃、氢氧化物 Al(OH)₃ 也一样,酸来溶、碱来也溶。核心记忆链只有一句:Al → Al³⁺(酸中) 或 AlO₂⁻(碱中),Al(OH)₃ 卡在中间。图像题的全部秘密,就是看你"往哪个方向、谁滴进谁"——先出沉淀还是先无沉淀,顺序一变,曲线就变。抓住这条三角,半壁江山就稳了。
本包导航: ① 铝单质(酸碱通吃 + 铝热 + 钝化) → ② 氧化铝 Al₂O₃(两性氧化物) → ③ 氢氧化铝 Al(OH)₃(两性 + 制备) → ④ 铝三角 Al³⁺/Al(OH)₃/AlO₂⁻ 相互转化 → ⑤ 图像题(滴加顺序决定曲线) → ⑥ 明矾净水与除杂。每节都配可拨动的可视化。

① 铝单质:银白色、活泼、却穿着"防护衣"

铝是银白色、密度小、较活泼的金属。它表面天生有一层致密的氧化膜(Al₂O₃)保护自己,所以看起来"不怕水不怕空气"。一旦膜被破坏(如汞齐化、酸碱侵蚀),里面的铝就非常活泼。

酸碱通吃:两条都放 H₂↑

与强酸:2Al + 6HCl → 2AlCl₃ + 3H₂↑　(铝被氧化为 Al³⁺)
与强碱:2Al + 2NaOH + 2H₂O → 2NaAlO₂ + 3H₂↑　(生成偏铝酸钠)

⚠ 铝与 NaOH 反应,不是"NaOH 里的钠或氧被还原"!被还原的是水中的 H(H⁺→H₂),被氧化的是铝(Al→AlO₂⁻ 中 +3 价)。NaOH 只是把生成的 Al(OH)₃ 溶掉、让反应持续。

铝热反应(高温、放大量热,用于焊接钢轨)

2Al + Fe₂O₃ →(高温) Al₂O₃ + 2Fe　(铝把铁从氧化物里"抢"出来)

本质是铝的还原性强,能夺取金属氧化物里的氧。放热极多,足以熔化生成的铁 → 现场焊接铁轨。类似地可还原 MnO₂、Cr₂O₃、V₂O₅ 等。

常温钝化(冷的浓硫酸、浓硝酸)

常温下铝(和铁)遇浓 H₂SO₄、浓 HNO₃ 表面被迅速氧化成致密氧化膜而钝化,阻止反应继续 → 所以可用铝槽车运输冷浓硝酸。注意是"常温、浓"三个字缺一不可(加热或稀酸就会继续反应)。

交互 1 · 铝:遇酸放 H₂、遇碱也放 H₂

点一种试剂,看铝片投进去后气泡(H₂)不断冒出的动画,以及对应的化学方程式。两边都放氢气,这就是铝"两面派"的直观证据。

② 氧化铝 Al₂O₃:典型的两性氧化物

Al₂O₃ 白色固体,熔点很高(约 2050℃)→ 用作耐火材料(耐火坩埚、耐火管)。天然刚玉、红蓝宝石的主要成分也是 Al₂O₃。

两性:酸也溶、碱也溶

与强酸:Al₂O₃ + 6HCl → 2AlCl₃ + 3H₂O
与强碱:Al₂O₃ + 2NaOH → 2NaAlO₂ + H₂O

两性氧化物定义:既能与酸反应生成盐和水,又能与碱反应生成盐和水的氧化物。Al₂O₃ 是最经典的例子。

⚠ 电解熔融 Al₂O₃ 是工业炼铝的方法(加冰晶石降低熔点),而不能电解熔融 AlCl₃(AlCl₃ 是共价化合物、熔融不导电)。这是"物质类别"隐藏考点。

③ 氢氧化铝 Al(OH)₃:两性氢氧化物 + 制备陷阱

Al(OH)₃ 是白色胶状沉淀,不溶于水;有较强吸附性(可吸附色素、杂质)。医药上用作胃酸中和剂(中和过多胃酸而不刺激):Al(OH)₃ + 3HCl → AlCl₃ + 3H₂O。

两性:酸溶生成铝盐,碱溶生成偏铝酸盐

与强酸:Al(OH)₃ + 3HCl → AlCl₃ + 3H₂O　(离子:Al(OH)₃ + 3H⁺ → Al³⁺ + 3H₂O)
与强碱:Al(OH)₃ + NaOH → NaAlO₂ + 2H₂O　(离子:Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O)

⚠ Al(OH)₃ 只溶于强酸强碱!不溶于弱酸弱碱(如氨水、碳酸)。所以向铝盐里加氨水生成的 Al(OH)₃ 不会被过量氨水溶解——这正是实验室制备它的关键。

实验室制备:用可溶铝盐 + 氨水(不用 NaOH!)

AlCl₃ + 3NH₃·H₂O → Al(OH)₃↓ + 3NH₄Cl　(离子:Al³⁺ + 3NH₃·H₂O → Al(OH)₃↓ + 3NH₄⁺)

⚠ 为什么不用 NaOH? 因为 NaOH 是强碱,一旦过量就会把刚生成的 Al(OH)₃ 溶解成 AlO₂⁻,得不到沉淀。氨水是弱碱,溶不掉 Al(OH)₃,加多也不怕 → 制备 Al(OH)₃ 首选氨水。

受热分解

2Al(OH)₃ →(Δ) Al₂O₃ + 3H₂O　(不稳定,加热分解为氧化铝和水)

④ 铝三角:Al³⁺ ⇌ Al(OH)₃ ⇌ AlO₂⁻ 相互转化

🗣️ 三个"铝家族成员",两两之间靠加酸或加碱来回跑。 Al³⁺ 在酸的地盘,AlO₂⁻ 在碱的地盘,Al(OH)₃ 卡在中间当"中转站"。想搞懂所有图像题,先把这张三角的六条箭头背熟。

六条转化路线(全部要会写离子方程式)

转化加什么离子方程式
Al³⁺ → Al(OH)₃适量碱/氨水Al³⁺ + 3NH₃·H₂O → Al(OH)₃↓ + 3NH₄⁺
Al³⁺ → AlO₂⁻过量 NaOHAl³⁺ + 4OH⁻ → AlO₂⁻ + 2H₂O
Al(OH)₃ → Al³⁺强酸Al(OH)₃ + 3H⁺ → Al³⁺ + 3H₂O
Al(OH)₃ → AlO₂⁻强碱Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O
AlO₂⁻ → Al(OH)₃适量酸/通 CO₂AlO₂⁻ + CO₂ + 2H₂O → Al(OH)₃↓ + HCO₃⁻
AlO₂⁻ → Al³⁺过量强酸AlO₂⁻ + 4H⁺ → Al³⁺ + 2H₂O

⚠ 注意"适量"与"过量"的区别:Al³⁺ 加过量 NaOH → 直接到 AlO₂⁻(1:4);AlO₂⁻ 加过量盐酸 → 直接到 Al³⁺(1:4)。而通 CO₂(弱酸)只能把 AlO₂⁻ 变到 Al(OH)₃,再多也不会溶(弱酸溶不掉 Al(OH)₃)。

⑤ 图像题:滴加顺序不同,曲线就不同(经典压轴)

这是铝章节最会"分尖子生"的题。核心只有一句:看谁滴进谁、谁先过量。 沉淀量随滴入体积的变化(先增后减、先无后有…)全由顺序决定。

情形 A:向 AlCl₃ 中逐滴加 NaOH(碱滴进酸)

开始 NaOH 少,先生成沉淀并增多:Al³⁺ + 3OH⁻ → Al(OH)₃↓;沉淀到最大后继续加 NaOH,沉淀被溶解减少:Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O。→ 曲线"先上升到顶,再下降到 0"。生成用 3 份 OH⁻、溶解用 1 份 OH⁻ → 前后体积比 3:1。

情形 B:向 NaOH 中逐滴加 AlCl₃(酸滴进碱)

开始 NaOH 过量,滴入的 Al³⁺ 直接变成 AlO₂⁻,一开始无沉淀:Al³⁺ + 4OH⁻ → AlO₂⁻ + 2H₂O;等 NaOH 被耗完后再滴 AlCl₃ 才开始产生沉淀并增多。→ 曲线"先平(无沉淀),后上升"。

情形 C:向 NaAlO₂ 中逐滴加盐酸(酸滴进偏铝酸盐)

先生成沉淀并增多:AlO₂⁻ + H⁺ + H₂O → Al(OH)₃↓;沉淀到最大后继续加盐酸,沉淀被溶解:Al(OH)₃ + 3H⁺ → Al³⁺ + 3H₂O。→ 曲线"先上升到顶,再下降到 0"。生成用 1 份 H⁺、溶解用 3 份 H⁺ → 前后体积比 1:3。

⚠ 记这三条口诀:碱滴 Al³⁺ = 先增后减(3:1);Al³⁺ 滴碱 = 先无后增;酸滴 AlO₂⁻ = 先增后减(1:3)。方向一反,图像天差地别。

⑥ 明矾净水与除杂应用

明矾 化学式 KAl(SO₄)₂·12H₂O(十二水合硫酸铝钾),是一种复盐,溶于水完全电离出 K⁺、Al³⁺、SO₄²⁻。

净水原理:Al³⁺ 水解生成 Al(OH)₃ 胶体

Al³⁺ + 3H₂O ⇌ Al(OH)₃(胶体) + 3H⁺

Al³⁺ 水解生成的 Al(OH)₃ 胶体表面积大、有很强的吸附能力,能吸附水中悬浮的泥沙、杂质使其聚沉,从而净水。注意:明矾只能净水(除去悬浮杂质),不能杀菌消毒(杀菌用氯气、漂白粉等)。

除杂思路(利用两性)

混合物除杂方法原理
Al₂O₃ 中混有 Fe₂O₃加过量 NaOH,过滤Al₂O₃ 溶(→AlO₂⁻),Fe₂O₃ 不溶被滤掉
Fe₂O₃ 中混有 Al₂O₃加过量 NaOH,取滤渣同上,留下的不溶滤渣即 Fe₂O₃
Mg 中混有 Al加过量 NaOH,过滤Al 与 NaOH 反应溶掉,Mg 不反应留下
MgCl₂ 溶液中混有 AlCl₃加过量 NaOH 再过滤Al(OH)₃ 溶成 AlO₂⁻,Mg(OH)₂ 不溶被滤除

⚠ 利用铝的两性除杂:凡是"铝和某金属/其化合物分开"的题,几乎都是"加过量 NaOH,谁溶谁被除或谁留谁被留"。想清楚要保留的是溶液还是滤渣。

⑧ 易错集(高频翻车点)

1. 制备 Al(OH)₃ 用氨水不用 NaOH。NaOH 过量会把沉淀溶成 AlO₂⁻;氨水(弱碱)溶不掉,加多也不怕。

2. 滴加顺序不同,现象/图像不同:碱滴 Al³⁺ 先沉后溶(3:1);Al³⁺ 滴碱先无沉淀后有;酸滴 AlO₂⁻ 先沉后溶(1:3)。

3. 铝与 NaOH 反应,是铝被氧化、水中的 H 被还原放出 H₂;NaOH 中的钠、氧都没变价。

4. 偏铝酸盐(AlO₂⁻)遇过量盐酸:先生成 Al(OH)₃ 沉淀又被溶解(→Al³⁺);但通 CO₂(弱酸)只到 Al(OH)₃ 就停,不再溶。

5. Al(OH)₃ 只溶于强酸强碱,不溶于氨水、碳酸等弱酸弱碱。

6. 电解熔融 Al₂O₃ 炼铝(加冰晶石),不能电解熔融 AlCl₃(共价、熔融不导电);也不能电解 AlCl₃ 水溶液(会先放氢)。

7. 常温钝化必须是浓 H₂SO₄/HNO₃ 且常温;加热或用稀酸会继续反应放气(稀硝酸不放 H₂,放 NO)。

8. 明矾能净水(吸附聚沉)但不能消毒杀菌;两者是不同功能,别混。

⑨ 尖子生拔高视角

▸ 把"两性"统一成一句话: 铝家族只在中间的 Al(OH)₃ 是"客观存在的沉淀",两端 Al³⁺(强酸区)和 AlO₂⁻(强碱区)都是澄清的。图像的所有拐点,都是"沉淀出现"或"沉淀消失"这两件事。

▸ 图像量化——记住三个体积比: 碱滴 Al³⁺,生成:溶解 = 3:1(耗 OH⁻ 之比);酸滴 AlO₂⁻,生成:溶解 = 1:3(耗 H⁺ 之比)。会这两个比,沉淀-体积图的横坐标拐点就能精确定位、反算浓度。

▸ "先加谁"决定初始斜率: 少量试剂滴入大量另一方 → 另一方过量 → 滴入的先"走到底"(如 Al³⁺ 滴入过量碱先全变 AlO₂⁻,故初段无沉淀)。用"过量方决定去向"一招通杀顺序题。

▸ 与其它两性的横向联系: Zn、ZnO、Zn(OH)₂ 也两性(考纲要求较低,但同一套逻辑)。把 Al 的模型迁移过去,遇到"既溶于酸又溶于碱"就条件反射想两性。

▸ 计算题技巧——守恒 + 极值: 混合滴加题用"铝原子守恒""电荷守恒"列式;沉淀最大值就是"全部铝以 Al(OH)₃ 存在"那一刻,用 n(Al) 直接算 m(Al(OH)₃)。
