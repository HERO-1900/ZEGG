钠及其化合物

从银白色的活泼金属钠,到 Na₂O₂ 的"供氧魔法",再到纯碱与小苏打这对孪生兄弟——一条"钠家族"的转化链条。把每个反应讲成人话、再动手拨一拨,选择填空里的钠题就能稳稳拿分。

🗣️ 把钠家族想成一个"活泼又善变"的大家庭: 钠单质是家里最"暴脾气"的——见水就炸、见氧就变,所以只能泡在煤油里"冷静"。它烧出来有两个儿子:低调的 Na₂O(普通碱性氧化物)和身怀绝技的 Na₂O₂(能吸 H₂O、CO₂ 放氧气,是潜水艇和防毒面具的"氧气罐")。
再往下,家族里最出名的是一对孪生兄弟:纯碱 Na₂CO₃ 稳重、小苏打 NaHCO₃ 娇气(一加热就分解)。考试就考:谁和谁反应、放不放气、加热分不分解、怎么鉴别、怎么互相转化。抓住"钠先跟水反应、Na₂O₂里氧是−1价自己歧化、小苏打热不稳定"这几句,钠这一章就通了。
本包导航: ① 钠单质 → ② 氧化钠 vs 过氧化钠 → ③ 氢氧化钠 → ④ 碳酸钠 vs 碳酸氢钠 → ⑤ 焰色反应与侯氏制碱法。每节都配可拨动的可视化;最后是全题型地图 + 易错集 + 拔高 + 自测。

① 金属钠(Na)

钠是最典型的活泼金属:银白色、质软(可用刀切)、密度比水小、熔点低。化学性质极活泼,极强的还原性(Na − e⁻ → Na⁺)。

钠与氧气:温度不同,产物不同

常温:4Na + O₂ = 2Na₂O　(白色)
加热(点燃):2Na + O₂ Δ Na₂O₂　(淡黄色)

这是一个高频对比考点:同样是钠与氧气,常温慢慢氧化生成 Na₂O,加热剧烈燃烧、发黄光,生成过氧化钠 Na₂O₂。

钠与水:剧烈反应,五字现象"浮熔游响红"

2Na + 2H₂O = 2NaOH + H₂↑
现象说明的性质
浮 浮在水面钠的密度比水小
熔 熔成小球反应放热 + 钠熔点低
游 四处游动生成的 H₂ 推动小球
响 发出嘶响反应剧烈、放出气体
红 使酚酞变红生成了碱性的 NaOH

钠与盐溶液:先与水反应!

钠投入盐溶液(如 CuSO₄、FeCl₃ 溶液),不会置换出金属,而是先与水反应生成 NaOH 和 H₂,再由 NaOH 与盐反应生成沉淀。

2Na + 2H₂O = 2NaOH + H₂↑　然后　2NaOH + CuSO₄ = Cu(OH)₂↓ + Na₂SO₄

保存: 钠易与空气中 O₂、水反应,故保存在煤油(或石蜡油)中,隔绝空气与水(钠的密度大于煤油,沉在底部)。

⚠ 钠"活泼到"根本挤不到盐里去换金属——它先把水"搞定"了。所以"钠置换出铜"是错的。

② 氧化钠 Na₂O vs 过氧化钠 Na₂O₂

🗣️ Na₂O 是"老实"的碱性氧化物;Na₂O₂ 是"魔术师"。 关键差别在氧的化合价:Na₂O 里氧是 −2 价(正常),Na₂O₂ 里氧是 −1 价(过氧根 O₂²⁻),不上不下,所以既能升(被氧化)又能降(被还原)——反应时自己"歧化"。

项目Na₂O(氧化钠)Na₂O₂(过氧化钠)
颜色白色淡黄色
氧的化合价−2−1
类别碱性氧化物过氧化物(非碱性氧化物)
与 H₂ONa₂O+H₂O=2NaOH2Na₂O₂+2H₂O=4NaOH+O₂↑
与 CO₂Na₂O+CO₂=Na₂CO₃2Na₂O₂+2CO₂=2Na₂CO₃+O₂
氧化性/漂白无强氧化性,可漂白

Na₂O₂ 的核心:氧的"自身歧化"

在 2Na₂O₂+2H₂O=4NaOH+O₂↑ 中,Na₂O₂ 里的 O 由 −1 价:一半升到 O₂ 的 0 价(被氧化),一半降到 NaOH 中的 −2 价(被还原)。所以 Na₂O₂ 既是氧化剂又是还原剂,H₂O、CO₂ 只是"配角",不参与化合价变化。

2Na₂O₂ + 2CO₂ = 2Na₂CO₃ + O₂　每 2 mol Na₂O₂ 转移 2 mol e⁻,放出 1 mol O₂

用途: 供氧剂(潜水艇、呼吸面具、防毒面具)——吸收人呼出的 CO₂ 和水蒸气,同时放出 O₂。

⚠ 放出的 O₂ 来自 Na₂O₂ 自身的歧化,不是来自 H₂O 或 CO₂!且 Na₂O₂ 不是碱性氧化物(碱性氧化物与水只生成碱,不放氧)。

交互 2 · Na₂O₂ 与 H₂O / CO₂:放氧 + 电子转移(歧化)

选一个反应,点"运行",看 Na₂O₂ 分解:一半氧升为 O₂ 冒出、一半氧降入产物;弧线标出电子转移。理解"氧化剂=还原剂=Na₂O₂"。

③ 氢氧化钠 NaOH(烧碱 · 火碱 · 苛性钠)

白色固体,易潮解(可作干燥剂,但不能干燥酸性气体如 CO₂、SO₂)、极易溶于水且放热,是强碱。

三大类反应

① 与酸中和:NaOH+HCl=NaCl+H₂O。

② 与酸性氧化物:CO₂+2NaOH=Na₂CO₃+H₂O(CO₂少量);CO₂+NaOH=NaHCO₃(CO₂过量)。

③ 与两性金属/氧化物:2Al+2NaOH+2H₂O=2NaAlO₂+3H₂↑;Al₂O₃+2NaOH=2NaAlO₂+H₂O。

⚠ 潮解 ≠ 风化。NaOH 潮解(吸水变潮),同时会吸收空气里 CO₂ 变质生成 Na₂CO₃,所以敞口久置的 NaOH 常含 Na₂CO₃。

交互 3 · CO₂ 通入 NaOH:少量 → Na₂CO₃,过量 → NaHCO₃

拖动"通入 CO₂ 的量",看产物如何随 CO₂/NaOH 比例变化:先生成 Na₂CO₃,CO₂ 继续增多则转成 NaHCO₃。

④ 碳酸钠 Na₂CO₃ vs 碳酸氢钠 NaHCO₃

🗣️ 纯碱 Na₂CO₃(苏打)稳重,小苏打 NaHCO₃ 娇气。 记三件事:①谁更"碱"——同浓度 Na₂CO₃ 碱性更强;②谁怕热——NaHCO₃ 一加热就分解;③谁和酸反应更猛——NaHCO₃ 更快更剧烈。
项目Na₂CO₃(纯碱/苏打)NaHCO₃(小苏打)
俗名纯碱、苏打小苏打
溶解性易溶(溶解度更大)可溶(溶解度较小)
水溶液强碱弱酸盐,水解显较强碱性水解显弱碱性
热稳定性稳定,加热不分解不稳定,受热分解
与盐酸反应,放 CO₂(稍缓)反应更剧烈更快,放 CO₂

关键方程式

受热分解:2NaHCO₃ Δ Na₂CO₃ + H₂O + CO₂↑
与盐酸:Na₂CO₃ + 2HCl = 2NaCl + H₂O + CO₂↑
与盐酸:NaHCO₃ + HCl = NaCl + H₂O + CO₂↑

相互转化(必背双向)

Na₂CO₃ + CO₂ + H₂O = 2NaHCO₃　(通 CO₂:碳酸钠 → 碳酸氢钠)
2NaHCO₃ Δ Na₂CO₃ + H₂O + CO₂↑　(加热:碳酸氢钠 → 碳酸钠)
NaHCO₃ + NaOH = Na₂CO₃ + H₂O　(加碱:碳酸氢钠 → 碳酸钠)

鉴别方法

① 加热法:分别加热,产生使澄清石灰水变浑浊气体的是 NaHCO₃(Na₂CO₃ 不分解)。

② 盐酸法:滴加同浓度盐酸,产气更快更剧烈的是 NaHCO₃。

③ BaCl₂/CaCl₂ 溶液法:产生白色沉淀的是 Na₂CO₃(BaCO₃↓);NaHCO₃ 不沉淀。

⚠ 除去 Na₂CO₃ 固体中的 NaHCO₃:加热即可(NaHCO₃ 分解成 Na₂CO₃)。但除去 NaHCO₃ 溶液中的 Na₂CO₃:应通入足量 CO₂(Na₂CO₃+CO₂+H₂O=2NaHCO₃),绝不能加热!

⑤ 焰色反应与鉴别 · 侯氏制碱法

焰色反应是元素的物理性质(不是化学变化):把待测物蘸在洁净铂丝(或铁丝)上,在酒精灯火焰上灼烧,观察火焰颜色。

元素焰色备注
钠 Na黄色只要含 Na 就显黄,极灵敏
钾 K紫色需透过蓝色钴玻璃观察(滤掉钠的黄光)

操作要点: 铂丝先蘸稀盐酸灼烧至无色 → 再蘸样品灼烧;换样品前重复清洗。

侯氏制碱法(联合制碱法)简介

我国化学家侯德榜发明,以食盐(NaCl)、氨(NH₃)、二氧化碳(CO₂)为原料制纯碱 Na₂CO₃。核心:向饱和食盐水中先通 NH₃ 再通 CO₂,析出溶解度较小的 NaHCO₃:

NaCl + NH₃ + CO₂ + H₂O = NaHCO₃↓ + NH₄Cl
再加热:2NaHCO₃ Δ Na₂CO₃ + H₂O + CO₂↑(CO₂ 循环利用)

副产品 NH₄Cl 可作氮肥;优点是原料利用率高、无大量废弃物,是化学工业的骄傲。

⑦ 易错集(高频翻车点)

1. 钠投入盐溶液能"置换出金属"——错!钠先与水反应生成 NaOH 和 H₂,再由 NaOH 与盐反应(生成沉淀,不生成金属单质)。

2. Na₂O₂ 与 H₂O、CO₂ 反应放出的 O₂ 来自 H₂O 或 CO₂——错!O₂ 来自 Na₂O₂ 自身的歧化(O 由 −1 价一半升 0、一半降 −2)。

3. 把 Na₂O₂ 当成碱性氧化物——错!碱性氧化物与水只生成碱、与酸只生成盐和水,而 Na₂O₂ 与水放 O₂,是过氧化物。

4. 认为 NaHCO₃ 热稳定——错!NaHCO₃ 受热易分解,Na₂CO₃ 才稳定。这是热稳定性对比的核心。

5. 除 NaHCO₃ 溶液中的 Na₂CO₃ 用"加热"——错!加热会把 NaHCO₃ 也分解掉,应通入足量 CO₂。(除 Na₂CO₃ 固体中的 NaHCO₃ 才用加热。)

6. 钠与氧气产物写反:常温 Na₂O、加热 Na₂O₂,别记反。

7. 焰色反应写成"化学变化"——错,它是元素的物理性质;看钾焰色要透过蓝色钴玻璃。

8. NaOH 潮解可作干燥剂,但不能干燥 CO₂、SO₂、HCl 等酸性气体(会反应)。

⑧ 尖子生拔高视角

▸ Na₂O₂ 的"守恒魔法": 2Na₂O₂+2CO₂ 与 2Na₂O₂+2H₂O 中,可把 CO₂、H₂O 看成"CO₂·H₂O 的组合"。混合气(CO 与 H₂ 燃烧产物、或 CH₄ 燃烧产物)通过足量 Na₂O₂,固体增重等于"通入气体的质量减去放出 O₂ 的质量";凡是符合 (CO)ₘ(H₂)ₙ 通式的气体,固体增重等于气体本身质量。这类"差量/守恒"题是压轴常客。

▸ 电子转移量速算: Na₂O₂ 无论与 H₂O 还是 CO₂ 反应,每 2 mol Na₂O₂ 放 1 mol O₂、转移 2 mol e⁻。抓住这个比例,气体量与转移电子数一并秒算。

▸ 向 Na₂CO₃ 溶液逐滴滴盐酸的"分步"图像: 先 CO₃²⁻+H⁺=HCO₃⁻(不放气),盐酸滴完这一步后再 HCO₃⁻+H⁺=H₂O+CO₂↑(才放气)。所以"先无气、后放气",且两段消耗盐酸体积相等。若把两者互滴顺序反过来(把 Na₂CO₃ 滴入盐酸),则一开始盐酸过量,直接放气——顺序不同现象不同,是鉴别 Na₂CO₃ 与盐酸的经典无试剂鉴别。

▸ 用"钠守恒"贯穿转化链: Na → Na₂O₂ → NaOH → Na₂CO₃ → NaHCO₃,无论怎么变,钠元素总量守恒。计算题里抓住 n(Na) 不变,常能一步列式。

▸ NaHCO₃ 与 NaOH、与盐酸都反应(既能与酸又能与碱),体现 HCO₃⁻ 的"两性"行为;而 Na₂CO₃ 只与酸反应。这决定了它们在除杂、转化中的不同角色。
