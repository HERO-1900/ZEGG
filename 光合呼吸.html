光合作用 / 呼吸作用全攻坚

光反应、暗反应、有氧/无氧呼吸、真净光合的关系、影响因素曲线、限制因子、密闭装置昼夜变化、农业增产——把"气体进出"理顺,曲线题一眼看穿。

🗣️ 光合 vs 呼吸(人话版): 把叶肉细胞想成一座带太阳能板的工厂。光合作用是"充电+造货":太阳能板(光反应,在类囊体)把光变成电池(ATP)和还原剂(NADPH),再用它们把进货的 CO₂ 拼装成糖(暗反应,在基质)。呼吸作用是"烧货发电":把糖拆开放能,供细胞日常开销。
关键账本:真(总)光合 = 净(表观)光合 + 呼吸。你从外面"看到"的(净光合)只是工厂产量减去自己消耗后剩下的那部分。

① 光合作用:光反应 + 暗反应

光反应在类囊体薄膜上,把光能转成 ATP 和 NADPH,并光解水放出 O₂;暗反应在叶绿体基质里,用 ATP/NADPH 把 CO₂ 固定成糖。

光反应:H₂O →(光、色素)→ O₂↑ + [H](NADPH) + ATP
暗反应:CO₂ + C₅ →(固定)→ 2C₃ →([H]、ATP 还原)→ (CH₂O) + C₅
光反应暗反应(卡尔文循环)
场所类囊体薄膜叶绿体基质
条件光、色素、酶多种酶(需光反应供能)
原料H₂O、ADP、Pi、NADP⁺CO₂、C₅、ATP、NADPH
产物O₂、ATP、NADPH(CH₂O)、ADP、Pi、NADP⁺、C₅

色素与吸收光谱

叶绿体色素含叶绿素 a、叶绿素 b(主吸收红光、蓝紫光,呈绿色)和类胡萝卜素(胡萝卜素、叶黄素,吸收蓝紫光)。可用纸层析法分离:扩散最快(最上)的是胡萝卜素,其次叶黄素、叶绿素 a、叶绿素 b。

⚠ 红光、蓝紫光被吸收最多 → 用绿光照射光合最弱。提取色素用无水乙醇,加 SiO₂(研磨充分)和 CaCO₃(防色素被破坏)。

② 呼吸作用:有氧三阶段 + 无氧两类

有氧呼吸(三阶段)

阶段场所反应产能
一细胞质基质葡萄糖→2丙酮酸 + [H]少量 ATP
二线粒体基质丙酮酸 + H₂O → CO₂ + [H]少量 ATP
三线粒体内膜[H] + O₂ → H₂O大量 ATP
C₆H₁₂O₆ + 6O₂ + 6H₂O → 6CO₂ + 12H₂O + 大量能量

无氧呼吸(两类,均在细胞质基质)

乳酸型:C₆H₁₂O₆ → 2 乳酸(C₃H₆O₃) + 少量能量　【人/动物、马铃薯块茎、乳酸菌】
酒精型:C₆H₁₂O₆ → 2 酒精 + 2CO₂ + 少量能量　【酵母菌、多数植物】

⚠ 无氧呼吸第一阶段与有氧相同(都在细胞质基质生成丙酮酸);区别在第二步。乳酸型不放 CO₂,酒精型放 CO₂——这是判断呼吸类型的关键气体信号。

③ 真光合 = 净光合 + 呼吸(账本拆解)

实验测得的是净(表观)光合(如叶片向外释放的 O₂、装置内 CO₂ 的减少)。真(总)光合还要加回细胞自己呼吸消耗掉的那部分。

三条等式记死:
真光合(总) = 净光合(表观) + 呼吸
真光合 O₂ 产生量 = 释放 O₂(净) + 呼吸耗 O₂
真光合 CO₂ 消耗量 = 吸收 CO₂(净) + 呼吸放 CO₂

⑤ 补偿点与饱和点(交互图)

光补偿点:光合 = 呼吸,净光合 = 0(表观无气体进出)。
光饱和点:再增光合速率不再上升(此时限制因子变为 CO₂/温度)。
CO₂ 补偿点:CO₂ 浓度低到使光合 = 呼吸的那个浓度。

⑥ 真/净光合的实验测定

测什么方法读数含义
净光合(表观)光下测装置内 O₂ 增加 / CO₂ 减少= 真光合 − 呼吸
呼吸黑暗(遮光)测 O₂ 消耗 / CO₂ 增加= 呼吸量
真(总)光合净光合 + 呼吸(两次实验相加)真实合成量

黑白瓶法(水生)

白瓶(透光)既光合又呼吸 → 测净光合;黑瓶(遮光)只呼吸 → 测呼吸;初始瓶测起始溶氧。真光合 = (白瓶−初始) + (初始−黑瓶) = 白瓶 − 黑瓶。

⚠ 必须设黑暗(或黑瓶)单独测呼吸,否则只能得净光合,无法算真光合。这是实验题最常考的"为什么还要再做一组"。

⑦ 限制因子判别器

"短板原理":多个因素同时作用时,光合速率由当前最缺的那个(限制因子)决定。曲线上"还在上升"的横轴变量就是限制因子;"已平台"则限制因子是别的。

限制因子判别器

0→强30
0→高80
低→高50

判别逻辑:谁离"够用"最远,谁就是当前限制因子;提高它最有效。

⑧ 密闭装置昼夜气体变化

密闭容器里放绿色植物,白天净放 O₂、净吸 CO₂(光合 > 呼吸);夜间只呼吸,放 CO₂、吸 O₂。一昼夜的净变化看光合与呼吸总量谁大。

⑨ 农业应用:增产措施

目标措施原理
多光合合理密植、间作套种、延长光照时间(补光)增大有效受光面积/时间
提 CO₂大棚增施 CO₂(气肥)、增施有机肥提高暗反应原料浓度
调温度大棚白天适当升温、夜间适当降温白天促光合,夜间抑呼吸、减少有机物消耗
减消耗夜间降温、贮藏作物低温低氧降低呼吸速率,保住有机物

核心思路:白天想方设法把真光合做大,夜间想方设法把呼吸压小,净积累才最多。

⑩ 易错集

① 测到的是"净光合",别忘了加回呼吸才是"真光合"。

② 光补偿点处净光合=0,但真光合≠0(等于呼吸量),气体表观无进出 ≠ 没有光合。

③ 无氧呼吸全过程在细胞质基质;乳酸型不放 CO₂,酒精型放 CO₂。

④ 光反应产 [H] 是 NADPH;呼吸产 [H] 是 NADH——书写别混。

⑤ 突然停光:暗反应中 C₃ 积累↑、C₅ 减少↓([H]/ATP 供应中断);突然降 CO₂:C₃↓、C₅↑。要会判这两组反向变化。

⑥ 限制因子:曲线"平台段"提高横轴变量无效,要提高别的因素才有用。

⑦ 类胡萝卜素不能直接进行光反应放氧,只是辅助色素吸光后传给叶绿素 a。

⑪ 尖子生拔高视角

▸ "瞬时变化"四连判:停光 → C₃↑C₅↓(ATP/[H]断供,固定还在、还原停);来光 → C₃↓C₅↑;升 CO₂ → C₃↑C₅↓;降 CO₂ → C₃↓C₅↑。记住"谁多了就堵在它前面"。

▸ 气体守恒法解装置题:密闭装置内 CO₂ 与液滴的变化 = 光合 CO₂ 净吸 − 呼吸 CO₂ 净放;若用 NaOH 吸收 CO₂,则液滴只反映 O₂ 变化(净光合放氧或呼吸耗氧)。先想"瓶里谁在动"。

▸ 限制因子的"对照设计":要证明某因素是限制因子,固定其他因素只变它,看速率是否上升;上升即为限制因子。这是探究题标准答法。

▸ 真光合的两套口径换算:O₂ 口径与 CO₂ 口径在(CH₂O)合成时按 1:1 对应(呼吸商=1 的理想情形),葡萄糖合成需 6CO₂ 配 6O₂,做题时统一单位再相加。
