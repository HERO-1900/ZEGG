非谓语动词

一个句子只能有一个谓语动词,剩下的动词都得"退位"变成非谓语——不定式 to do、动名词/现在分词 doing、过去分词 done。它们不再管时态人称,而是去当主语、宾语、定语、状语、补语。判断只靠两问:①它在句中作什么成分 ②它和自己的逻辑主语是主动(doing)还是被动(done)、是将来/目的(to do)还是已完成。抓住这两问,整章融会贯通。

🗣️ 非谓语(人话版): 英语一个句子里只能有一个真正的谓语动词(有时态、跟主语一致的那个)。可其他动作也想出场怎么办?就让它们换装:想表"目的/将来"就穿 to do;想表"主动进行"就穿 doing;想表"被动完成"就穿 done;想把一件事"名词化"当主/宾语就用动名词 doing。看到题先别慌,只问自己两句话:这个动词在句子里干什么活(成分)?它和它的主人是"施动"还是"受动"? 答完这两问,该填 to do / doing / done 立刻见分晓。
一句话口诀: 一句一谓语,余皆非谓语;先看啥成分,再看主被动。主动进行 doing,被动完成 done,将来目的 to do。 逻辑主语(动作真正的发出者)是判断主被动的钥匙——它"做"这个动作用 doing,它"被"这个动作用 done。

① 三种非谓语形式:to do / doing / done

to do　·　doing　·　done

同一个动词,退位成非谓语后有三副面孔,各管一片语义:

形式核心语义典型例
to do 不定式将来 / 目的 / 一次性、待发生I came here to see you.（目的）
doing 动名词抽象 / 习惯性 / 把动作"名词化"Reading is fun.（当主语）
doing 现在分词主动 / 正在进行the running water（正在流的水）
done 过去分词被动 / 已完成the broken window（被打破的窗）

⚠ doing 有两个身份:当"名词"用叫动名词(Reading is fun),当"形容词/表主动进行"用叫现在分词(the running water)。它俩长得一样但功能不同,别纠结名称,抓功能。

🎯 互动 1 — "一句一谓语"演示:多个动词时,只留一个谓语,其余变非谓语

红块=真正的谓语动词(有时态);其余动词被"降级",按语义穿上 to do / doing / done。一句话里谓语只能有一个,看它怎么把别的动词挤成非谓语。

② 作什么成分:主 / 宾 / 定 / 状 / 补

非谓语能顶替名词、形容词、副词的几乎所有位置。判断第一问就是"它在句子里坐哪个座位":

成分常用形式例句
主语doing / to doSwimming keeps me fit. / To learn English takes time.
宾语doing / to doI enjoy reading. / I want to leave.
定语doing / done / to dothe sleeping baby / a broken cup / a book to read
状语doing / done / to doHearing the news, she cried. / To pass, he studied hard.
补语(宾补)do / doing / done / to doI saw him run / running. / I had my hair cut. / I want you to go.

⚠ "作什么成分"决定用哪一大类,"主动还是被动"再决定 doing 还是 done。两问要连着问,缺一步就容易填错。

🌳 互动 2 — 非谓语选择决策树:成分→主被动→选 to do / doing / done

先选"作什么成分",再选"主动还是被动",决策树会点亮该走的路径,给出该用的形式和例句。多切换几次,把两问的组合走一遍。

③ 主动 vs 被动:doing 主动,done 被动

这是全章的命门。判断的钥匙是逻辑主语——非谓语动作真正的"发出者"。

逻辑主语"做"动作 → doing　|　逻辑主语"被"动作 → done
短语逻辑主语关系形式
the falling leavesleaves叶子自己在落(主动)doing
the fallen leavesleaves叶子已经落了(完成)done
the exciting newsnews消息令人兴奋(主动施加)doing
the excited crowdcrowd人群被激动了(被动感受)done

-ing/-ed 情感形容词是高频考点:令别人如何 → -ing(exciting, boring, surprising);人自己感到如何 → -ed(excited, bored, surprised)。口诀:物/事用 -ing,人的感受用 -ed。

⚠ 别用中文"意思"硬猜。永远回到"逻辑主语是施动者(doing)还是受动者(done)"这一条铁律上。

⚖️ 互动 3 — doing vs done 主被动判断器:逻辑主语是"做"还是"被做"

左天平托盘=逻辑主语,右托盘=动作。逻辑主语发出动作,天平倒向 doing(主动);逻辑主语承受动作,倒向 done(被动)。点不同短语看天平往哪边压。

④ 独立主格:逻辑主语单列出来

当非谓语的逻辑主语和句子主语不一致时,不能直接甩一个分词(会成悬垂!),要把逻辑主语写在分词前面,构成"独立主格结构"。

名词/代词 + doing / done / to do / 形容词 / 介词短语
例句说明
Weather permitting, we will go.weather 是 permit 的逻辑主语,主动 → doing
The work done, they went home.work 被 do,被动完成 → done
He sat there, eyes closed.eyes 被 close,被动 → done(=with eyes closed)
There being no bus, we walked.there be 结构的独立主格 → being

常带 with:with + 名词 + doing/done/介词短语,表伴随状态。如 with the door locked(门被锁着)、with tears running down(泪流着)。

⚠ 判断独立主格里用 doing 还是 done,仍然回到那一问:这个名词(逻辑主语)是主动做还是被动被做。permit 是天气"允许"(主动),done/closed 是被别人做(被动)。

⑤ 只接 to do / 只接 doing 的动词

有些动词的宾语挑食,只收 to do 或只收 doing;还有一批两者都行但意思不同。这是选择题重灾区,必须记牢。

只接 to do只接 doing
want, hope, wish, decide, agree, refuse, manage, offer, promise, plan, expect, choose, pretend, afford, fail, learnenjoy, finish, mind, avoid, practice, suggest, consider, imagine, admit, deny, keep, miss, risk, escape, delay, give up

口诀: 只接 doing 的常记"一坚持二完成三悔恨四停止" + enjoy/avoid/mind…;只接 to do 的多为"想要/决定/答应"类表将来倾向的动词——这正呼应 to do 的"将来"内核。

两者都行但意思变:remember / forget / regret / try / stop / go on / mean

动词+ to do(未做/将做)+ doing(已做/正做)
remember记得去做(还没做)记得做过(已发生)
forget忘了去做忘了做过
stop停下来去做另一事(目的)停止正在做的事
try努力/设法去做尝试做做看
regret遗憾地要去做(通知等)后悔做过

⚠ 判断口诀:to do 指向"还没做/将要做"(将来性);doing 指向"已经做/正在做"(已然性)。这和三形式的核心语义完全一致,别死背,顺着"将来 vs 已然"推。

🔀 互动 4 — remember/stop/try 决策器:to do(将来/目的) vs doing(已然/正在)

时间轴:如果动作指向"未来/还没做"→ to do;指向"过去/正在做"→ doing。点四个例子,看小箭头在时间轴上落向左(已然)还是右(将来)。

⑥ 感官 / 使役动词后的宾补

感官动词(see, watch, hear, feel, notice…)和使役动词(make, let, have, help)后接省 to 的不定式(do)或分词作宾语补足语,三种形式各有含义:

see sb do　·　see sb doing　·　see sb done
形式含义例句
see sb do看见做了全过程(完整/经常)I saw him cross the road.（看着他走过去了）
see sb doing看见正在做(某一瞬间/进行)I saw him crossing the road.（正走到一半）
see sth done看见被动作用(宾语被做)I saw the tree cut down.（看见树被砍倒）

使役 make/let/have + sb do(省 to):make him go, let me try, have sb do。但被动语态里 make 要还原 to:He was made to work. help 特殊:help sb (to) do,to 可省可留。

⚠ 高频陷阱:①感官/使役后省 to(saw him go 不是 to go);②但变被动 was seen to go / was made to work,to 又回来了;③宾语被动作用时用 done(had my car repaired 让车被修)。

👁️ 互动 5 — 感官/使役宾补:see sb do / doing / done 对比

同一个句子 "I saw the man ___ the box",切换 do / doing / done,画面展示"搬完了 / 正在搬 / 箱子被搬走",体会三种宾补的时态与语态差别。

⑦ 非谓语作状语:分词状语填空

分词短语放句首或句尾作状语,表时间、原因、条件、结果、伴随、方式。形式还是那一问决定:

逻辑主语关系形式例句
主动 / 同时进行doingWalking home, I met Tom.（一边走一边）
被动doneGiven more time, we could finish.（被给…条件）
已完成(强调先后)having doneHaving finished, he left.（做完之后）
目的 / 将来to doHe got up early to catch the bus.

铁律: 分词状语的逻辑主语必须等于句子主语。Walking home, I met Tom → 走的是 I,一致,✓。若不一致就是悬垂分词(见易错集)。

⚠ 状语位置的分词,先找句子主语,判断它和分词是主动(doing)还是被动(done),再看是否强调"先后完成"(having done)。三步走稳。

✏️ 互动 6 — 分词作状语填空:判断主被动/完成,选正确形式

屏幕给出带空的句子,点下面的选项(doing / done / having done / to do)。判对了变绿并解释为什么,判错变红提示回到"主被动/完成"那一问。

⑧ 全题型地图

题型A — 作主语/宾语(动名词 vs 不定式)

触发信号:空格在句首当主语,或在动词/介词后当宾语;句中已有谓语,这个动词无处安放。

标准动作:把动作"名词化"→ 常用 doing;若强调"具体一次、将来性"或句型 It is + adj + to do,用 to do。介词后永远用 doing。

示例(带解):Reading aloud helps memory.（主语,习惯性动作→动名词 doing）/ He is good at swimming.（介词 at 后→doing）/ It is important to plan ahead.（It...to do 句型→不定式）

陷阱:介词后误用 to do(look forward to doing 不是 to do);句首主语两可时看语义(抽象习惯→doing)。

题型B — 作定语(doing / done / to do)

触发信号:空格紧挨一个名词,修饰它;这个名词是分词的逻辑主语。

标准动作:名词主动/正进行→doing;名词被动/已完成→done;表"将要做、待做"→to do。

示例(带解):the developing country（正在发展→doing）/ the developed country（已发展→done）/ the boy reading a book（男孩主动读→doing）/ the problem to be solved（待解决→to do 表将来被动）。

陷阱:developing(发展中) vs developed(发达)一字之差意思反;分不清名词是施动还是受动。

题型C — 作状语(分词状语)

触发信号:空格在句首/句尾,带逗号,和主句用逗号隔开,补充时间/原因/条件/结果。

标准动作:先找句子主语,判断它与分词主动(doing)还是被动(done);强调先完成用 having done;表目的用 to do。

示例(带解):Seeing the teacher, the students stood up.（学生主动看→doing,时间/原因状语）/ Praised by all, she felt proud.（她被表扬→done,被动状语）

陷阱:悬垂分词——分词逻辑主语和句子主语不一致(见易错集①);该用完成体却用一般式。

题型D — 作宾补(see sb do / doing / done)

触发信号:感官动词(see/hear/watch/feel/notice)或使役动词(make/let/have/help)+ 宾语 + 空格。

标准动作:看完全过程用 do(省 to);正在进行用 doing;宾语被动作用用 done。被动语态里 to 要还原。

示例(带解):I heard someone knock at the door.（听见敲了→do 全程）/ I heard someone knocking.（正敲→doing）/ I had my bike repaired.（车被修→done）

陷阱:该省 to 却加 to(saw him go);变被动忘还原 to(was made to work);宾语被动时误用 doing。

题型E — doing vs done 主被动辨析

触发信号:选项里 doing 和 done 同时出现,或 -ing/-ed 情感形容词二选一。

标准动作:锁定逻辑主语,一问定音:它施动用 doing,它受动用 done;物/事令人怎样→-ing,人感到怎样→-ed。

示例(带解):The news is exciting.（消息令人兴奋→-ing）/ We are excited.（人被激动→-ed）/ the rising sun（太阳自己升→doing）/ the risen sun(此处rise不及物,一般用 the sun having risen)。

陷阱:用中文语感猜;忘了"人用 -ed,物用 -ing";不及物动词没有被动 done(如 happen 无 happened 作被动定语)。

题型F — 不定式表目的

触发信号:问"为了什么、目的是"、句中有"in order to / so as to"或可替换为"为了…"。

标准动作:用 to do 作目的状语(可强调为 in order to do / so as to do);置句首用 To do…;否定目的 in order not to do。

示例(带解):He worked hard to pass the exam.（为通过考试→to do 目的）/ To save money, she walks to work.（句首目的状语）

陷阱:目的状语误用 doing(应 to do);so as to 不能放句首;否定位置 in order not to do。

⑨ 易错集

① 悬垂分词:分词状语的逻辑主语必须=句子主语。✗ Walking home, the rain started.(雨不会走路!)✓ Walking home, I was caught in the rain。填空先核对主语一致。

② doing 还是 done:永远回到逻辑主语"施动(doing)/受动(done)"。the falling leaves(自己落)vs the fallen leaves(已落被动/完成)。

③ 谓语 vs 非谓语:一句只能一个谓语。若句中已有谓语动词,再来的动词一律变非谓语;两个动词想并列必须用 and 连接或从句。

④ remember/forget/stop + to do vs doing:to do=还没做/将去做;doing=已经做/正在做。顺"将来 vs 已然"推,别死记。

⑤ 感官/使役宾补:后接省 to 的 do(saw him go);变被动 to 还原(was seen to go);宾语被动用 done(had it done)。

⑥ 介词后用 doing:所有介词(含 to 作介词时)后跟动名词。look forward to doing, be used to doing, devote…to doing。别当不定式 to do。

⑦ -ing/-ed 情感词:物/事用 -ing(interesting, boring),人的感受用 -ed(interested, bored)。The book is boring; I am bored.

⑧ 不及物动词无被动 done:happen, appear, exist 等不能有被动完成的 done 作定语(不能说 the happened thing,用 what happened)。

⑨ having done vs done:强调"先完成再有主句"用完成式 having done;单纯被动/状态用 done。Having finished the work, he left.

⑩ 尖子生拔高视角

① "两问法"是唯一底层算法: 无论题目多花哨,先问成分(主/宾/定/状/补→定大类),再问逻辑主语的主被动/时态(施动 doing/受动 done/将来 to do/先完成 having done)。把这两问写在草稿上过一遍,任何非谓语题都能拆开。别背零散规则,背算法。

② 三形式的语义内核是连贯的: to do 的"将来/目的"和 remember to do 的"还没做"、the book to read 的"待读"是同一个内核;doing 的"主动进行"贯穿现在分词定语、伴随状语、动名词;done 的"被动完成"贯穿定语、状语、宾补。看穿内核,规则自动统一。

③ 逻辑主语是暗线,独立主格是它的显形: 平时逻辑主语藏着(靠上下文找),一旦它和句子主语不同,就必须显形为独立主格(Weather permitting)或 with 结构(with the door locked)。发现"分词的主人和句子主语不是一个"→立刻想到独立主格,而不是硬塞分词造成悬垂。

④ 被动语态是 to 的"试金石": 感官/使役后省 to,但一变被动 to 就现身(He was made to work / She was seen to leave)。凡遇 be + 过去分词 + 空,警惕"这里要不要还原 to"——这是区分尖子生和普通生的细节题。

⑤ 时间轴思维破解 to do/doing 之争: 把动作放到时间轴上:指向未来/尚未发生→to do;已经发生/正在进行→doing。stop to rest(停下,去休息:休息在未来)vs stop resting(停止正休息的事)。一根时间轴,秒杀 remember/forget/regret/stop/try/mean 全家。
