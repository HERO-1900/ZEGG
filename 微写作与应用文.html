微写作与应用文

微写作(小作文)是一场戴着镣铐的短舞:题目让你写书信、倡议书、演讲稿、通知、推荐语、新闻点评……你就得穿上那身「制服」。它考的不是文采,是你会不会按文体规矩说话——文体对不对、格式全不全、语言得不得体、内容扣没扣要求。四把尺子,任何一把塌了都直接扣分。这一页把每种文体的骨架拆给你看,再给你工具当场练格式和得体。

🗣️ 微写作(人话版): 它就像给不同场合穿对制服。写信要有称呼、问候、落款;演讲要有呼告和号召;倡议书要分条列出行动;颁奖词要凝练又抒情。麻雀虽小、五脏俱全——短短 150~200 字里,该有的部件一个都不能少,该用的谦敬语一个都不能错。评分四关:文体对、格式全、语言得体、内容扣要求。别写成大作文的缩水版,它是另一套规矩。
四关口诀: ①选对文体(题里「写一封信/一则倡议书/一段演讲」就是文体指令,穿错制服直接崩);②格式齐全(称呼顶格、正文空两格、落款靠右下);③语言得体(对人用敬辞、称己用谦辞、公开场合要庄重);④内容扣题(任务要求的信息点一个不漏)。四关全过,分才拿满。

① 微写作是什么:一道题,四把尺子

微写作(高考/期末常见 10 分左右)是短小的实用文。它和大作文的最大区别:大作文比的是思想与文采,微写作先比的是规矩。阅卷老师的四把尺子几乎是「一票否决」式的:

尺子问的是塌了会怎样
文体你写的是不是题目要求的那种文体?写成别的文体,基本判低档
格式称呼/正文/落款等部件齐不齐、位置对不对?缺一个部件扣一档
语言谦敬得体吗?场合庄重吗?口吻对吗?用错谦敬/太随便,扣分
内容题目点名要的信息,写全了吗?漏信息点=没扣题,失分

⚠ 微写作最冤的失分:内容写得挺好,却忘了称呼或漏了落款——格式一关白丢分。先把「骨架」搭齐,再填「血肉」。

📄 互动 1 — 各文体格式模板:点一个文体,看它的「骨架」长什么样

每种文体都有固定「部件」。灰标签是位置提示(顶格/空两格/居中/右下)。先记住骨架,考场上照着填内容就不会漏部件。

② 书信:最常考,格式最讲究

书信是微写作的「常客」,也是格式最容易丢分的。它有五个部件,位置各有规矩:

部件位置规矩示例
①称呼顶格写,后加冒号敬爱的王老师:
②问候语另起一行,空两格您好!
③正文接问候,空两格起段(主体内容)
④祝颂语「此致」空两格;「敬礼!」另起顶格此致 / 敬礼!
⑤署名日期右下角,署名在上、日期在下学生 李明 / 2026年7月11日

称呼要「得体」:对老师加「敬爱的」,对同学加「亲爱的」,对长辈加「尊敬的」。称呼透露你和对方的关系与敬意,是第一印象。

⚠ 三大高频错:①称呼不顶格或漏冒号;②「敬礼!」没另起顶格,写成一行;③署名日期堆在左边或漏日期。

✉️ 互动 2 — 书信格式检查器:点亮四大部件,看它们的正确位置
点上面的按钮,逐个点亮书信的部件,右侧「纸」上会显示它的正确位置。

书信 = 称呼 + 问候正文 + 祝颂 + 署名日期。位置错了,即使内容满分,格式分也拿不全。把这张「纸」印在脑子里。

③ 实用文:倡议书 · 通知 · 启事(讲清事)

这一类的共同点:标题居中、正文说清事、落款在右下。语言要简明、准确、有条理,别抒情。

倡议书 — 号召大家一起做

标题(居中):关于XX的倡议书
称呼(顶格):广大同学们:
正文:①为什么(背景/意义) ②怎么做(分条列出行动) ③呼吁
落款(右下):倡议人 / 日期

倡议书的灵魂是「分条的具体行动」——不能只喊口号,要写出「①…②…③…」让人照着做。结尾要有号召力。

通知 — 告诉大家一件事

标题(居中):通知
称呼(顶格):被通知的对象(可省)
正文:把「谁、何时、何地、做什么、有何要求」讲清楚
落款(右下):发通知的单位 / 日期

通知讲究「五要素齐全」:时间、地点、事项、对象、要求。少一个,别人就办不成事。语言要干脆、无歧义。

启事 — 公开告知(寻物/招领/征集)

标题(居中):寻物启事 / 招领启事 / 征集启事
正文:事由 + 具体信息(时间/地点/特征/联系方式)
落款(右下):启事人 / 日期

⚠ 这三类最忌抒情和华丽。倡议书没有「分条行动」、通知漏了「时间地点」、启事漏了「联系方式」——都是致命伤。

④ 有感染力的文体:演讲稿 · 颁奖词 · 推荐语

这一类反过来:要有温度、有文采、能打动人。格式相对宽松,但「口吻」和「感染力」是评分核心。

演讲稿 — 面对听众,要「鼓动」

称呼(顶格):尊敬的老师、亲爱的同学们:
开头:问候 + 亮出话题(可用设问/引用抓住人)
主体:分层说理,多用呼告、反问、排比,增强气势
结尾:发出号召 + 致谢(谢谢大家!)

演讲稿的命门是「对象感 + 鼓动性」:心里始终装着台下的听众,多用「我们」「让我们」,句子要有节奏、能点燃情绪。一般不写落款。

颁奖词 — 献给获奖者的诗

一段话:概括事迹 + 提炼精神 + 抒情礼赞
口吻:常用第二人称「你」,凝练、有画面感、有文采

颁奖词像「感动中国」的解说词:短、美、有力。先说他做了什么(事迹),再拔高到他象征了什么(精神),最后深情礼赞。

推荐语 — 我为什么推荐他/它

一段话:推荐对象 + 理由(特点/事迹/价值) + 推荐结论

推荐语要有理有据、真诚有说服力:说清「好在哪」,别空夸。推荐书、推荐人、推荐活动都是这个套路。

⚠ 演讲稿写成「书面报告」(没有对象感)、颁奖词写成「流水账」(只叙事不抒情)、推荐语写成「广告口号」(只夸不说理)——都跑偏了。

⑤ 新闻点评:有事实,更要有观点

新闻短评考的是你的态度和思辨。它不是复述新闻,而是就事论事、亮明观点、讲出道理。

①引:简要概括新闻事实(一两句,别复述全文)
②评:亮出你的观点/态度(鲜明,一句话立场)
③析:分析说理(为什么这样看,讲清逻辑)
④结:总结/升华/呼吁(收束,回扣观点)

四字诀:引—评—析—结。核心是「观点鲜明 + 就事论事」:先摆事实,再表态,然后说清「凭什么」,别空泛喊口号,也别偏题跑到别处。

⚠ 两大坑:①光复述不评论(变成新闻摘要,没观点);②观点悬空(只表态不分析,没说服力)。评论的分量在「析」。

🎯 互动 3 — 文体识别:读任务,判断该用哪种文体(先想再点)

审题第一步永远是「认文体」:任务里的动词和场景就是线索——「号召大家」→倡议书,「通知同学」→通知,「颁给他」→颁奖词,「谈看法」→新闻点评。选错文体,后面全白搭。

⑥ 得体:谦敬语与场合口吻

「得体」= 说话看对象、场合、身份。核心是一对镜像:敬辞抬高对方,谦辞压低自己。用反了,是硬伤。

类别敬辞(对别人/别人的)谦辞(对自己/自己的)
父亲令尊、令严家父、家严
意见高见、高论拙见、浅见、愚见
作品大作、佳作(拜读)拙作、拙文(请斧正/指正)
赠送惠赠(别人送我)敬赠、奉上(我送人)
到来光临、莅临(别人来)拜访、登门(我去)
询问垂询、垂问(尊长问我)请教、动问(我问人)
学校贵校敝校

场合口吻:公开、正式场合(演讲、倡议、通知)要庄重、书面;私人书信可以亲切。别把「哥们儿」「超燃」搬进正式文体,也别把公文腔塞进温情的颁奖词。

⚠ 经典错:把「令尊」当自己父亲、请人改文章说「拜读」(该说「斧正/指正」)、自己作品叫「大作」。敬谦一颠倒,得体分立刻没。

🤝 互动 4 — 得体用语辨析:选出符合场合的说法(谦敬 / 口吻)

口诀:「敬人谦己」——凡是抬高对方的用敬辞,凡是说自己的用谦辞。拿不准时想一句:这话是在「抬对方」还是「贬自己」?

⑦ 字数意识:短,但要「五脏俱全」

微写作一般限150~200 字(以题目要求为准)。字数太少=内容不足;超字数=不简洁。要在有限字数里把部件写全、把要点写到。

先列骨架(部件)每部件配1~2句删掉废话套话数一数信息点留20字机动

写前 30 秒做三件事:①认文体(穿哪身制服);②列部件(骨架有哪些);③圈要点(题目点名要什么)。三步做完再动笔,格式和内容都不会漏。

⚠ 别用大作文的「凤头豹尾」堆砌法。微写作字字有用,开头不必绕圈子,直接进入文体的第一个部件。

⑧ 全题型地图

题型A — 书信类(感谢信/建议信/邀请信/家书)

触发信号:「写一封信」「给…写信」「以书信形式」,或点名感谢/建议/邀请某人。

标准动作:称呼(顶格,得体)→ 问候(空两格)→ 正文(空两格,扣事由)→ 此致/敬礼! → 署名 / 日期(右下)。

格式范例(带解析):
敬爱的李老师:(顶格+敬称+冒号)
　　您好!(空两格,问候单独成行)
　　三年来,是您……(正文扣「感谢」)。
　　此致(空两格)
敬礼!(另起,顶格)
　　　　　　学生 张明(右下,署名)
　　　　　　2026年7月11日(署名下,日期)

陷阱:漏称呼/落款;「敬礼!」不另起顶格;称呼不得体(给老师写「亲爱的」偏随便)。

题型B — 倡议书型

触发信号:「写一则倡议书」「号召大家…」「发出倡议」,主题多为环保、节约、读书、文明等。

标准动作:标题居中 →「广大同学们:」→ 意义/背景 → 分条列出具体行动(①②③) → 呼吁 → 倡议人 / 日期。

格式范例(带解析):
关于节约用水的倡议书广大同学们:
　　水是生命之源……(意义)。为此我们倡议:
　　1. 随手关紧水龙头;(分条=灵魂)
　　2. 一水多用,循环利用;
　　3. 发现漏水及时报修。
　　让我们从点滴做起!(呼吁)
　　　　　　　　高二(3)班全体同学 / 2026年7月11日

陷阱:只喊口号没有「分条行动」;漏标题或落款;写成抒情散文。

题型C — 演讲稿型

触发信号:「写一篇演讲稿」「作为代表发言」「国旗下讲话」,场景为晚会、班会、典礼。

标准动作:称呼(尊敬的…亲爱的…)→ 开头问候+亮话题 → 主体分层+呼告/排比 → 结尾号召+「谢谢大家!」。不写落款。

格式范例(带解析):
尊敬的老师、亲爱的同学们:
　　大家好!今天我演讲的题目是《……》。(亮话题)
　　……(主体,多用「让我们」「难道不是吗?」增强鼓动)。
　　同学们,让我们……!(号召)谢谢大家!(致谢收尾)

陷阱:没有对象感(不像在对人讲);缺号召/致谢;硬加落款(演讲稿不需要)。

题型D — 推荐语 / 颁奖词型

触发信号:「为…写一段推荐语」「写一则颁奖词」「向…推荐一本书/一个人」。

标准动作:推荐语=对象+理由(特点/价值)+结论;颁奖词=概括事迹+提炼精神+抒情礼赞(常用第二人称「你」)。凝练有文采。

格式范例(带解析):
颁奖词:你像一株倔强的向日葵。跌倒了,你爬起来把全程跑完;(事迹)你让我们懂得,输赢之外还有更重的东西——坚持。(精神+礼赞)
推荐语:我推荐《平凡的世界》。它写尽普通人的奋斗与温情,(理由)能让每个平凡的你,找到不平凡的勇气。(结论)

陷阱:颁奖词写成流水账(只叙事不升华);推荐语只夸不讲理由;语言不凝练。

题型E — 新闻点评型

触发信号:「针对以上新闻,谈谈你的看法」「写一则新闻短评」「对此现象评论」。

标准动作:引(概括事实)→ 评(亮观点)→ 析(讲道理)→ 结(升华/呼吁)。观点鲜明、就事论事。

格式范例(带解析):
近日,某校设立「诚信考场」无人监考。(引:概括事实)此举值得点赞。(评:亮观点)它把监督权交还给学生,用信任培养自律,比一味监考更能立德树人。(析:讲理)愿这样的信任,越来越多。(结:呼吁)

陷阱:光复述新闻没观点;观点悬空缺分析;偏题评到别处去。

题型F — 通知 / 启事型

触发信号:「写一则通知」「拟一份启事」「寻物/招领/征集」。

标准动作:标题居中 → 正文把「时间/地点/事项/对象/要求」或「事由/特征/联系方式」讲清 → 落款单位 / 日期。简明无歧义。

格式范例(带解析):
通知各班同学:
　　定于本周五下午三点,在操场举行大扫除,(时间+地点+事项)请各班自带工具准时参加。(要求)
　　　　　　　　校学生会 / 2026年7月11日

陷阱:漏时间/地点(通知);漏联系方式(启事);标题不居中;语言啰嗦或抒情。

⑨ 易错集

① 文体穿错制服:让写倡议书却写成演讲稿、让点评却复述新闻。审题先圈「文体动词」。

② 书信漏部件:忘称呼、忘「敬礼!」另起顶格、忘署名日期。骨架先搭齐。

③ 倡议书没分条:只喊「让我们行动起来」,没有「①②③具体做法」——灵魂丢了。

④ 通知/启事漏要素:通知漏时间地点、启事漏联系方式,别人办不成事。

⑤ 谦敬颠倒:「令尊」当自己爸、请人改文说「拜读」、自己作品叫「大作」。敬人谦己。

⑥ 场合口吻错位:正式文体用网络语/口水话,或温情颁奖词塞公文腔。

⑦ 演讲稿加落款:演讲稿一般不写署名日期;倡议/通知/启事/书信才需要落款。

⑧ 颁奖词写成流水账:只叙事不升华、不抒情,没「提炼精神」这一步。

⑨ 新闻点评观点悬空:表了态却不讲「为什么」,评论没分量。分量在「析」。

⑩ 字数失控:太短内容不足,超字数不简洁。先列骨架,每部件配一两句。

⑩ 尖子生拔高视角:各文体万能模板

① 书信模板:「敬爱的X:/ 　　您好!/ 　　(由头一句,点明写信缘由)+ (主体,扣感谢/建议/邀请,分点更清晰)+ (期待/祝愿一句)。/ 　　此致 / 敬礼! / 　　　X的Y(署名)/ 年月日」。关键:称呼得体 + 由头开门见山 + 落款别漏。

② 倡议书模板:「关于__的倡议书(居中)/ 广大__:/ 　__的现状/意义(1句)。为此,我们倡议:/ 　1.__;2.__;3.__(动词开头,可操作)/ 　__,让我们从我做起!/ 倡议人 / 日期」。关键:分条行动是命脉,动词开头最有力。

③ 演讲稿模板:「尊敬的__、亲爱的__:/ 　大家好!(设问/引用抓人)→ 今天我想说__ → 分层:是什么/为什么/怎么办,穿插『让我们』『难道不…吗』→ 号召:同学们,让我们__!谢谢大家!」。关键:全程对象感 + 排比反问点火 + 不落款。

④ 颁奖词模板:「(比喻起句,如『你是…』)→ 概括他做了什么(事迹,画面感)→ 提炼象征了什么(精神)→ 深情礼赞一句」。关键:事迹要具体、精神要拔高、语言要凝练,善用第二人称『你』。

⑤ 推荐语模板:「我(向你)推荐__ → 它/他好在哪(1~2个理由,具体)→ 它能给你带来什么(价值/结论)」。关键:理由要真诚具体,别空夸,落在『对读者的意义』上。

⑥ 新闻点评模板:「近日,__(引,1句概括)→ 对此,我认为__(评,亮观点)→ 因为__/这说明__(析,讲理,可正反)→ 愿__/期待__(结,升华呼吁)」。关键:引—评—析—结,重心放在『析』,就事论事不跑题。

⑦ 通用心法:微写作 = 「先认文体、再搭骨架、后填血肉、末查得体」。四步走完,格式全、内容扣、语言得体三关一起过。把六套模板背成肌肉记忆,考场上任何小作文都是「套骨架 + 换内容」。
